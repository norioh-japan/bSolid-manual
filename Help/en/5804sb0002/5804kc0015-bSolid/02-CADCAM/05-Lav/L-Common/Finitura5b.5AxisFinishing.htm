5-axis finishing

Description of the machining operation

Generates a finishing operation to give the previously rough-shaped or pocketed piece a final finish; only for machines with 5-axis milling unitWorking unit equipped with a system for pointing the tool in every possible direction (see the machine user instructions). (5-axis working head(1) Side faces of the pieces (see "sides of the piece"). (2) Working unit (woodworking machines).).

This type of machining operation can be applied to analytic surfacesFigures of a three-dimensional area, that represent a thin open or closed structure., large bas-reliefs, or two geometric elements generated to create a profile on which the tool can be spread.

SWITCH

SWITCH

Swaps the upper and lower geometric elements.

Swarfing mode

SWMD

Strategy type to generate the path to follow during the machining operation.

List of available options:

Sync. upper and lower curve

Machining operation that divides the drawn path curves (figure 3) into steps of equidistant length; the tool axis is aligned with each pair of steps.

Synch. with director lines

Machining operation with tool tilt established by the drawn director lines A (figure 19). The lines allow the tool to be tilted in specific points along its path. In this case, choose the lines in the "Director lines" data field.

243. Representation of the data

A. Lines for establishing the tool tilt.
B. Drawn geometric elements that represent the path curves.
Synch. with main direction

Machining operation that tries to keep the tool parallel with the axis chosen in the "Main direction axis" field, applying the tilt only if necessary.

Best adaptation

Machining operation that tries to adapt the tool tilt in the best possible way to the two path curves (i.e. the two drawn geometric elements - ref. A) and to the selected surface (ref. B) in the "Director surfaces" data field (figure 20). In this way, the tool always tries - on the basis of the two curves - to remain perpendicular to the chosen surface.
