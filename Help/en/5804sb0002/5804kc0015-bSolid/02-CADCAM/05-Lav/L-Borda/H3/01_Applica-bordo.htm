Reversal

RV

Enables (box tickedClick on a check box to tick it and thereby enable that function.) or disables a reversal of the gluing(1) Set of operations for attaching the edgebanding strip to the piece. operation direction.

To perform the machining operation, the unit does not begin from the initial point of the profile, but from the end point. Only used if the profile has been generated in the anti-clockwise direction.

Offset for machining operation start

OFG

Transfer of the initial machining point (on the same profile trajectory) in relation to the initial programmed geometry point.

A positive valuePredefined / pre-set. Standard information useful for the control, if the requested information is not entered. shifts the machining start point after the initial profile point, while a negative value shifts it forward.

Advance feed variation

T2D

Used to make the edgebanding strip slide slightly after being cut with the shears, so that it arrives correctly in contact with the edge trimming(1) Piece edge trimming: operation for making cuts on the side faces (heads) in order to square the piece. (2) Border edge trimming: cuts at the ends of the edgebanded side. blade of the gluing unit.

If the edgebanding strip taken from the magazine does not reach the blade, enter a value higher than zero.

Magazine position

DES

Used to define the position of the magazine when the gluing unit descends near the programmed point.

High

The magazine stays up (it doesn't descend with the unit).

Low

The magazine descends with the unit.

Loading method

MEL

Used to indicate how to load the edgebanding strip in the gluing unit.

Automatic

Manual edgebanding strip loading disabled. The edgebanding strip is automatically taken from the magazine.

Manual - left

Manual loading of the edgebanding strip on the left-hand part of the machine.

Manual right

Manual loading of the edgebanding strip on the right-hand part of the machine.

Manual central

Manual loading of the edgebanding strip on the central part of the machine.

Glue roller copying

GRC

Movement of the glue roller to increase the pressure on the side to be edgebanded.

Data check

EDC

Enables or disables the machine data check, to see whether the data tally with the programmed machining operation.

Tracing on start of edgebanding strip

ACTT

Enables (box ticked) or disables the use of the tracing device (part of the gluing unit) to detect the correct position of the edgebanding strip initial point, so as to adjust the closure of the ring.

Tracing speed

WTE

The speed that the gluing unit must reach during the tracing phase in order to detect the correct position of the edgebanding strip initial point.

Speed variation distance

DTE

Distance between the point where the second presser unit is activated and the strip closure point, for tracing the strip at the set speed.

285. Representation of the data
Presser-sensor offset

OPE

Distance between the point of contact of the second presser roller and the point where the strip is detected with the thickness tracer.
