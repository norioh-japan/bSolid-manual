List of data fields

Maximum distance

DIST

Constructs polylines with short or long elements. It is the maximum distance between two points within which the polyline can be constructed. To construct the polyline, the function only uses those points whose distance is less than this valuePredefined / pre-set. Standard information useful for the control, if the requested information is not entered.. A low value will produce polylines with short elements, while a high value will produce polylines with long elements.

Chordal tolerance

TOLL

Excludes from the polyline construction any points that are too far from the polyline itself. This is therefore the distance of the point in question from the polyline, expressed in millimetres. A low value will simplify the polyline, making it straighter. A high value will make it more angular.

Minimum tangency angle

ANG_MIN_TANG

Excludes from the polyline construction any points with a tangency angle less than this value. A high value will simplify the polyline, making it straighter. A low value will make it more angular.

NOTES:

Indicate 180° to annul the checks on the tangency angles. A connection is always created between two elements whose tangency angle is between 0° and 15°, so a curve is created with the next point.
