# 円弧 ![円弧アイコン](../../../FIGURE/15-icone/b15b0001/Draw_Arc_2PointRadius.png)

## 説明

開始点および終点という2つの頂点のみを持つ曲線のセグメントをトレースする。

2つの点を修正するまたは特定の特性を持つ円弧をトレースするには、データエリアの関連フィールドを使用します。

## 円弧を挿入するステップ

1. **描画バー**の![円弧](../../../FIGURE/15-icone/b15b0001/Draw_Arc_2PointRadius.png)ボタンをクリックします。

2. 円弧のタイプデータフィールドの関連ボタンをクリックすることによって作成する円弧のタイプを選択します。

![円弧作成手順](../../../FIGURE/10-videateComplete/arte4/b10b0399.gif)

3. 円弧を開始したい点をクリックします。

4. 円弧を終了したい点をクリックします。

![円弧描画プロセス](../../../FIGURE/10-videateComplete/arte4/b10b0398.gif)

5. **取り消す**ボタンをクリックして**円弧**カードを閉じます。

以前の円弧の終点と接続している一連の円弧を描画するには、標準バー上の**Snap**ツール（スナップツール）の関連コマンドを使用します。

## 一般的パラメーター

### 円弧のタイプ
**TYP**

円弧のタイプを選択するために使用されます。

#### ![3点円弧](../../../FIGURE/15-icone/b15b0001/Draw_Arc_3Points.png) 3点
3点を通る円弧を作成する。

#### ![2点半径円弧](../../../FIGURE/15-icone/b15b0001/Draw_Arc_2PointRadius.png) 2点+半径
円弧の半径および開始点と終点の座標を基準値として使用して、円弧を作成する。

#### ![2点中心円弧](../../../FIGURE/15-icone/b15b0001/Draw_Arc_2PointCenter.png) 2点+中心点
円弧の中心点の座標および開始点と終点を基準値として使用して、円弧を作成する。
