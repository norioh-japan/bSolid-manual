Spreading / Projection

Description of the machining operation

Used to spread or project a machining operation on a surfaceFigures of a three-dimensional area, that represent a thin open or closed structure.; in this case, the machining operation will follow the surface on which it is spread or projected.

In order to be applied, the projection/spreading therefore needs a machining operation A and a surface B (figure 1).

276. Example:

A. Milling to be projected;
B. Surface on which to project the milling;
C. Projected milling;
D. Projection simulation.

Mode

CBO

Used to select a machining method.

Projection

Activates the projection of the machining operation on the surface.

Spreading

Activates the spreading of the machining operation on the surface.

Path to be projected

WORK

Shows the list of machining operations created, so you can select the one to be projected or spread on the basis of the type chosen in the "Mode" data fieldA defined area containing data or commands (data field or command field)..

Reference surface

SURF

Shows the list of surfaces created, so you can select the one on which to project or spread the machining operation on the basis of the type chosen in the data field"Mode" .

Accuracy

PR

Tolerance for defining the precision of the machining operation on curved surfaces, in order to calculate the tool trajectories.

The valuePredefined / pre-set. Standard information useful for the control, if the requested information is not entered. to be entered refers to the chord height necessary to ensure that the arc will be more or less precise. A value lower than 0.05 creates more precise arcs; the calculation is slower. A value higher than 0.05 creates less precise arcs; the calculation is quicker.

277. Representation of the data

Offset in Z

OFFSET

For "Projection" mode only. Distance in Z from the initial point of the machining operation selected in the "Path to be projected" data field in order to increase the machining depth.

Output format

AXESN

For "Spreading" mode only. Used to decide what type of machining operation to apply.

5 axes

The final machining operation is performed with the 5-axis milling unitWorking unit equipped with a system for pointing the tool in every possible direction (see the machine user instructions)..

3 axes

The final machining operation is performed with the 3-axis milling unit.

Lead-in point selection

Used to activate the cursorMobile element that is visible on the screen, and which moves as the mouse is moved. in order to insert (in the drawing) the starting point for spreading the machining operation selected in the "Mode" data field.

Click(1) Indicates the action of pressing (and immediately releasing) the mouse button on an object (icon, button, etc.) located under the pointer on the screen. (2) (verb) Pressing and immediately releasing the left mouse button to enable the function of a selected command. on the adjacent buttonCommand field that is responsive to mouse clicks and that enables operations to be activated., position the cursor on the graphics, and click on the surface to establish the lead-in point.
