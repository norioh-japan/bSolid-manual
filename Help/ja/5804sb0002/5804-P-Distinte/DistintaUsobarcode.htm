バーコードリーダーをアクティブ化する

バーコードリーダーのユーザーを有効にするには、以下の手順を行う：

1. 一般設定環境にアクセスします。
2. BarCodeカードをクリック（1）画面上のポインタの下にあるオブジェクト（アイコン、ボタンなど）の上でマウスボタンを押す（そしてすぐに離す）行為を示します。（2）（動詞）選択したコマンドの機能を有効にするため、マウスの左ボタンを押してすぐに離します。します。
3. チェックボックスバーコード受信を有効化をクリックします。
4. リーダーを設定するために必要なデータを入力します。

新規の作業リストを作成する

1. リスト環境にアクセスします。
2. ボタンをクリックします。
3. アイコンをクリックすることによって、バーコード読み取りの使用を有効にします。

アイコンの青色は、バーコードモードがアクティブであることを示しています。

4. ドロップダウンメニュードロップダウンボタンをクリックすると、垂直または水平方向に広がるオプションまたは手段のリスト。のオプションを使用して、作業リストを追加します。

作業リストのコンパイルと実行

アイコンの隣のドロップダウンメニューサブメニュー（特定の機能/作業を実施できるようにするオプションのリスト）を生成するコマンド（マウスで選択する）のリスト。のオプションを使用して、以下の操作を行うことができます：

プログラムを追加する
1. 新規の作業リストを作成する.
2. 追加プログラムオプションを選択します。
3. 特定のリーダーを使用して、リストに挿入するバーコードをスキャンします。

各バーコードを読み込むと、新規の行がリストに挿入されます。

4. ボタンをクリックして、コンパイルされたリストを挿入します。
プログラムを選択して実行します。
1. 実行するプログラムを含むリストを開きます。
2. アイコンをクリックすることによって、バーコード読み取りの使用を有効にします。
3. 作業リストのプログラムを選択し実行しますオプションを選択します。
4. 特定のリーダーを使用して、現在のリストから選択するバーコードをスキャンします。
読み込むバーコードは、リスト内の行のラベルデータフィールドデータまたはコマンドを含む所定のエリア（データフィールドまたはコマンドフィールド）。に表示されたラベル名のみを表示しなければなりません。

この手順によって、リーダーが読み込んだプログラムが自動的に起動できます。次の読み込みが行われるたび、リストの最後にプログラムが追加されます。

プログラムを追加して実行します。
1. 新規の作業リストを作成する.
2. 追加プログラムオプションを選択します。
3. 特定のリーダーを使用して、リストに挿入するバーコードをスキャンします。

各バーコードを読み込むと、新規の行が挿入され、選択したプログラムが自動的に起動されます。

リストをインポートする
1. 新規の作業リストを作成する.
2. 作業リストのインポートオープンドキュメント（ファイル）に、別のプログラムで作成された情報を、現在使用されているものとは別の形式で挿入すること。オプションを選択します。
3. リーダーを使用して、バーコードをスキャンします。

リストはバーコードで読み込んだリストに表示されるプログラムの数と同じ数の行が追加されます。

4. ボタンをクリックして、コンパイルされたリストを挿入します。
