Notes for creating the variables

When creating the name of the variable, bear in mind the following rules:

  •    it must be different from the name of the piece variables and the machining operation parameters;
  •    it must never start with a number, but always with a letter of the alphabet;
  •    it must not contain spaces and other special characters (e.g. operators such as: +; <; >; -; x; etc.);
  •    it must never contain more than 255 characters;
  •    it can be written either in capitals or in lower case letters.

When creating the valuePredefined / pre-set. Standard information useful for the control, if the requested information is not entered. of the variable, bear in mind the following rules:

  •    it can be a number;
  •    it can be a string - in this case, it must be contained within quotation marks (“LENGTH”);
  •    it can be a string plus a numerical, variable value; in this case, they must be separated by the arithmetic operator “+”.
Example: “LENGTH”+F6LX; where “LENGTH” represents the string and F6LX is the value of the length of face number 6.
  •    it can be an expression containing predefined parameters (e.g. LPX; LPY, etc.), or variables created in the variables table.

To create programs that are easier for other users to read and interpret, you are advised to give logical names to the variables. For example, a variable that has to contain or calculate the number of repeats can be called “nrRep”, where “nr…” stands for "number" and “…Rep” stands for repeats.

When there are several pieces, you can create strings containing the parameters of a specific piece. The syntax for indicating a geometric or technological parameter associated with a piece is as follows: indicate the name of the piece (e.g. P1003), followed by a dot and then the variable or parameter (see "The system variables").

Example: the expression P1003.F6.ANG means that the value of the variable created must contain the value of the angle (ANG) of face number 6 (F6) of the piece called P1003.

Removing the variables from the table

1. Select the variable to be removed from the table.
2. Click on the button.
NOTES:

The removal of "Global" type variables affects the programs those variables are found in. Before removing them therefore, you are advised to always check whether there are programs containing this type of variable.
