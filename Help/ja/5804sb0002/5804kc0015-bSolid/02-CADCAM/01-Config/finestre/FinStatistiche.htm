4. ダイアログボックス 統計

2. ツールバーボタンのリストを含むソフトウェアインターフェイスエリア（コマンドフィールド）。のボタンをクリックします。
3. ボタンをクリックします。
説明

ダイアログボックス 統計で、費用と生産性を管理することができます。

CAM統計値

エリア 加工作業

加工作業を選択して、統計値事前定義/プリセット。要求された情報が入力されていない場合、制御に便利な標準的な情報。を計算することができるようになります。

加工作業を有効にするには、隣のチェックボックスをクリックします。

すべて選択

テーブルにリストされたすべての加工作業を有効にします。

すべて選択解除する

テーブルにリストされたすべての加工作業を無効にします。

計算

エリア時間とコストに指示された値によって、有効にされた加工作業の時間および費用の計算を開始します。

計算結果はテーブル加工作業のリストに表示される 。

合計

計算した加工作業の費用、時間、長さの合計結果を示すウインドウを表示します。

印刷

ダイアログボックスを表示して、作成した統計値を印刷します。

パス内の加工作業を分ける

テーブルに、各形状エレメント特定のプロパティに関連付けられた抽象的なエレメント。点、ライン、面およびスペースはすべて形状エレメントです。で分けられた加工作業のリストを表示します。

例えば、単一のフライス加工が2つの別の形状に適用された場合、加工作業に関連する計算は2つに分けられる：最初の形状に1つのフライス加工を、そして2番目の形状に1つのフライス加工を適用します。

エリア 時間とコスト

作業者の費用や材料など、生産時間および費用（時間または分）に関連する値を指定することができます。

これらのフィールドデータまたはコマンドを含む所定のエリア（データフィールドまたはコマンドフィールド）。で指定されたデータに基づいて、システムは以下のエリアに指示された統計値を作成する：エリア 加工作業のリストおよびエリア ツールリスト。

エリア 加工作業のリスト

機械の費用や機械加工に費やした時間など、生産統計値を表示します。

コラムツールには、加工作業で使用したツールの名前が表示されます。

エリア ツールリスト

費用やツール使用時間など、各加工作業で使用されたツールに関連する統計値データを表示します。

 

CAD統計値

エリア 数量

アクティブなドキュメントに存在する形状エレメントの数量をタイプ分に表示します。

 

エリア 測定して検証

エレメントの寸法や閉じたパスの数量など、アクティブなドキュメントに存在する形状エレメントに適用された計算結果を表示します。

こうした確認をするには、計算ボタンをクリックします。
