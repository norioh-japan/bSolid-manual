Using snap tools on objects (Osnap)

The Osnap (or magnetConnects the point of the new outline to the elements of the existing objects, such as the end of a line, the centre of a circle, the mid-point of a side, etc.) commands bind the cursorMobile element that is visible on the screen, and which moves as the mouse is moved. to a precise point on an objectGeneral term used to define any drawing created in the drawing area (e.g. piece, texts, flat figures, etc.). (e.g. the middle of a circle, the mid-point or end of an object).

When one of the snap tools“Magnet” commands for connecting the various geometric elements: search for mid-points, outline extremities, etc. is active, if the cursor is moved near to a point of an object that the snap refers to, the cursor is attracted to that point. With the snap tools active, the cursor becomes sensitive and links the start point of the new geometric elementAn abstract element associated with specific properties: the point, line, plane and space are all geometric elements. to those object parts activated with the snap tools.

Choosing the snap tools to be kept always active

1. Click(1) Indicates the action of pressing (and immediately releasing) the mouse button on an object (icon, button, etc.) located under the pointer on the screen. (2) (verb) Pressing and immediately releasing the left mouse button to enable the function of a selected command. on the buttonCommand field that is responsive to mouse clicks and that enables operations to be activated. on the status bar.
2. Select the required snap tools (that will turn from grey to black) and save with the Ok button.
The relative iconsGraphic image used to show a data item, button, etc. on the status bar will light up.

Activating the selected snap tools

To make the selected snap tools active, click on the Osnap button on the status bar.

Using snap tools on the grid

The Snap commandTool designed to perform a specific function/operation. For example, the “Print” command allows you to print a document. limits the cursor movement to the snap gaps and the set gridA network of lines. For a detailed description, see the paragraph dealing with the grid. points.

To activate or deactivate it, click on the Snap button on the status bar.

With this command, you can carry out the following operations:

  •    draw, forcing the cursor to move in the gap steps defined in the Snap stepAcronym of "STandard for the Exchange of Product model data". Type of normalised, neutral file used as a graphic data (3D, 2D and wireframe) exchange format between the various CAD and CAM systems. data fieldA defined area containing data or commands (data field or command field).;
  •    align the objects during the copying or movement phase, by linking them to the grid;
  •    rotate the objects, binding them to the corners of the grid.

Orthogonal option

When creating or modifying objects, for purposes of practicality and precision you can limit the movement of the cursor by binding it to the vertical or horizontal axis.

When drawing lines or working on the created objects, the cursor tends to follow the horizontal or vertical axis when the orthogonal command is activated. The directions to be followed are determined by the selected construction planePlane on which a flat geometry is constructed. The default plane is the XY plane..

Locking the cursor

If you press the Alt or TabLabel which, if selected with the cursor, displays a list of data. key of the keyboard while you are drawing, the cross cursor is temporarily abandoned (it is locked) so you can move to the data areaGeneral term to define a specific area for data entry. and change the drawing tool or the parameter values.
To restore the use of the cursor, press the Alt or Tab key again.
