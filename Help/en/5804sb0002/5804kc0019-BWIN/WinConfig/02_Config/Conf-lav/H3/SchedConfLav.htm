Machining operation configuration card

For the description of the machining operation cards found in the data areaGeneral term to define a specific area for data entry., see Machining operations bar.

498. Example of a machining operation card


Fields of the card

Add tag

Add a new machining operation.

Remove tag

Delete the selected machining operation.

Select the machining operation

Used to choose the type of machining operations to be executed:

  •    Boring: bSolid > environment Cad/Cam > Boring
  •    Cuts: bSolid > environment Cad/Cam > Cutting
  •    Milling: bSolid > environment Cad/Cam > Milling
  •    Pocketing: bSolid > environment Cad/Cam > Pocketing / Rough-shaping

In this case, the adjacent data area will show the card with the data of the selected machining operation.

Application side

Side to be machined.

Enable splinter-guard management

Enables (box tickedClick on a check box to tick it and thereby enable that function.) or disables the creation of the splinter-guard machining operation on the corners of the piece.

Apply according to opening

Used to define when to apply the machining operation.

Inside only

The machining operation is only applied to sashes with an outward opening(doors/windows) Type of outward opening of the sash/door, with rotation around the vertical axis and handle on the right-hand upright of the opening sash/door..

Outside only

The machining operation is only applied to sashes with an inward opening(nesting) Slang term deriving from "windows", openings in the walls, and indicates the "openings" in the piece, passthrough machining operation of open and/or closed surfaces..

Element application

Used to define which pieces should be machined.

All

The machining operation is applied to all the pieces of the window.

Lower

The machining operation is only applied to the lower crosspiece(doors/windows) Also: crosspiece. Horizontal part of the window/door. of the window.

All, except lower

The machining operation is applied to all the pieces of the window, apart from the lower crosspiece.

Apply if the content is a sash

Enables (box ticked) or disables the applicationA synonym of "program" - software with the specific task of managing data, graphics, etc. of the machining operation to frames that receive the sash.

Apply if the content is a central panel

Enables (box ticked) or disables the application of the machining operation to frames that receive the central panel(doors/windows) Smooth or machined (faceted) wooden panel for decorating the window..

Apply if the content is glass

Enables (box ticked) or disables the application of the machining operation to frames that receive the glass.

Pass on entire side length

Enables (box ticked) or disables the application of the machining operation to the entire length of the window piece.
