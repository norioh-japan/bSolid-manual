Description

To enter the instruction
1. Click(1) Indicates the action of pressing (and immediately releasing) the mouse button on an object (icon, button, etc.) located under the pointer on the screen. (2) (verb) Pressing and immediately releasing the left mouse button to enable the function of a selected command. on the card"Machining operation timeline" .
2. Position the cursorMobile element that is visible on the screen, and which moves as the mouse is moved. between two machining operations.
3. Open the functions bar.

4. Click on the buttonCommand field that is responsive to mouse clicks and that enables operations to be activated..
5. Enter the ISO instruction between quotation marks in the relevant data fieldA defined area containing data or commands (data field or command field)..

Tab Advanced

Name

NMC

Function name.

The text entered in this field will rename the function text string (the one near the iconGraphic image used to show a data item, button, etc. of the chosen tool) to distinguish it from the others in the "Geometries and machining operations" tree list. If this field is left empty, the system will automatically enter the default text.

Execute machining

EXE

Allows the user to enter a condition for a function to be performed or not.

Insert a formula or choose one of the two options:

  •    False: do not carry out.
  •    True: carry out.
Save Default

Used to save the data set as default - always shown when using the specific function.

Instructions allowed

  •    Comments= recognisable by the semicolon, e.g. “;milling”.
  •    Skip line = “JM “ or “JP”.
  •    End of program instructions = “%%”.
  •    Labels = recognisable by the colon, e.g. “:<label>”.
