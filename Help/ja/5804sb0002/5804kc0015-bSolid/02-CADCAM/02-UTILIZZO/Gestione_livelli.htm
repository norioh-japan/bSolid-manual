レイヤーとは

レイヤー描画を行うために作成されたドキュメントのレベル。（レベル）ツールは、描画されたオブジェクトをまとめ、他から独立して管理することができるグループ内にそのオブジェクトを並べるために使用されます。

レベルの使用は、数多くの形状および/または加工作業を含む複雑なドキュメントを作成する際に、作業を円滑に進め正しくまとめるのに役に立つ。例えば、様々なレベルにまたがって作業が分配され、描画の特定のエレメント（例えば、他の上部に位置するエレメントなど）のみを修正したい場合、修正するオブジェクトに関わるもの以外のすべてのレベルをロックすることができます。

各レベルは、以下の特性を描画したオブジェクトに割り当てる。

  •    ベースの線の色
  •    線のタイプ（連続線、点線など）
  •    線の太さ
  •    オブジェクトの表示またはその他
  •    オブジェクトのロック（レベルをロックする/解除するも参照）
  •    面の材料。

描画したオブジェクトの特性には、オブジェクトが属するレベルに関係なく修正できるものがあります。例えば、レベルにある特定のオブジェクトの色および表示など。

描画するレベル

描画する時は、作成されたオブジェクトは自動的にアクティブなレベル（標準バーのドロップダウンメニューの前景にあるレベル）に割り当てられる。

このドロップダウンメニュードロップダウンボタンをクリックすると、垂直または水平方向に広がるオプションまたは手段のリスト。で描画のレベルを選択する、または異なったレベルを選択したオブジェクトに割り当てることができます。

  •    描画のレベルを選択するには、標準バーのドロップダウンメニューサブメニュー（特定の機能/作業を実施できるようにするオプションのリスト）を生成するコマンド（マウスで選択する）のリスト。をクリック（1）画面上のポインタの下にあるオブジェクト（アイコン、ボタンなど）の上でマウスボタンを押す（そしてすぐに離す）行為を示します。（2）（動詞）選択したコマンドの機能を有効にするため、マウスの左ボタンを押してすぐに離します。して、提供されたリストからレベルを選択します。
  •    オブジェクトを1つのレベルから別のレベルへ移動するには、オブジェクトを選択して標準バーのドロップダウンメニューをクリックして、提供されたリストからレベルを選択します。

レベルをロックする/解除する

その中にあるオブジェクトが修正されないようにレベルをロックするには、標準バーの開いたパッドロックをクリックします。そうすると、パッドロックが閉じます。

閉じたパッドロックは、レベルがロックされたことを示します。開いたパッドロックは、レベルが解除されたことを示します。

新規レベルを作成する

1. ステータスバーのボタンをクリックします。レベル管理ダイアログボックスが開きます。
2. 新規...ボタンをクリックします。
3. レベルの名前および開いたダイアログボックスのレベルのオブジェクトの特性を特定する
。Aコマンド特定の機能/作業を実施するために設計されたツール。たとえば、「Print」コマンドではドキュメントを印刷できます。を使用して、材料の特性を特定する。ボタンマウスのクリックに対応し、実施する作業を有効にするコマンドフィールド。Bをクリックして設定を確定します。
4. OKボタンをクリックして、新規レベルの作成を確定しダイアログボックスを閉じます。

レベルを修正および削除する

修正する
1. ステータスバーのボタンをクリックします。レベル管理ダイアログボックスが開きます。
2. 修正するレベルに対応するテーブルの行を選択し、ボタンをクリックします。変更する
3. 修正ダイアログボックスで必要な修正を行う。ボタンBをクリックして設定を確定します。
4. ボタンOKをクリックして、レベルのテーブルにある修正を確定し、ダイアログボックスを閉じます。
削除する
1. ステータスバーのボタンをクリックします。レベル管理ダイアログボックスが開きます。
2. テーブルから、削除するレベルに対応する行を選択し、ボタン削除をクリックして確定します。
3. ボタンOKをクリックして、レベルのテーブルからレベルの削除を確定し、ダイアログボックスを閉じます。
