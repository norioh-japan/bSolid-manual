4. Dialogue box Statistics

Description

The dialogue box Statistics allows to control the costs and productivity.

CAM statistics

Area Machining operations

Allows to choose the machining operations to enable to calculate the statistics.

To enable the machining operation, click on the adjacent check box.

Select all

Enables all machining operations listed in the table.

Deselect all

Disables all machining operations listed in the table.

Calculate

Starts the calculation of the times and costs for the machining operations enabled according to the values indicated in area Time and costs.

The calculation results are displayed in the table List of machining operations.

Total

Displays the window with the total results of the costs, times and length of all machining operations calculated.

Print

Displays the dialogue box to print the statistics generated.

Separates machining operations in paths

Displays in the table the list of machining operation separated by individual geometric elements.

For example, if a single milling was applied for two separate geometries, the calculation related to that machining operation is divided into two: one milling for the first geometry and one milling for the second geometry.

Area Time and costs

Allows to define the values relative to the production time and costs (hours or minutes): cost of the operator, the material(nesting) Type of material to be associated with the articles and sheets of the nesting project., etc.

Based on the data defined in these fieldsA defined area containing data or commands (data field or command field)., the system generates the statistics indicated in the areas below: area List of machining operations and areaTools list.

Area List of machining operations

Displays production statistics: cost of the machine, the time spent machining, etc.

Column Tool shows the name of the tool used in the machining operation.

Area Tools list

Display the statistical data related to the tools used in each machining operation: costs, tool usage times, etc..

 

CAD statistics

Area Quantity

Shows the amount of geometric elements present in the active documentElectronic worksheet on which the geometric and technological parameters are inserted., divided by type.

 

Area Measure and verify

Display the results of the calculation applied to the geometric elements present in the active document: element dimensions, amount of paths closed, etc..

To perform this kind of verification, click the button Calculate.
