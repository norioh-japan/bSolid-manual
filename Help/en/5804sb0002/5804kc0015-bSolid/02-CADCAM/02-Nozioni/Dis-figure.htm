Drawing flat figures

To draw flat figures, proceed as follows:

1. Click on the required buttonCommand field that is responsive to mouse clicks and that enables operations to be activated. in the drawing bar.
2. Click on the point where you want to start the geometry. To define the type of co-ordinates, use the specific tools in the data areaGeneral term to define a specific area for data entry..

You can temporarily lock the cursorMobile element that is visible on the screen, and which moves as the mouse is moved. to enter the values in the parameters areaPart of the data area with the cards for managing the geometric and technological parameters. (see Locking the cursor).

3. Click on the point where you want to end the geometry.

To obtain greater drawing precision, you can use the specific snap tools“Magnet” commands for connecting the various geometric elements: search for mid-points, outline extremities, etc. of the status bar: Using snap tools on the grid; Orthogonal option; Using snap tools on objects (Osnap).
