Export the work table tooling

Enables (box ticked) or disables the export of data relating to the position of the suction cups and mobile panel supports.
In this case, the processed ISO file contains the section with the positions of all the work table elements.

Export the tool list

Enables (box ticked) or disables the export of data relating to the tools in the databaseData bank. Stored data arranged in an organised manner so that they are readily available using a search and management program..
In this case, the processed ISO file also contains the section with the list of tools used.

Activation of the levelling cycle

Enables (box ticked) or disables the support panelWooden panel positioned beneath the piece to be machined, in order to protect the work table objects from any through machining operations. levelling cycle.

Raise stops on vacuum

This field is only visible on machines equipped with auxiliary stops. Enables (box ticked) or disables the use of the auxiliary stops.

Magazine tooling configuration

This field is only visible on machines equipped with a tool magazine"Multistore".
It is used to insert the number corresponding to the tooling configuration of the tool magazine "Multistore" (from 1 to 99) saved previously. This means you can recall the tooling of this magazine in the active program.
