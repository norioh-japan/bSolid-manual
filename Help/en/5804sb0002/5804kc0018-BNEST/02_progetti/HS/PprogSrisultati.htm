Results

Working lists creation mode

Used to choose how to generate the work lists.

Divides identical layouts

Used to choose how to generate the files of the nesting(nesting) Arranging several shapes on a sheet so they are perfectly interlocked. project processing results.

In case of labelling only

Only if the panels/sheets have to be labelled, the system creates the same number of files as the number of sheets generated in the results. Each sheet is a machining programType of document (file) containing all the instructions needed for machining the pieces of wood, glass or marble..

Always

The system always creates the same number of files as the number of sheets generated in the results. Each sheet is a machining program.

Never

When the system generates programs, if many identical patterns are found, it produces a single program for them that will then be machined the amount of times necessary to realize all the pieces inserted in these patterns.

Loading origins list for all the sheets

Reference origins of the work table to be enabled on the machine in order to push the panel/sheet to be machined.

Enter the origins separated by a comma.

Short sheets maximum width

Maximum dimension in Y of the nesting program panel/sheet in order to be considered small and therefore use the management of the origins defined in the data fields "Loading origins list for short sheets".

Loading origins list for short sheets

Reference origin of the work table for pushing the small sized panel/sheet.

Enter the origins separated by a comma.

Work origins modification for the small rests only

Enables (box ticked) or disables the modification of the work table's reference origins for small sized remainders only.
