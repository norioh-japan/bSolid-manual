Cutting

Description of the machining operation

Generates horizontal and vertical cuts on the surfaceFigures of a three-dimensional area, that represent a thin open or closed structure. of the piece.

The programmable machining operations are:

  •    Cuts from a geometry (i.e. they can be applied to the two-dimensional geometries created on the faces of the piece).
  •    Repeated cuts, starting from the position of the first programmed cut.

Reverse travel

RV

Used to reverse the tool advance direction. The tool for performing the machining operation does not begin at the start point of the programmed geometry, but at the end point.

Activates the inversion. Inverts the construction direction of the programmed geometry.

Deactivates the inversion. Restores the original construction direction (inverted at the starting stage).

Blade correction

BRC

Enables or disables the correction of the blade radius as it enters and leaves the piece. The CAMAcronym of "Computer Aided Manufacturing". Type of software allowing the use of a CAD to control and manage the functioning of the operating machines. A CAD/CAM system enables pieces to be designed, drawn and machined. performs a series of calculations, allowing the blade to start and end the machining operation at the programmed points.

If you click on the first button, the blade will make the cut by descending onto the machining start point with its lower tip A (figure 6). If you click on the second button, the blade will make the cut by descending onto the machining start point with its side edge B (figure 6).

Two-directional passes

BDR

Authorises (box ticked) the tool to work in alternate directions when making the next movements. The tool changes direction every time it makes a new movement. This function is enabled by clicking on the box.

NOTES:

This function is not taken into consideration if you have programmed a through operation divided into several movements, because the blade is inverted to make the last cut anyway, regardless of whether this field is enabled or not.

End pass

Excess material

OVM

Excess material(nesting) Type of material to be associated with the articles and sheets of the nesting project. that you want to leave during the cut. The valuePredefined / pre-set. Standard information useful for the control, if the requested information is not entered. indicated in this field is linked to the type of correction defined in the "Tool radius correction" field.

  •    With left or right correction, when a negative value is set the tool removes more material, and when a positive value is set the tool will remove less material.
  •    With a central correction, when a negative value is set the machine operation is moved sideways to the right of the programmed trajectory and when a positive value is set the machining is moved to the left of the programmed trajectory.
