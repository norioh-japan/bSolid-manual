Angular position setting

Description

Inserts an angular position between the two elements.

101. Example

A. Centre
B. Start point
C. End point
D. Distance

Centre

X

X1

X co-ordinate of the central position-setting point (ref. A, figure 1).

Y

Y1

Y co-ordinate of the central position-setting point (ref. A, figure 1).

Z

Z1

Z co-ordinate of the central position-setting point (ref. A, figure 1).

Start point

X

X2

X co-ordinate of the first position-setting point (ref. B, figure 1).

Y

Y2

Y co-ordinate of the first position-setting point (ref. B, figure 1).

Z

Z2

Z co-ordinate of the first position-setting point (ref. B, figure 1).

End point

X

X3

X co-ordinate of the second position-setting point (ref. C, figure 1).

Y

Y3

Y co-ordinate of the second position-setting point (ref. C, figure 1).

Z

Z3

Z co-ordinate of the second position-setting point (ref. C, figure 1).

Dimensioning parameters

Distance

DST

Distance of the position from the central point, to move it closer or further away (ref. D, figure 1)

Text

TXT

Shows the valuePredefined / pre-set. Standard information useful for the control, if the requested information is not entered. of the measured angle. The text box can be modified (if the "Enable parametric mode" box has been enabled, you can enter a variable or a simple text).
