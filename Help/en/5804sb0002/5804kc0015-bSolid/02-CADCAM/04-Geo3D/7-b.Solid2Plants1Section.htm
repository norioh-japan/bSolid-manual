Steps for creating a surface
1. Select the piece face on which you want to draw the first geometry.

2. Select the piece face on which you want to draw the second geometry.

3. Select the piece face on which you want to draw the open outline that must touch both the closed geometries.

4. Click(1) Indicates the action of pressing (and immediately releasing) the mouse button on an object (icon, button, etc.) located under the pointer on the screen. (2) (verb) Pressing and immediately releasing the left mouse button to enable the function of a selected command. on the buttonCommand field that is responsive to mouse clicks and that enables operations to be activated. in the drawing barBar listing the drawing tools..
5. Enter the first geometry in the "First section" data fieldA defined area containing data or commands (data field or command field)..

6. Enter the second geometry in the "Second section" data field.

7. Enter the outline in the "Path geometry" data field.

8. Use the specific buttons of the data areaGeneral term to define a specific area for data entry. to close or open the surface.

9. Click on the "Ok" button to generate the surface.

Mode

Opening

OB

Allows the user to choose whether to create a three-dimensional closed or open figure above, below, or on both sides.

Open

Three-dimensional open figure (without upper and lower face). In this case, an open surface is generated.

Closed

Three-dimensional closed figure. In this case, a closed surfaceThree-dimensional figure composed of many surfaces in order to generate a closed, empty object (e.g. a cube). is generated.

Open below

Three-dimensional figure without lower face. In this case, an open surface is generated.

Open above

Three-dimensional figure without upper face. In this case, an open surface is generated.

Accuracy

PR

Index to define the precision of the three-dimensional figure created. Values lower than 1,000 make the object more precise.
