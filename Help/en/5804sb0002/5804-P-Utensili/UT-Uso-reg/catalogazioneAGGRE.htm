Creating the aggregates

1. Click on the button to visualise the dialogue boxNew aggregate.
2. Choose the mode type in the Creation mode data fieldA defined area containing data or commands (data field or command field)..
3. Enter the code or name of the aggregate in the relevant data field.
4. Fill in the data fieldsA defined area containing data or commands (data field or command field).. (Compilation will vary according to the type of aggregate to be registered).

The aggregate created will be added to the databaseData bank. Stored data arranged in an organised manner so that they are readily available using a search and management program. table.

Creating aggregates via a copy

1. Select the table row with the name of the aggregate to be copied.
2. Click on the button.
3. Enter the name in the relevant data field of the dialogue box.
4. Click on the button to save the settings and close the dialogue box.
5. Fill in the data fields.

Deleting aggregates

To delete an aggregate from the table, select it then click on the button and confirm.

Renaming aggregates

To rename an aggregate in the table, select it then click on the button.
Enter the new name in the Name data field, then confirm with the button.

Searching and filtering aggregate

1. Click on the button to visualise the Search for aggregate dialogue box.
2. Fill in the relevant field on the basis of the type of search you want to carry out.
3. Click on the button to save the settings, then close the dialogue box and start the search.

Only the filtered aggregates will be shown in the table.

Tooling the active aggregate

Open the aggregate to be tooled, then proceed as follows:

1. Select the spindle to be tooled from the "Subspindles" tabLabel which, if selected with the cursor, displays a list of data..
2. Click twice on the iconGraphic image used to show a data item, button, etc. of the tool in the relevant area at the bottom.
