The colours of the graphics

Yellow

Object highlighted by passing over it with the cursorMobile element that is visible on the screen, and which moves as the mouse is moved..

Blue

Object selected by clicking on it with the cursor.

Red

Object in danger: possible collision between machine objects.

For example, the tool-holders turn red when you select a tool with a diameter that exceeds the limits defined in the overall tool-holder dimensions.

Tree list showing machine parts

Right-hand area, containing the icons of the various machine parts which, if selected, show the list of sub-elements: tool-holders and spindles.

This area interacts with the image of the graphics area. By clicking on an object icon, you can see the list of the parts that form it and view the detail in the graphics area.

To highlight (in the graphics area) the object shown in the tree list, pass over the icon with the mouse without clicking; the icon will turn orange and the relative object of the graphics area will turn yellow (ref. B, figure 2).

To select a tool-holder or spindle, use the cross cursorType of cross-shaped cursor. or click(1) Indicates the action of pressing (and immediately releasing) the mouse button on an object (icon, button, etc.) located under the pointer on the screen. (2) (verb) Pressing and immediately releasing the left mouse button to enable the function of a selected command. on the tree list icon; the selection will turn blue (ref. A, figure 2).

Tool area

Defined area in the lower part of the environment, containing the cards with the list of all the tools to be installed: mills, bits, aggregates, deflectors, etc.
