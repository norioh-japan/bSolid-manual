# ポリゴン ![ポリゴンアイコン](../../../FIGURE/15-icone/b15b0001/Draw_Polygons.png)

## 説明

正多角形（正三角形、ひし形、5角形、6角形など）または星形のポリゴンを、構成円の内側（内接）または外側（外接）に作成します。

辺の数および構成円周半径によって、ポリゴンの形および大きさは変わる。

## ポリゴンを描画するステップ

1. **描画バー**の![ポリゴン](../../../FIGURE/15-icone/b15b0001/Draw_Polygons.png)ボタンをクリックします。

2. ポリゴンのタイプデータフィールドの2つのツールのうちの1つをクリックすることによってポリゴンのタイプを選択します。

![ポリゴン作成手順](../../../FIGURE/10-videateComplete/arte4/b10b0404.gif)

3. 構成のタイプデータフィールドの2つのツールのうちの1つをクリックすることによってポリゴンをどう構成するかを選択します。

4. 描画エリアでクリックすることによってポリゴンの中心点を挿入します。カーソルをドラッグしてポリゴンを作成します。

![ポリゴン描画プロセス](../../../FIGURE/10-videateComplete/arte4/b10b0405.gif)

5. ポリゴンを終了したい点をクリックします。

6. **取り消す**ボタンをクリックして**ポリゴン**カードを閉じます。

## 一般的パラメーター

### ポリゴンのタイプ
**TYP**

ポリゴンのタイプを選択するために使用されます。

#### ![正ポリゴン](../../../FIGURE/15-icone/b15b0001/Draw_Polygon_Polygon.png) 正の
正多角形（正三角形、5角形、6角形など）を作成します。

#### ![星形ポリゴン](../../../FIGURE/15-icone/b15b0001/Draw_Polygon_Star.png) 星形
星形のポリゴンを作成します。

### 構成のタイプ
**POT**

ポリゴンの寸法を指定するための構成のタイプを選択するために使用されます。

#### ![外半径](../../../FIGURE/15-icone/b15b0001/Draw_Polygon_Radius.png) 外半径を含む
外接した構成円の中心点の半径および寸法に基づいて、ポリゴンを作成します。図1を参照。

#### ![内半径](../../../FIGURE/15-icone/b15b0001/Draw_Polygon_Apothema.png) 内半径を含む
外接した構成円の半径および辺心距離（ポリゴン内部の三角形の高さ）に基づいて、ポリゴンを作成します。図2を参照。

### 辺の数
**NS**

作成する辺の数。

### 半径
**R1**

### 内部半径（辺心距離）
**AP**

ポリゴンの内部の三角形の高さ。

正多角形の構成では（図1）、ポリゴンそのものの辺心距離、つまり内部の三角形の高さを表す。

星形のポリゴンの構成では（図2）、星形のポリゴンに内接する構成多角形の内部の三角形の高さ（辺心距離）を表す。

**R**

中心点から始まる、XY構成平面上のポリゴンの回転角度。

**図3. 正多角形**

![正多角形](../../../FIGURE/08/arte4/b08b0236.png)

a. 角度（[回転](#回転)）

### 面取りのタイプ
**CT**

選択した形状の鋭角を面取りし、円弧または直線を挿入するために使用されます。

#### ![面取り無し](../../../FIGURE/15-icone/b15b0001/Draw_Rectangle_Edge.png) 面取り無し
鋭角は面取りしない。

#### ![円形面取り](../../../FIGURE/15-icone/b15b0001/Draw_Rectangle_Fillet.png) 円形の面取り
曲線状の面取りを挿入する。

#### ![直線面取り](../../../FIGURE/15-icone/b15b0001/Draw_Rectangle_Chamfer.png) 直線の面取り
真っすぐな面取りを挿入する。

### 面取りの値
**CD**

**円弧の面取り![円形面取り](../../../FIGURE/15-icone/b15b0001/Draw_Rectangle_Fillet.png)：**

2つの連続する接線でないエレメントに丸みを付けるために使用される円弧の半径。

**直線の面取り![直線面取り](../../../FIGURE/15-icone/b15b0001/Draw_Rectangle_Chamfer.png)：**

2つの連続する接線でないエレメントの角からの距離 **D**
