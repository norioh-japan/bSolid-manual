バキュームモジュールの起動

説明

ツリーリストノードと呼ばれる複数の項目を含む構造。木のように分岐（枝）や二次分岐があります。最も高レベルにあるノードはルートと呼ばれます。そのたすべては単独のノードから派生し、他のノードを持つことができます。で "加工作業のタイムライン"、特定の成形可能な吸引カップでバキュームの使用を有効/無効にするための指示を入力します。

指示を入力するために
1. カードをクリック（1）画面上のポインタの下にあるオブジェクト（アイコン、ボタンなど）の上でマウスボタンを押す（そしてすぐに離す）行為を示します。（2）（動詞）選択したコマンドの機能を有効にするため、マウスの左ボタンを押してすぐに離します。する"加工作業のタイムライン"。.
2. 2つの加工作業の間にカーソル画面上に表示される可動エレメントであり、マウスの移動に合わせて移動します。を当てる。
3. 機能バーを開きます。

4. ボタンマウスのクリックに対応し、実施する作業を有効にするコマンドフィールド。 をクリックします。

主要データ

名前

NAME

有効にされる成形可能な吸引カップのインデックス、吸引カップが置かれるモバイルパネル・サポートのインデックス。引用符の間に入力されます（例えば、"1.2"は、最初のモバイルパネル・サポートのクランプの番号2）。

値

VALUE

次のオプションの1つを選択します：

非アクティブ

バキューム無効。

アクティブ

バキューム有効。

タブ より高度な

名前

NMC

機能 の名前。

このフィールドデータまたはコマンドを含む所定のエリア（データフィールドまたはコマンドフィールド）。に入力されたテキストは、"形状および加工作業"ツリーリストの中で他のものから区別するために機能のテキスト文字列（選択されたツールのアイコンの近くにあるもの）の名前を変更する。このフィールドが空になったら、システムは自動的にデフォルトのテキストを入力する。

機械加工を実行する

EXE

機能を実行するかどうかの条件をユーザーが入力できるようになる。

式を挿入するまたは2つのオプションのうち1つを選択する：

  •    False: 実行しない。
  •    True：実行する。
デフォルトの保存

デフォルトとして設定されたデータを保存するために使用され、特定の機能を使用するときは常に表示されます。
