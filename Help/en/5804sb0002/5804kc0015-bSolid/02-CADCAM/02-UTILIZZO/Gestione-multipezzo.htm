Managing the multi-piece

Description

The multi-piece allows you to programme the machining operations that can be applied to several pieces, saving them in a single documentElectronic worksheet on which the geometric and technological parameters are inserted..

In this case, only one ISOAcronym of the "International Standardisation Organisation", that determines technological and industrial standards. File format containing the programming language of the numerical control machine. file is generated and the pieces in the program must all be loaded on the work table so they can be machined together.

How to design a document

A multi-piece document can be designed in two different ways:

  •    Creating a single piece
  •    Creating several pieces
Creating a single piece

Inserting machining operations on several pieces

There are various methods for inserting machining operations.

Parametric concepts

If you use the multi-piece function, by parameterising the program you can use not only the personalised variables but also the piece parameters, applying them to the other pieces.

For example:

The dimensions of piece 1, called "P1001" , are 600, 400 and 20mm.

To ensure all the pieces are identical to the first one, and adapt to any variations in its measurements, insert the variables of the first piece in the following data fields of the other ones:

Piece length

P1001.LPX

Piece width

P1001.LPY

Piece thickness

P1001.LPZ

The same holds true for all the parameters of the geometries and machining operations, which can be used anywhere. The important thing is that they are followed by the name of the reference piece (e.g. P1001, for the parameters to be linked to piece 1).

Multi-piece simulation

The document created with the multi-piece function brings together the machining operations of all the pieces in a single file, so the simulation in the Cad/Cam environment includes all the programmed pieces.

The implementation of the CAM optimisations takes into account all the machining operations applied to all the pieces, so as to minimise the time taken by the machine.
