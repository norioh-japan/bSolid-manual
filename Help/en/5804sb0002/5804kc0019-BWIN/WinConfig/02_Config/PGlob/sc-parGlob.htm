Global parameters

Enable surfaces

Enables (box tickedClick on a check box to tick it and thereby enable that function.) or disables the calculation of the surfacesFigures of a three-dimensional area, that represent a thin open or closed structure. in the bWindows environment, and therefore the visualisation of the shaped (i.e. machined) pieces.

If this box is disabled, the pieces are shown in their rough form, without the shape of the machining operation.

Lead-in angle

For the description, see the data fields Angle of the machining operation inputs of environment Cad/Cam ( bSolid > environment Cad/Cam> Angle ).

This value is only applied to window elements shorter than the value defined in the "Maximum element length for lead-in for machining operations on head" field.

Percentage lead-in length

For the description, see the data fields Percentage radius of the machining operation inputs of environment Cad/Cam ( bSolid > environment Cad/Cam> Percentage radius).

Execute the inner side first when machining arcs

Indicates (box ticked) that the sides that can be machined first are the internal sides of the arc-shaped crosspiece(doors/windows) Also: crosspiece. Horizontal part of the window/door..

Limit angle for considering the joints as tangent

Minimum angle between the upright and the crosspiece, beyond which the system applies a tangent joint.

Minimum length of the unprocessed piece for the elements

Minimum length of the rough pieces in order to be managed by the loader/unloader device.

Dowel default length

Default length of the dowels to be applied to the Dowels length data field in the dowel configuration tabLabel which, if selected with the cursor, displays a list of data. (see Dowel area).

Enable export of glass geometries

Enables (box ticked) or disables the creation of the file (bSolid) with the glass geometries during the processing of the window project.

Enables the clamping algorithm log

Enables (box ticked) or disables the creation, in the log fileA file in which the activities and/or operations performed by, for example, an application, a server, a command interpreter, etc. are recorded., of errors concerning the automatic clampPressure exerted by the presser roller to hold the edgebanding strip in place. positioning calculation.

Enable smart deflector

Enables (box ticked) or disables the management of the deflector so that it can deflect the flow of chips toward the suction hood in the best manner possible.
