Origins

In this card, you can make associations between origins in order to perform the machining operations in pendularA machining operation performed on one part of the work table while the piece loading/unloading operations are taking place on the opposite side of the table (to reduce machining times). mode.

In each text box, enter the origin to be associated with the one indicated in the fieldA defined area containing data or commands (data field or command field). name itself.

420.

Support

In this card, you can define the limits that may be reached by each single mobile panel support.

Lower stroke limit of mobile panel support 1

Used to define the minimum position that may be reached by the individual support. Define this position for each support. These data are only visible for the ATS work table.

Upper stroke limit of mobile panel support 1

Used to define the maximum position that may be reached by the individual support. Define this position for each support. These data are only visible for the ATS work table.

Carriages

In this card, you can define the lower and upper limits that may be reached by each single carriage.
