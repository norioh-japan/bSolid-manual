Description

The Optionsdialogue box is used to define the configuration data of the Cad/Cam environment, and certain default data of the applicationA synonym of "program" - software with the specific task of managing data, graphics, etc. itself (language, etc.).

NOTES:

The set data are those defined as default data. Their modification by untrained personnel could lead to problems.

CAD parameters

Gap between tangency angles

Used to define the minimum and maximum values of the angles within the geometries, to establish whether two parts are tangent to each other.

Maximum continuity distance

Used to define the default values of the distance beyond which two parts are no longer considered when checking continuity.

Maximum length of small parts

Used to define the length below which the geometric parts are considered "small".

Accuracy of estimates

Used to define the default valuePredefined / pre-set. Standard information useful for the control, if the requested information is not entered., in order to make the calculations more accurate. A lower value makes the calculation more accurate.

Solidification precision

Used to define the default value for establishing the precision of the transformation of two-dimensional objectsGeneral term used to define any drawing created in the drawing area (e.g. piece, texts, flat figures, etc.). into surfacesFigures of a three-dimensional area, that represent a thin open or closed structure.. A lower value makes the transformation more precise.

Display

Visualise skeleton

The active button is used to enable mesh displayType of visualisation showing the skeleton of the three-dimensional figures: hidden internal lines and arcs, and profile. (see also "Mesh visualisation ").

Visualise photorealistic

The active button is used to enable photorealistic displayType of display that gives three-dimensional figures a realistic appearance, with colours and shading. (see also "Photorealistic visualisation ").

Surface finishing

Enables (box tickedClick on a check box to tick it and thereby enable that function.) or disables the accurate graphic restitution of the surfaces in the drawing areaArea dedicated to the creation and modification of the geometries/machining operations..

Profile shading

Enables (box ticked) or disables the shading of surface profiles in the drawing area.

Activates the raytrace lights

Enables (box ticked) or disables the visualisation of the lights and shadows on surfaces in the drawing area.

Set upper background

Used to define the background colour for the drawing area. Click on the button, choose the colour from the colour palette, then confirm.

Set lower background

Used to define the background colour for the drawing area. Click on the button, choose the colour from the colour palette, then confirm.

Setting the material properties

Used to define the default materials (applicable to the drawing level), for the realistic visualisation of the surfaces and selections, and for the machining operations. Choose the required material(nesting) Type of material to be associated with the articles and sheets of the nesting project. from the drop-down menuList of options or instruments which expand vertically or horizontally when you click on the drop-down buttons..
The buttons " Set standard material", " Set selection " and " Set machining operation" beneath the drop-down menuList of commands (to be selected with the mouse) that develop a sub-menu, i.e. a list of options that allow certain functions/operations to be carried out. open the Materials management dialogue box.

6. Dialogue box Materials management

Used to enter, modify or delete all the materials to be machined (that can be selected from the drop-down menu).

Material name

Used to select the material to be modified, or enter the name of the new material.

Insert

Enters the new material in the archive.

Modify

Modifies the material selected from the list.

Delete

Deletes the material (selected from the list) from the archive.

Exit

Closes the dialogue box.

Visualisation data

Shows the data fieldsA defined area containing data or commands (data field or command field). for modifying (via the slide bars) the visualisation characteristics of the materials.

Image

Shows the system windowA window for navigating within the Windows operating system. for choosing the bitmapGraphic format representing the images as a series of dots (pixels). Bitmaps are created in various file formats (GIF, JPEG, BMP etc.), and can be read by various graphics programs (Photoshop, Paint etc.). image representing the material.

Standard material

Shows the graphics of how the drawing object is coloured.

Material selection

Shows the graphics of how the selected drawing object is coloured.

Machining operation material

Shows the graphics of how the machining operation is coloured.

Set standard material

Shows the dialogue boxMaterials management, to set the colour of the drawing objects.

Set selection

Shows the dialogue boxMaterials management, to set the colour of the selected drawing objects.

Set machining operation

Shows the dialogue boxMaterials management, to set the colour of the machining operations.

Visualise icon with axes

Enables (box ticked) or disables the visualisation - in the drawing area - of the iconGraphic image used to show a data item, button, etc. representing the Cartesian axesThe three straight lines (x, y and z) used to associate the point of the table with its co-ordinates. of the construction planePlane on which a flat geometry is constructed. The default plane is the XY plane..

Curve accuracy

Used to define the default value for establishing the visualisation accuracy of the curves. A lower value makes the visualisation more precise.

Number of decimal figures

Predefined number of the decimal figures displayed.

Position-setting styles

D.  Delete the position-setting style (selected from the list) from the archive.

Used to choose a position-setting style (from those in the archive) from the drop-down menu.

Arrow

Arrow height

Character
