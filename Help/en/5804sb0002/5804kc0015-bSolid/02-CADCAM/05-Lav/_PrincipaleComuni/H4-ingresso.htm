Lead-in

In this section, you can define how the tool should enter the piece.

The fieldsA defined area containing data or commands (data field or command field). are set automatically (with predefined values) after you have selected a tool. These fields may vary according to the type of tool chosen and the type of machining operation.

Type

Movement performed by the tool in order to enter the piece to be machined.

Choose the desired input type from the drop-down menuList of options or instruments which expand vertically or horizontally when you click on the drop-down buttons..

None

The tool descends at right angles, and directly arrives at the machining start point.

738. None

Line

The tool descends by following a line perpendicular to the face. It enters the piece and reaches the machining start point via a short linear path to correct its position.

739. Line



Arc

The tool descends by following a line perpendicular to the face. It enters the piece and reaches the machining start point via a short arc path to correct its position.

740. Arc



Line-arc tangent

The tool descends by following a line perpendicular to the face. It enters the piece and reaches the machining start point by forming a tangent line and arc. The correction is made along the line and arc.

741. Line-arc tangent



3D line-arc

The tool descends by following a tilted line. It enters the piece and reaches the machining start point via a short arc path to correct its position.

742. 3D line-arc



Helix

The tool generates a line in the air, parallel to the face along which it makes the correction. It descends along a spiral path and enters the piece, directly reaching the machining start point.

743. Helix



3D line

The tool generates a line in the air, parallel to the face along which it makes the correction. It descends along a tilted path and enters the piece, directly reaching the machining start point.

744. 3D line



3D arc

The tool generates a line in the air, parallel to the face along which it makes the correction. It descends along an arc path and enters the piece, directly reaching the machining start point.

745. 3D arc



Corrected line

The tool generates a line in the air, parallel to the face along which it makes the correction. It descends along a line perpendicular to the plane and enters the piece, directly reaching the machining start point.

746. Corrected line



Follow 3D profile

The tool descends along a tilted line and enters the piece, directly reaching the machining start point. The use of this type of entrance is only allowed with closed profiles.

747. Follow 3D profile



Element extension

Used to bring forward the entry of the tool in the piece. The tool descends by following a line perpendicular to the face. It enters the piece in the point established in this fieldA defined area containing data or commands (data field or command field)., then reaches the machining start point by correcting its position. Indicate the distance in the "Section length" data field.

748. Element extension



3D element extension

Used to bring forward the entry of the disc-shaped tool in the piece.
The tool descends by following a line perpendicular to the face. It enters the piece in the point established in this field, then reaches the machining start point by correcting its position. Indicate the distance in the "Section length" data field.

Correction in air

Enables (box tickedClick on a check box to tick it and thereby enable that function.) or disables the correction of the tool position while it is moving, before it descends towards the piece.

Angle

 

Tool lead-in angle. The type of angle to be entered varies according to the type of lead-in previously selected.

749. Angle

A.  Lead-in angle (ang) for line-type lead-in.
B.  Lead-in angle (ang) for arc-type lead-in.

For the "3D line-arc" type lead-in, the arc opening(nesting) Slang term deriving from "windows", openings in the walls, and indicates the "openings" in the piece, passthrough machining operation of open and/or closed surfaces. angle must be taken into consideration.
For "Helix" type lead-in, the angular width of the helix must be taken into consideration.

Section length

Linear length of the segment for bringing forward the entry of the tool in the piece in order to correct its position.

Percentage radius

Percentage for modifying the radius of the arc created by the tool when it approaches the piece.

Insert tabbing

Enables (box ticked) or disables the linear transfer of the tool as it enters the piece, in order to create a small stepAcronym of "STandard for the Exchange of Product model data". Type of normalised, neutral file used as a graphic data (3D, 2D and wireframe) exchange format between the various CAD and CAM systems.. This transfer is calculated on the programmed machining depth.

If the box is ticked, the following data fields are shown: "Tabbing length"; "Tabbing position"; "Variance".

Tabbing length

Linear length of the segment for transferring the tool.

• See ref. C, figure 13.
Tabbing position

Position, in mm, for determining the transfer start point, calculated starting from the Z co-ordinate of the milling start point.

• See ref. B and P, figure 13.
Variance

Distance in millimetres from the lead-in point of the machining operation for transferring the tool so that it descends with a slight tilt.

  •    3D line
  •    3D arc
