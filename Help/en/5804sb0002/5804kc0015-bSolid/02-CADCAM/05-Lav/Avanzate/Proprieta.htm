Execute machining

EXE

Allows the user to enter a condition for a machining operation to be performed or not.

Insert a formula or choose one of the two options:

  •    False: do not carry out.
  •    True: carry out.

If the commandTool designed to perform a specific function/operation. For example, the “Print” command allows you to print a document. execution is deactivated, the text string of the command icon in the geometries and machining operations tree list will turn light grey.

For example, by entering the formula lpx>2000, if the x length of the piece is greater than 2000 mm, the machining operation is displayed and then performed.

Layer

LAY

Enables user to select the level to be assigned to the machining operation.

Save Default

Used to save the data set as default - always shown when using the specific machining operation command.
