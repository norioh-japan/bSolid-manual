Creating the sheets from an import

To create a new sheet by taking the data from an external file (csv, txt, etc.), proceed as follows:

1. Click on the button.
2. Click on the button.
3. Choose the file to be imported from amongst the operating system foldersContainer for storing files or documents., then click on Open.
4. Click on the Confirm button of the relative importInsert, in an open document (file), information created with another program and in a different format from the one currently being used. window.

The relevant table will contain a new row with the name of the imported sheet.

Duplicating sheets

To create a sheet by copying an existing one, proceed as follows:

1. Click on the button.
2. Select the table row with the name of the sheet to be copied.
3. Click on the button.
4. Enter the name of the new sheet in the appropriate data field, then confirm.
5. Modify the necessary data.

For a description of the data fields, see Description of the data fields.

6. Click on the Save button.
7. Click on the Close button to close the data area.
Deleting sheets

To delete one sheet or more from the table, proceed as follows:

1. Click on the button.
2. Select the table row you want to delete.
3. Click on the button.
4. Confirm the operation carried out.
NOTES:

If the sheet created has been inserted in the project, it cannot be deleted.
