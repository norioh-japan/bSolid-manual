Edgebanding concepts

Programming concepts

The operations for applying the edgebandingSet of operations for covering the edge of the piece (gluing, cleaning, rounding off, etc.). strip to the piece are programmed in the Cad/Cam environment.

The machining operations must be applied to the geometric profile, and the number of sides they are applied to is linked to the selection of the single geometry elements.

Each profile consists of geometric elements (lines and arcs), and represents the work trajectory to be followed. Each element is linked to the one that follows it and the one that precedes it; this link allows you to establish the workable sides, and therefore to define two types of profile: the “disjointed” or “open” profile, and the “closed” profile.

Open or disjointed profile (open ring)

When the geometric elements used to form the profile are contiguous but not tangent, they generate tangency interruptions and therefore a non-continuous trajectory (a discontinuous geometry); in this case, the profile is defined as “disjointed” or “open” and the machining operation is defined as “open ring edgebanding”.

If you select an element of the profile in the graphics area, the system will automatically search for the elements contiguous or tangent to it, in order to determine the workable side.

Closed profile (closed ring)

When the elements used to form the profile are contiguous and tangent, the trajectory generated is unique and it corresponds to a single side of the piece as it starts and ends in the same point. In this case, the profile is defined as “closed” and the machining operation is defined as “closed ring edgebanding”.

31. Representation of closed ring profiles

If you select an element of the profile in the graphics area, the system will automatically select the whole profile.
