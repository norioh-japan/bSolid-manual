Creating tools

1. Click on the button to visualise the New tool dialogue box.
2. In the Tool name data fieldA defined area containing data or commands (data field or command field)., enter the code or name of the tool.
3. Choose the class by clicking on the relevant iconGraphic image used to show a data item, button, etc. in the Tool class field; the page for choosing the model will appear.
4. Click on the model icon in the Models available field. The card for the new tool will appear in the main part of the environment.
5. Fill in the data fieldsA defined area containing data or commands (data field or command field).. (Compilation will vary according to the type of tool to be registered).
6. Save tools, by clicking on button Save. The tool created will be added to the databaseData bank. Stored data arranged in an organised manner so that they are readily available using a search and management program. table.

Creating tools via a copy

1. Select the table row with the name of the tool to be copied.
2. Click on the button.
3. Enter the tool name in the relevant data field of the Copy tool dialogue box.
4. Click on the button to save the settings and close the dialogue box. The card for the new tool will appear in the main part of the environment.
5. Fill in the data fields.

Save the tools respecting the safety standards

When creating a new tool, to save it correctly in the database in compliance with the safety standards, the following operations must be performed.

  •    Saving in the environment Tools:
  •    Modify the data fields Maximum diameter (card Dimensional data) entering a valuePredefined / pre-set. Standard information useful for the control, if the requested information is not entered. greater than the tool diameter. If the value indicated should be less than that defined in the data field Diameter, it will not be possible to proceed with the saving of the tool.
  •    Modify the data fields Max rotation (card Speed) entering a value greater than the tool rotation. If the value indicated should be less than that defined in the data field Rotation, it will not be possible to proceed with the saving of the tool.
  •    Confirm the previously entered values, related to the maximum diameter and the maximum rotation, in the appropriate text boxes of the dialogue box Confirm the data entered that appears after saving the tool.

If the data is not confirmed the tool cannot be saved.

  •    Saving in the environment Tooling:
  •    Confirm the creation of the new tools inserted in the database, by clicking on the button Okin the dialogue box Save toolingthat appears after saving the tooling.

Deleting tools

To delete a tool from the table, select it then click on the button and confirm.

Renaming tools

To rename a tool in the table, select it then click on the button.
Enter the new name in the Name data field, then confirm with the button.

Create a model from a tool

1. Select the table row with the name of the tool to be saved.
2. Click on the button. The Create template from tool dialogue box opens.
3. Indicate the model name in the Template name data field.
4. Enter a short description in the Template description data field.
5. Click on the button to save the settings and close the dialogue box.

The new model created will appear in the Models available list of the New tool dialogue box.

Finding and filtering the tools

Search the tools to be shown in the table
1. Click on the button to visualise the Search for tool dialogue box.
2. Fill in the required field on the basis of the type of search you want to make (see "Dialogue box Search for tool").
3. Click on the button to save the settings, then close the dialogue box and start the search. Only the filtered tools will be shown in the table.
Filtering the tools with the filter instruments

To show only the required tools in the table, click on one of the buttons of the filter area (see "Filters area" and "Filter tools").

Filtering the tools on the basis of the key word or name
NOTES:

This filter is only valid on tools for machining wood.

  •    To show - in the table - only the tools containing specific key words indicated in the relevant tool filter card, enter one or more words separated by a comma in the text box near the data field Name (see "Filters area").

The search for the key word is made within the filters defined in the tool properties (see "Filters"; tools for wood).

  •    To filter the tools on the basis of the alphanumericMade up of numbers and letters. string indicated in the Name data field of the tool card, enter the string in the Name text box of the filters area.
For example, if you enter a single letter, all the tools whose name contains that letter will be filtered.

Defining the tool data during the registration phase

Adding bodies

The tool shape is made up of several geometric shells. During the registration phase, you must describe all the shells that form the tool, starting from the spindle connection device.

To add a shell, proceed as follows:

1. Select the type of tool connection device in the "Type of connection device" data field of the Morphology tabLabel which, if selected with the cursor, displays a list of data.: the drawings will be updated.
2. Click on the “+” button near the title of the section. A shell is added to the drawing, and the section is increased with the new sections, with the specific data field of that shell.
3. In the "Body type" data field, select the corresponding geometric shape: the drawings will be updated.
4. Fill in the data fields that define the shape of the shell (see the "Morphology" section), and add all the necessary shells.
5. Click on the Save button to save the settings and close the tool card.
To save without closing the card, click on the Apply button; in this case, click on the Undo button to close the card.
Adding check points
NOTES:

This procedure is only valid on tools for machining wood.

See also:
• "Control points"

The check points are additional points that can be specified on a tool profile. They are used as a reference for the machining operation (e.g. to exploit a part of the tool profile while milling, or to position the tool higher or lower than the nominal lead-in point).

To add check points, proceed as follows:

1. Click on the “+” button near the title of the Reference list section to add a new reference.
2. Click on the Select reference from graphic area button and position the cursorMobile element that is visible on the screen, and which moves as the mouse is moved. on the measured graphics of the tool.
3. Click along the tool profile to define the check point; the system calculates the tool diameter in that point, and the distance from the spindle nose.
4. Click on the Save button to save the settings and close the tool card.
To save without closing the card, click on the Apply button; in this case, click on the Undo button to close the card.
Adding key words for the search
NOTES:

This procedure is only valid on tools for machining wood.

See also:
• "Filters"

It is useful to enter key words for each tool when you need to create a lot of tools, because it simplifies the search and only shows the required tools in the table.

During the search in the table, you will only have to enter the key words in order to see the list of required tools.

To add key words, proceed as follows:

1. Select the type of machine in the specific data field.
2. Click on the “+” button near the title of the Filter by family section to bring up the text box where the key word is to be inserted.

To insert several key words, click on the “+” button again.

To insert other key words, use the Filter by applicationA synonym of "program" - software with the specific task of managing data, graphics, etc. section.

3. Click on the Save button to save the settings and close the tool card.

To save without closing the card, click on the Apply button; in this case, click on the Undo button to close the card.
