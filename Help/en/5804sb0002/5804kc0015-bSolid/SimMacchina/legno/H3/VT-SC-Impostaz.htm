Prefix for the geometries, indicating the areas to be locked

Used to define the prefix (e.g. RAW_) to be inserted in the field with the name of the geometry, to specify that that geometry is a piece to be locked.
If the prefix is not used, the positioner uses the standard piece as a locking reference.

Prefix for the geometries, indicating the areas to be left free

Used to define the prefix (e.g. WRK_) to be inserted in the field with the name of the geometry, to specify that that geometry is a piece that must not be locked.

Discretisation step

Allows to define the subdivision of the locking area, in order to perform a more precise control between the suction cups and the machining operations, checking for possible collisions. The lower the value, the more accurate the positioning.

The automatic setting is zero (recommended optimal value); in this case the calculation is automatically performed based on the dimensions of the piece.

File containing the default tooling to be used

Used to choose which work table configuration file to use during the semi-automatic positioning.
