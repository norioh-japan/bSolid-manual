Type of optimisation

TYPE_OF_OPTIMIZATION

Used to select the type of optimisation.

By work list

Used to arrange the machining operations in order, following the sequence indicated in the source program.

By Time

Used to minimise machining times.

By type of machining operation

Used to prioritise the machining operations on the basis of the type.

By work list and head boring

Used to prioritise the boring operations that can be optimised, grouping them together at the start of the program or with the non-optimisable operations so as to make the least possible number of descents.

By tool

Used to prioritise the machining operations, following the tool sequence defined in the relevant section.

Increase in the deflector rotation angle - can be calculated on the "Fixed deflector angle" data item of the milling operation.

This data item is only applied if the "Fixed deflector" and "Maintain deflector" data fields have been enabled in the machining operation.
