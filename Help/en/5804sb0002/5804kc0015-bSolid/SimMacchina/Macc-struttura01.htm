Graphics area

Shows the card with the machine graphics and the graphics of the piece to be processed (but only if this is positioned on the work table).

TabActive machine

Shows the tree list with the icons of the machine parts and work table that interact with the image of the graphics area. By clicking on a tree list object, you can see the list of the parts that form it and view the detail in the graphics area.

To highlight (in the graphics area) the object shown in the tree list, pass over the icon with the mouse without clicking; the icon will turn orange and the relative object of the graphics area will turn yellow (ref. B, figure 2).

416. Example:

The list of all the machine origins is displayed by clicking on the origin icon. The selected origins are blue (ref. A); the highlighted origin is yellow in the graphics area and orange in the tree list (ref. B).

A. Yellow or Orange: object highlighted by passing over it with the cursor.
B. Blue: object selected by clicking on it with the cursor.

Displays data related to the selected object in the tree list: e.g. the positions of the carriages, planes, etc..

TabAxes

Shows the list of machine axes in order to move the various parts and position them in a precise place.

To move an axis, position the cursor on the horizontal slide bar and drag it, keeping the left mouse key pressed (figure 5).

Object cards

Shows the objects (clampsPressure exerted by the presser roller to hold the edgebanding strip in place. and shapeable suction cups) catalogued in the magazine, to be used for tooling the work table.

Tab Supports

Shows the list of all the available types of support (e.g. shapeable suction cups, etc.). For the technical characteristics of the supports, refer to the machine user instructions.

Tab Uniclamp/Hyperclamp

Shows the list of all the available types of clamp (e.g. Uniclamp, etc.). For the technical characteristics of the clamps, refer to the machine user instructions.

Objects management button
 

Tools all the selected carriages.

Opens the dialogue box in order to define the number of objects belonging to the single selected type. The number above the object icon indicates the quantity.

CardSimulation

The card Simulation visualises the simulator in order to carry out virtual checks on the machine and verify the machining results. Shows (on the screen) what will actually happen to the piece.

Related topics
• Simulator commands

To launch the simulation and check the movement of the work table objects, use the following commands:

Speed

Allows you to increase or reduce the speed of the virtual simulation.

ATTENTION:

The results of the collision checkA check of the positioning of the moving objects on the table (e.g. shapeable suction cups, mobile panel supports, etc.) so they do not collide with each other or knock against the tools or the piece. during the software simulation depend on the performance of the PC, so correct results cannot be guaranteed for speeds of more than 1x. Always carry out a 1x simulation to make sure there will be no collisions.

Repetition

Enables (box tickedClick on a check box to tick it and thereby enable that function.) or disables the repetition of the machining operation.

Collision check

Enables (box ticked) or disables a check of the positioning of the moving objects on the table (e.g. shapeable suction cups, mobile panel supports, etc.) so they do not collide or knock against the tools.

Waste removal

Allows to choose how to eliminate the wastePart left over after a machining operation (residue). In nesting projects this definition also applies to waste. material(nesting) Type of material to be associated with the articles and sheets of the nesting project.. If to eliminate it during simulation, after each pause, etc.

Material removal

Enables (box ticked) or disables the visualisation of the material to be removed.

Quality

Allows you to increase the precision of the end result. By increasing the precision, you will improve the visualisation of the processed part.

Hide piece

Enables or disables the display of the piece. TickClick on a check box to tick it and thereby enable that function. the box to hide the piece.
