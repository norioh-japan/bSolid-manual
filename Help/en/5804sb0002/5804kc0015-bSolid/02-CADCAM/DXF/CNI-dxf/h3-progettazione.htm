Creating the piece

The piece must be geometrically represented by a closed rectangle, using the commandTool designed to perform a specific function/operation. For example, the “Print” command allows you to print a document. for generating closed polygons, lines or polylinesSet of consecutive straight lines, connected to each other at the ends..

The dimensions along the X and Y axes are provided directly by the geometry, whereas the thickness must be entered in the layer associated with the rectangle.

If the depth is not specified it will be assumed to be zero.

Example:

To draw a piece with dimensions 1000 x 300 x 30.20 (mm), it will be necessary to associate a rectangle 1000mm wide and 300mm high with the layer PANNELLO30K20.

Creating the general face

With this way of creating the DXF files, the concept of a "general face" does not exist.

Creating standard borings

Bores can be represented in two different ways:

  •    by a circle used to represent a vertical bore, to be machined on face 0 or on face 5 of the piece.
Vertical boring operations represented by circles

The vertical bore, which can be machined on face 0 or on face 5 of the piece, must be represented by a circle, which must be drawn inside the rectangle representing the face required.

The diameter of the bore is given by the diameter of the circle, whereas the depth must be specified in the layer. If the depth is not specified it will be taken to be zero.

The bore applicationA synonym of "program" - software with the specific task of managing data, graphics, etc. point coincides with the centre of the circle.

Example:

To create a vertical bore with a depth of 15mm, associate the layer VERTICA15 with the drawing of the bore.

  •    by a rectangle or a closed polygonal line to be drawn on face 0, used to represent bores to be machined with the horizontal spindle on the side faces of the piece.
Horizontal boring operations represented by rectangles

When a bore is created using a rectangle, its diameter and depth are obtained from the geometry.

  •    The depth is indicated by the length of segment B (figure 1).
  •    The application point is the average point of the segment, representing the bore diameter.

Creating angular borings

With this way of creating the DXF files, only the “AR” rotations of the spindle axis on the X-Y axis are managed, not the “AZ” tilts of the spindle rotation axis in relation to the xy plane (i.e. the tilt of the spindle in space).

It is thus possible to create bores on the side faces (ref. A, figure below) by rotating the axis of the spindle on the xy plane.

Creating the cuts

A cut is represented geometrically by a line.

The layer associated with the line can be used to specify both the type of machining operation and the cut depth. If the depth is not specified it will be assumed to be 0.

Cuts on side faces are not managed.

Creating the milling

A milling operation can be geometrically represented by lines, arcs, circles, closed or open polygonal lines, and dots.

The dot is generally used to specify the co-ordinates for the start of the milling operation.

The layer associated with the geometric elements is used to specify both the type of machining operation and the milling depth. If the depth is not specified it will be assumed to be 0.

Milling operations on the side faces are not managed.

Ways of creating the milling operations
  •    To generate a milling operation containing several geometric elements, draw all the geometries A in one single layer.
  •    To ensure that each single geometric elementAn abstract element associated with specific properties: the point, line, plane and space are all geometric elements. is recognised as a single milling operation, you must create three layers (one for each geometric element - B, C, D).
