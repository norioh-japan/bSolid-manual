Configure the rules for the sequences

Description

The machining operation sequences are a set of machining operations arranged in a particular sequence that is then applied to selected geometries or surfacesFigures of a three-dimensional area, that represent a thin open or closed structure. so as to facilitate programming.

For example, if you need to generate a rough-shaping, a milling with shaped mill and a finishing in order to profile a piece, you can use the sequences to create a group containing all these operations with different tools. It will be inserted in the Cad/Cam environment, and can be applied to the drawn profile.

Create the sequences

To create a new machining operation sequence, proceed as follows:

1. Click(1) Indicates the action of pressing (and immediately releasing) the mouse button on an object (icon, button, etc.) located under the pointer on the screen. (2) (verb) Pressing and immediately releasing the left mouse button to enable the function of a selected command. on the buttonCommand field that is responsive to mouse clicks and that enables operations to be activated..
2. Fill in the data fields.


Sequence name

Sequence name.

Sequence type

Sequence type. Choose the type in the drop-down menuList of options or instruments which expand vertically or horizontally when you click on the drop-down buttons..

Generic

Generic machining operation sequence. In this case, you can insert all the required machining operations.

Description

Comment or description of the sequence.

Image

Choose an image to be associated with the sequence.

3. Some types of sequence allow you to select the tools to be used for each machining operation. For each tool chosen, a machining operation is automatically created.
To insert the tools, click on each single iconGraphic image used to show a data item, button, etc. of area A; the tool will be automatically inserted in area B.

4. Click on the button to save the operations carried out and close the window, or click on the button to close the window without saving.
The card A will appear in the main part of the environment, showing the name of the newly created sequence.

5. Fill in the data fieldsA defined area containing data or commands (data field or command field). of area B. For the description, see "Sequences data".
When you click on a machining operation, the preview box C will show the cards containing the data of the selected machining operation. For a description of the data fields, refer to the paragraphs dealing with the type of machining operation chosen.

Search the sequence to be shown in the table

1. Click on the button to visualise the Sequence search dialogue box.
2. Fill in the relevant field on the basis of the type of search you want to carry out.

Name

Used to filter the search on the basis of the name.

Enter the sequence name.

Type

Used to filter the search on the basis of the type.

Enter the sequence type.

Description

Used to filter the search on the basis of the comment.

Enter the comment or description of the sequence.

3. Click on the button to save the settings, then close the dialogue box and start the search.

Only the desired sequences are shown in the table.

Filtering the sequences with the filter instruments

To show only the required tools in the table, click on one of the buttons of the filter area (see "Filter tools").
