Overview of the environment

When the applicationA synonym of "program" - software with the specific task of managing data, graphics, etc. starts up, the Tools environment shows the Tool data card with the table containing all the tools recorded in the database.

The visualisation of this environment is multi-card. This means that several cards can be opened simultaneously, so you can manage and control several tools.
