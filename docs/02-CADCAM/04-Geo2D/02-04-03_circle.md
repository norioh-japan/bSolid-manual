# 円 ![円アイコン](../../../FIGURE/15-icone/b15b0001/Draw_Circle_CenterRadius.png)

## 説明

円を作成します。

円を修正するには、データエリアの関連フィールドを使用します。

## 円を描画するステップ

1. **描画バー**の![円](../../../FIGURE/15-icone/b15b0001/Draw_Circle_2PointsRadius.png)ボタンをクリックします。

2. 円のタイプデータフィールドの2つのツールのうちの1つをクリックすることによって円のタイプを選択します。

![円作成手順](../../../FIGURE/10-videateComplete/arte4/b10b0407.gif)

3. 描画エリアでクリックすることによって円の中心点を挿入します。円の点または寸法を特定するには、データエリアの関連フィールドを使用します。

![円描画プロセス](../../../FIGURE/10-videateComplete/arte4/b10b0408.gif)

4. 円を終了したい点をクリックします。

5. **取り消す**ボタンをクリックして**円**カードを閉じます。

## 一般的パラメーター

### 円のタイプ
**TYP**

円のタイプを選択するために使用されます。

#### ![3点円](../../../FIGURE/15-icone/b15b0001/Draw_Circle_3Points.png) 3点を使用した円
3点を通る円を作成します。

#### ![中心半径円](../../../FIGURE/15-icone/b15b0001/Draw_Circle_CenterRadius.png) 中心点と半径が定められた円
中心点の座標および半径を基準値として使用した円を作成します。

#### ![中心直径円](../../../FIGURE/15-icone/b15b0001/Draw_Circle_CenterDiameter.png) 中心点と直径が定められた円
中心点の座標および直径を基準値として使用した円を作成します。

#### ![2点半径円](../../../FIGURE/15-icone/b15b0001/Draw_Circle_2PointsRadius.png) 2点と半径が定められた円
2点および半径を基準値として使用した円を作成します。

### 中心位置
**ALG**

2つの構成点に対する円の中心点の位置。
利用可能な2つのオプションのうち1つを選択します。

#### ![位置0](../../../FIGURE/15-icone/b15b0001/Draw_Circle_Pos1.png) Pos.0
2つの構成点を通る線の下に、円の中心点を移動する。

#### ![位置1](../../../FIGURE/15-icone/b15b0001/Draw_Circle_Pos2.png) Pos.1
2つの構成点を通る線の上に、円の中心点を移動する。
