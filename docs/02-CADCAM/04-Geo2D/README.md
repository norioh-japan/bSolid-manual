## 機能概要

2次元図形の作成は、CAD/CAM設計の基本となる重要な作業です。bSolidでは直感的なツールを使って、高精度の2次元図形を簡単に作成することができます。また、作成した2次元図形は、3次元モデリングの基礎としても利用できます。
