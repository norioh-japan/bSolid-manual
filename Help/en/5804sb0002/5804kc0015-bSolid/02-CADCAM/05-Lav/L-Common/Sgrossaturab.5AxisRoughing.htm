5-axis rough-shaping

Description of the machining operation

Generates rough-shaping operations; only for machines with a 5-axis milling unitWorking unit equipped with a system for pointing the tool in every possible direction (see the machine user instructions). (5-axis working head(1) Side faces of the pieces (see "sides of the piece"). (2) Working unit (woodworking machines).).

This type of machining operation can be applied to the analytic surfacesFigures of a three-dimensional area, that represent a thin open or closed structure. and large bas-reliefs.
