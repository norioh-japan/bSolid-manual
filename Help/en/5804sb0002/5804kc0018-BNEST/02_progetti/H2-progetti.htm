Archive table Projects

Shows the list of all the nesting projects created.

Each column of the table contains a summary of the specific data for each single objectGeneral term used to define any drawing created in the drawing area (e.g. piece, texts, flat figures, etc.). defined during the creation phase.

628. Representation of the archive

Description of the fields of the table:

Name

Project name.

Description

Brief description.

Status

Current nesting project status.

Can be processed

Grey LED: project still to be processed.

Being processed

Yellow LED: project being processed.

Processed

Orange LED: project processed but not yet transmitted.

If modifications are made while in this status, the LED will turn grey to show that the project needs to be processed again.

Being transmitted

Light blue LED: project being transmitted.

Transmitted

Blue LED: project transmitted to the machine.

 
Produced

Indicates (box tickedClick on a check box to tick it and thereby enable that function.) that the project has been produced.

Processing start date

Date (day, month and year) and time for starting the nesting project processing, after clicking on the commandTool designed to perform a specific function/operation. For example, the “Print” command allows you to print a document..

Processing end date

Date (day, month and year) and time for the completion of the nesting project processing.

Transmission start date

Date (day, month and year) and time for starting data transmission to the machine, after clicking on the command.

Transmission end date

Date (day, month and year) and time for the completion of data transmission to the machine.

Last modified

Date (day, month and year) and time of modifications to the nesting project.

To open the data page and modify the individual project, click(1) Indicates the action of pressing (and immediately releasing) the mouse button on an object (icon, button, etc.) located under the pointer on the screen. (2) (verb) Pressing and immediately releasing the left mouse button to enable the function of a selected command. on a row of the table.

page with the list of data relating to the project, and to the articles inserted in each single project.
