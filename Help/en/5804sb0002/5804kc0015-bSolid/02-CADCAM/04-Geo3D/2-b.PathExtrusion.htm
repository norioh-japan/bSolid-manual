Steps for creating a surface
1. Select the plane (or piece face) on which you want to draw.
2. Draw the closed geometry.

3. Draw the path geometry.

4. Click(1) Indicates the action of pressing (and immediately releasing) the mouse button on an object (icon, button, etc.) located under the pointer on the screen. (2) (verb) Pressing and immediately releasing the left mouse button to enable the function of a selected command. on the buttonCommand field that is responsive to mouse clicks and that enables operations to be activated. of the drawing barBar listing the drawing tools..
5. Enter the closed geometry in the "Geometry to extrude" fieldA defined area containing data or commands (data field or command field). of the data areaGeneral term to define a specific area for data entry..

6. Enter the path geometry in the "Extrusion path" field of the data area.

7. Click on the "Ok" button to generate the surface.
