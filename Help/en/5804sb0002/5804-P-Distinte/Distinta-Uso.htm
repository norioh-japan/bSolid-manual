Managing the work list

NOTES:

The List environment opens, always showing the last card used. In this case, you can create a new list, open one already in the databaseData bank. Stored data arranged in an organised manner so that they are readily available using a search and management program., or fill in the one currently open.

Creating a new work list

1. Access the List environment.
2. Click(1) Indicates the action of pressing (and immediately releasing) the mouse button on an object (icon, button, etc.) located under the pointer on the screen. (2) (verb) Pressing and immediately releasing the left mouse button to enable the function of a selected command. on the buttonCommand field that is responsive to mouse clicks and that enables operations to be activated..
3. Filling in the current work list.
4. Click on the button to save.

Opening a work list

1. Access the List environment.
2. Click on the button.

Filling in the current work list

After opening(nesting) Slang term deriving from "windows", openings in the walls, and indicates the "openings" in the piece, passthrough machining operation of open and/or closed surfaces. or creating a work list, proceed as follows:

1. Click on the button to insert the program files.

To insert several programs at the same time, use the Ctrl key of the keyboard.

2. Indicates - in the Quantity data fieldA defined area containing data or commands (data field or command field). - the number of times that the machine must execute the program.
3. Click on the button to save.
Modify the piece dimensions
1. Click on the button.
2. Act on the relevant data fieldsA defined area containing data or commands (data field or command field)..
Insert the stops
1. Click on the button to insert the stop.
2. Use the - commands to move the instruction and position it on the required row.
Inserting the row skip
1. Click on the button.
2. Open the drop-down menuList of options or instruments which expand vertically or horizontally when you click on the drop-down buttons. and choose the program you want to restart from.
3. Indicates - in the Quantity data field - the number of times that the machine must execute the program.
Calculating the machining times
1. Position the cursorMobile element that is visible on the screen, and which moves as the mouse is moved. on the row you want to start the time calculation from
2. Click on the button.

To reset the values of the times calculated, click on the button.

Inserting the documents in the list

1. Access the Cad/Cam environment.
2. Open or create the documentElectronic worksheet on which the geometric and technological parameters are inserted. to be inserted in the work list.
3. Click on the button to insert the document without opening the list, or press the CTRL key of the keyboard then click on the same button to insert it and pass directly to the List environment.

If no card has been opened in the List environment, the system will create a new one and insert the document in it. If, on the other hand, you want to insert the document in a specific card, open it before moving to the Cad/Cam environment.

The documents in the list can also be inserted via the Machine simulation environment.

Launching the work list

1. Access the List environment.
2. Opening a work list / Creating a new work list.
3. Filling in the current work list.
4. Select the row you want to start the machining operation from.
5. Click on the button.
