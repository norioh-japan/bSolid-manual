Piece

Description

Creates a new standard piece.

The piece can be created using two different methods:

  •    Create by entering dimensional parameters
  •    Creating by selecting a flat figure

List of origins

Sole origin

UniqueOrigin

Enables the use of a single origin on the work table.

Join tangent elements

TangentEntity

Joins the faces generated by a geometry and which have tangent elements, in order to create the smallest possible number of non-standard sides.

Finished piece geometry

PickingFinished

Used to generate the shape of the finished piece, starting from a flat figureTwo-dimensional geometric figure (e.g. polygon, circle, etc.) belonging to a single plane.. In this case, the dimensional data fields will be disabled.

Blank piece geometry

PickingRough

Used to generate the shape of the rough piece, starting from a flat figure. In this case, the dimensional data fields will be disabled.

Geometry name

NMC

Name of the geometry created.

The text entered in this field will rename the nodeFor the definition, refer to Tree list. text string (the one near the iconGraphic image used to show a data item, button, etc. of the chosen tool) to distinguish it from the others in the "Geometries and machining operations" tree listStructure containing a number of items, known as nodes, which in turn have branches and secondary branches like a tree. The node at the highest level is called the root. All the others derive from one single node and, in turn, can have other nodes.. If this field is left empty, the system will automatically enter the default text.

Execute geometry

EXE

Used to enter a condition for performing or not performing the commandTool designed to perform a specific function/operation. For example, the “Print” command allows you to print a document..

Insert a formula or choose one of the two options:

  •    False: do not carry out.
  •    True: carry out.

If the command execution is deactivated, the text string of the command icon in the geometries and machining operations tree list will turn light grey.

For example, by entering the formula lpx>2000, the piece is only displayed and created if its x length is greater than 2000mm.

Layer

LAY

Used to select the level to be assigned to the piece.

Virtual panel

Virtual

Used to indicate that the piece generated is virtual (only useful for simulating the machining operation in the bSolid simulator area). In this case, the simulator only visualises the personalised pieces, obtained by creating non-standard faces.

Movement in X

Enables (box ticked) the movement of the origin along the X-axis, if there are physical reasons on the machine that require this movement. Enables (box ticked) the movement of the origin along the Y-axis, if there are physical reasons on the machine that require this movement. Used with a type of piece stops that have a protruding coverings (known also as "2-movement stops").
