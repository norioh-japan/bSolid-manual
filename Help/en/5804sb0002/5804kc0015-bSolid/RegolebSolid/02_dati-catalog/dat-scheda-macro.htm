Tab Properties

Shows a list of the variables created.

In this section you can modify the variables created (arranging them, and modifying their characteristics and valuePredefined / pre-set. Standard information useful for the control, if the requested information is not entered.).

The explanation of the data shown on this tabLabel which, if selected with the cursor, displays a list of data. is based on a macro called Macro_Fresatura, in which a milling program with the following variables has been inserted:

  •    LUNGY: variable for defining the length (or base) of the rectangle to be machined.
  •    LUNGX: variable for defining the width (or height) of the rectangle to be machined.
  •    PUNTX: variable for defining the x coordinate of the initial point of the rectangle to be machined.
  •    PUNTY: variable for defining the y coordinate of the initial point of the rectangle to be machined.
  •    ROTAZ: variable for defining the rotation of the rectangle to be machined.

Tab Sections

In this section you can create new sections for grouping together the variables.

Tab Preview

Shows the preview of the data structure of the newly created macro - i.e. the one shown in the data area of the Cad/Cam environment while the macro is being applied.

Tab Special functions

In this section you can enable the data fields of the Advanced macro card.

Enable shift

Enables (box ticked) or disables the insertion of the X, Y and Z data fields in order to shift the macro applied.

The enabled box allows to view in the macros created, the section Shiftwith a list of the following data fields:

X

X position of the shift point, starting from the initial point of the geometry contained in the macro.

Y

Y position of the shift point, starting from the initial point of the geometry contained in the macro.

Z

Z position of the shift point, starting from the initial point of the geometry contained in the macro.

Enable rotation

Enables (box ticked) or disables macro rotation.

The enabled box allows to view in the macros created, the section Rotation and section Rotation pointwith a list of the following data fields:

Rotation angle

Macro rotation angle, starting from the rotation centre.

X

X co-ordinate of the macro rotation centre.

Y

Y co-ordinate of the macro rotation centre.

Z

Z co-ordinate of the macro rotation centre.

374. Macro rotation
A. Rotation angle
B. Rotation point
Enable mirroring

Enables (box ticked) or disables the flipping of the macro on itself.

The enabled box allows to view in the macros created, the section Mirrorwith a list of the following data fields:

Mirror in X

Used to flip the macro horizontally. In this case, the upturned macro will be specular to the original one in X.

375. Flipped macro
A. Normal macro
B. Horizontally upturned macro
Mirror in Y

Used to flip the macro vertically. In this case, the upturned macro will be specular to the original one in Y.

376. Flipped macro
A. Normal macro
B. Vertically upturned macro
Enable corner selector

Enables (box ticked) or disables the type of selection of the faces and corners to be displayed in the General data section of the macro.

Multiple face

Shows all the faces of the piece, with the relative corners.

Single face

Shows only the single face of the piece, with the relative corners.
