Steps for creating a surface
1. Select the piece face on which you want to draw.
2. Draw the open outlineFlat figure in which the first and last extremities do not coincide (they do not meet). of the first surface section.

3. Select another face and draw the open outline of the second surface section.
Make sure the direction of the two outlines is correct and, if necessary, alter it with the aid of the specific commandTool designed to perform a specific function/operation. For example, the “Print” command allows you to print a document. of the CAD options barBar listing the optional tools useful for drawing (e.g. the commands to rotate or resize the drawing, etc.).

4. Click(1) Indicates the action of pressing (and immediately releasing) the mouse button on an object (icon, button, etc.) located under the pointer on the screen. (2) (verb) Pressing and immediately releasing the left mouse button to enable the function of a selected command. on the buttonCommand field that is responsive to mouse clicks and that enables operations to be activated. of the drawing barBar listing the drawing tools..
5. Select the face representing the plane on which you want to develop the surface.
6. Enter the two outlines in the "Sections" data fieldA defined area containing data or commands (data field or command field)..
7. Click on the "Ok" button to generate the surface.

Sections

SEC

Used to select the open outlines (ref. A, figure 1), that contribute to creating the sections of the surface to be generated.

There may be more than two open outlines that together create the sections. The shape of the end surface depends on the direction of the outlines created (green arrow, figure 1).

Accuracy

PR

Index to define the precision of the three-dimensional figureGeometric figure of a three-dimensional space (e.g. the surfaces). created. Values lower than 1,000 make the objectGeneral term used to define any drawing created in the drawing area (e.g. piece, texts, flat figures, etc.). more precise.

Grade

DGR

Used to determine the degree for calculating the curvature points of the created surface. A higher degree enables broader, softer curves to be generated, resulting in more homogeneous surfaces.

The default valuePredefined / pre-set. Standard information useful for the control, if the requested information is not entered. is based on the number of linked outlines.
For example: to link 2 outlines a value of 1 is inserted; to link 3 outlines a value of 2 is inserted; to link 3 or more outlines a value of 3 is inserted. The value indicated must not be greater than the geometries selected, less one. Therefore, if four geometries are selected, a degree of 1, 2 or 3 can be indicated.
