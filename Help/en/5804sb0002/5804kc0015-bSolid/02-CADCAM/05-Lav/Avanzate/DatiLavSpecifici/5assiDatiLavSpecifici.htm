Specific machining data

Parameters for analytic surfaces

Cutting method

CUTM

Method with which the tool moves in order to perform the machining operation.

269. Methods
A. One direction
B. Zig Zag
C. Spiral
One direction

Machining operation for parallel lines; each single movement is separate from the following one, and the tool ascends.

• See reference A, figure 1.
Zig Zag

Continuous machining operation for parallel lines; the movements are all linked.

• See reference B, figure 1.
Spiral

Spiral machining operation; the movements are concentric.

• See reference C, figure 1.
Cutting order

CUTO

Direction in which the tool performs the machining operation.

270. Direction
A. Standard
B. From the centre outwards
C. From the outside inwards
Standard

Normal machining.

• See reference A, figure 2.
From the centre outwards

The machining operation begins from the inside and moves outwards.

• See reference B, figure 2.
From the outside inwards

The machining operation begins from the outside and moves inwards.

• See reference C, figure 2.

271. Representation of the data

Accuracy

CTOL

Tolerance for defining the precision of the machining operation on curved surfacesFigures of a three-dimensional area, that represent a thin open or closed structure., in order to calculate the tool trajectories.

The valuePredefined / pre-set. Standard information useful for the control, if the requested information is not entered. to be entered refers to the chord height necessary to ensure that the arc will be more or less precise. A value lower than 0.05 creates more precise arcs; the calculation is slower. A value higher than 0.05 creates less precise arcs; the calculation is quicker.
