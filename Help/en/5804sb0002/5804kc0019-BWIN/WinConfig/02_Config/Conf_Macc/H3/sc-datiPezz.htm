Piece data

NOTES:

The data fieldsA defined area containing data or commands (data field or command field). are only shown when you select a piece from the tree listStructure containing a number of items, known as nodes, which in turn have branches and secondary branches like a tree. The node at the highest level is called the root. All the others derive from one single node and, in turn, can have other nodes. inserted in a machine load(doors/windows) Positioning of the window pieces on the work table..

Piece origin

Reference origin of the work table for pushing the piece.

Element type

Used to identify the piece selected from the tree list. Choose the type in the drop-down menuList of options or instruments which expand vertically or horizontally when you click on the drop-down buttons..

Corner

Used to choose the reference corner of the piece selected from the tree list.

Angle for rotating the piece around the X axis.
