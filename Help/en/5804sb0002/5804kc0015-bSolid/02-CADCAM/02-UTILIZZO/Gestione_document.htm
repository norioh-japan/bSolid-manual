The document

The documentElectronic worksheet on which the geometric and technological parameters are inserted. is a graphic/technological file containing all the instructions needed to machine the pieces. We can therefore say that the document is simply a machining programType of document (file) containing all the instructions needed for machining the pieces of wood, glass or marble..

In the Cad/Cam environment, the documents are visualised with cards, so the display is "multi-document". This means that several documents can be managed simultaneously, and geometries/machining operations can be easily copied or moved from one document to another.

The colour settings of the graphics and lines are saved directly in the document, and do not affect the environment settings. Saving and exporting the data and graphics

bSolid allows you to save the data in a file that can only be read by the application software itself, or to convert them by saving them in the various standard formats that can be read by other application software packages.

The structure of the exported data depends on the file format chosen.
