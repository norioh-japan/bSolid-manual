Boring

Description of the machining operation

Generates horizontal and vertical borings on the surfaceFigures of a three-dimensional area, that represent a thin open or closed structure. of the piece.

Boring operations can be “optimised” and “non optimised”. Both types can be carried out using the boring unit spindles or the electrospindles. The NC will automatically search for tools according to the data set (diameter, type of bit, etc.) selecting the spindles best suited for the type of work started from among the ones present in the machine configuration. There is only one difference between these two types of machining operation, which relates to the order in which the machining programType of document (file) containing all the instructions needed for machining the pieces of wood, glass or marble. is carried out. In the case of “non optimised” machining operations the program will be carried out in sequence, as created by the operator, whereas in the case of “optimised” machining operations, the machining program will be processed by the NC which selects the fastest route to be used for machining operations.

The programmable machining operations are:

  •    Borings from a geometry (i.e. they can be applied to the two-dimensional geometries created on the faces of the piece).
  •    Repeated parametric borings, starting from the position of the first programmed bore.

The geometries that can be used are:

  •    circles;
  •    lines: in this case, the initial point of the line is the initial point of the machining operation, while the end point of the line corresponds to the boring depth.
  •    rectangles - see the "Rectangles as projections" data fieldA defined area containing data or commands (data field or command field)..

Piercing speed

DSP

Bit speed during the piece knock-through phase (can only be used for through borings).

Once the bit reaches the value set in the data field "Piercing distance", it carries out the last path of the machining operation at the speed set in this field. If this field is left empty, the system uses the speed defined in the tool parameters.
