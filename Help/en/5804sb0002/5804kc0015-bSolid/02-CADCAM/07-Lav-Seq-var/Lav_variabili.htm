Introductory remarks

The term variables is used to indicate a size or amount that can take on different values.

bSolid distinguishes between two types of variable: "Global" and "Document".

  •    The "Global" type variables are valid in all the programs. Every modification affects all the programs those variables are found in; you are therefore advised to avoid modifying this type of variable.
  •    The "Document" type variables are only valid in the program in which they were created.

The variables are managed (created, removed, ordered, saved, etc.) in the relevant section. To view them, click(1) Indicates the action of pressing (and immediately releasing) the mouse button on an object (icon, button, etc.) located under the pointer on the screen. Name

Name of the variable.

Variables of the same name can be created, but with the different option of the Context data field.

To decide which one to use (Global or DocumentElectronic worksheet on which the geometric and technological parameters are inserted.), the variable must be preceded by the following characters:

• @D.: indicates that the variable to be used is the one of the Document type.
• @G.: indicates that the variable to be used is the one of the Global type.
336. For example, let's suppose that two variables LUN have been created, one of the Global type with a value of 900 and the other of the Document type with a value of 500.
To use the Document type variable, enter "@D.LUN" in the required field.

Expression

Expression with absolute numerical values, formulas, variables, arithmetic operators (see "The arithmetic operators"), etc. It is necessary that each variable inserted in this data field was created before the variable that contains it.

337. Example: in order to function, the VARAB variable (whose expression contains the variables VARX01 and VARX02) must be inserted after the two variables it contains.

Value

Numerical value of the expression inserted in the previous field.

Category

Used to choose the unit of measurement.

Context

Type of variable that defines where to apply it - only in the active document, or everywhere.

The system variables

The system variables that can be used in the strings of the variables table are:

F:

variable of the face, to be followed by the number of the face (e.g. F6 is face number 6).

LX:

side of the face along the X-axis (length).

LY:

side of the face along the Y-axis (height).

LA:

face tilt angle.

For example: inserting the string P1010.F6.LA (piece name, number of the face and angle) in the "Expression" field, the "Value" field will show the data item referring to the tilt angle of the face created.
