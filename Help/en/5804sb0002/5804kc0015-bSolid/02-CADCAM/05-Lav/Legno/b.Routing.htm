Milling

Description of the machining operation

Generates milling operations on the surfaceFigures of a three-dimensional area, that represent a thin open or closed structure. of the piece. These can only be applied to two-dimensional geometries (flat figures and texts). The milling operations can be carried out with any type of milling cutter or aggregate(tool) Device fitted with tools..

The programmable machining operations are:

  •    Piece profile machining.

  •    Machining operations to be carried out on any piece face to create profiles or texts.

Bidirectional

BDR

Authorises (box ticked) the tool to work in alternate directions when making the next movements. The tool changes direction every time it makes a new movement. This function is enabled by clicking on the box.

If the "Incremental steps" box is ticked, the tool makes the first movement by descending evenly, then reverses its direction and descends again, evenly, for the next movement.

Incremental steps

INCSTP

Enables (box ticked) or disables the creation of vertical movements, constantly increasing the descent between one movement and the next. This field is only valid if the "Bidirectional" box is enabled.

Eliminate lead-outs

NEBS

Enables (box ticked) or disables the joining of the stages programmed in the "Vertical movements" data field (but only for closed profiles). When the box is ticked, the tool performs all the movements without emerging from the piece.
