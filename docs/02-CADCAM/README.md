## 主な機能

- 2次元および3次元の設計ツール
- パラメトリックデザイン機能
- 様々な加工操作（切削、ボーリング、ポケット加工など）
- シミュレーション機能との連携
- DXFなどの標準フォーマットのインポート/エクスポート
- マクロと加工シーケンスによる効率化
