By enabling the box during processing, the profiles defined as "overproduced" are inserted among the other profiles, only if there is available space.

Use whole priorities

The sheet contains the profiles to machine, taking into account the priorities defined in the data field "Priority" of each project article, without considering the available space.

For example: a project comprises these 3 profiles:

  •    A = 5 profiles with priority 1.
  •    B = 5 profiles with priority 2.
  •    C = 5 profiles with priority 3.

By enabling the box, the system firstly applies all profiles A, then if there is still space, applied profiles B, etc.

By disabling the box, the system controls the space available and attempts to introduce profiles A first, but if there is not space tries to insert profiles B or profiles C.
The result may be: 5 profiles A, 2 profiles B and 3 profiles C.
