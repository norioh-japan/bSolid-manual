Customised face

Description

Creates a non-standard faceThe side of the piece not belonging to the five standard sides; it has been obtained from a geometry., starting from a geometry (line, arc, polylineSet of consecutive straight lines, connected to each other at the ends., etc.).

Construction parameters

Name

NF

Number of the face created.

Face level with the geometry

PF

Enables (box tickedClick on a check box to tick it and thereby enable that function.) or disables the creation of the face on the active plane.

All faces

MF

Constructs all the sides of the closed geometric figureFlat figure in which the first and last extremities coincide (they meet). that has been selected. In this case, all the subsequent parameters are disabled.

Join tangent elements

TNG

Enables (box ticked) or disables the creation of a single face obtained by selecting a polylineSet of consecutive straight lines, connected to each other at the ends. with tangent elements.

Angle from path

ANP

If the box is ticked, the "Path" data fieldA defined area containing data or commands (data field or command field). is shown.

Angle

ANG

Tilt of the side created. See figure 1.

114. Representation of the data

Path

DIR

Used to choose the angle of the new side, starting from a geometry. Choose the geometry from the drop-down menuList of options or instruments which expand vertically or horizontally when you click on the drop-down buttons., or in the drawing areaArea dedicated to the creation and modification of the geometries/machining operations..

Physical face

PHF

Enables (box ticked) or disables the true creation of the face to be machined. If the box is ticked, you can indicate that the created side corresponds to the true face of the piece (i.e. it is not a virtual face).

Intersection

ITR

Extends the created side until it intersects the piece sides that are perpendicular to the starting geometry. See figure 2.

B. Extended side
Automatic length

AUL

Enables (box ticked) or disables the face height automatically, using the piece thickness. You can disable the box and enter the face height in the "Length" data field.

Length

LEN

Face height. This data field is only visible if the "Automatic length" box is disabled.
