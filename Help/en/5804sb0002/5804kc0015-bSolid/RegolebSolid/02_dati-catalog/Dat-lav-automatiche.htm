Rule detail

Parameters
Geometry

Type of geometry that the rule must be applied to.

Machining operation

Type of machining operation to be applied to the geometry.

Description

Description of the rule.

Container geometry

Enables (box ticked) or disables the applicationA synonym of "program" - software with the specific task of managing data, graphics, etc. of the rule only to those 2-dimensional closed geometries that contain other geometries.

Collect by

Used to define how to group the machining operations.

Face

All the geometries created are grouped according to the face. In this case, for each face containing geometries, a machining operation is created.

Piece

All the geometries created are grouped according to the piece. In this case, all the geometries created on any face are grouped together in a single machining operation.

Collection rule

Used to define how to group the machining operations on the basis of the geometric data. For example, by entering the parameter D, all the geometries with the same diameter will be grouped together. The parameters to be used are those of the geometries.

Machining operation data

Shows a summary of the parameters in the adjacent data areaGeneral term to define a specific area for data entry..

Rules
De Rule

Applies the machining operation only to those geometries with the characteristics indicated in the expression entered in the adjacent field. Enter the required expression in order to define the rule for applying the machining operation.

Face Rule

Applies the machining operation only to those geometries present in the face indicated in the adjacent field.

Intersect Edges Rule

Only appears in open geometries (e.g. open polylinesSet of consecutive straight lines, connected to each other at the ends. or lines). Applies the machining operation only to those open geometries that intersect a closed geometry.

Enable compensation

Enables (box ticked) or disables the application of the tool radius correction.
