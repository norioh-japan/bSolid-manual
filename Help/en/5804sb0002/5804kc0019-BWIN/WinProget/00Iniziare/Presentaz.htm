Information about the environment

The bWindows environment allows you to create the window design documentElectronic worksheet on which the geometric and technological parameters are inserted. in BWD format.

The design document

The document is a graphic/technological file containing all the instructions needed to create the window.

In the bWindows environment, the documents are visualised with cards, so the display is "multi-document".

This means that several documents can be managed simultaneously, passing from one to another.
