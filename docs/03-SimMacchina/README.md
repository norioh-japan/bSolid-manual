# bSolid マニュアル - マシンシミュレーション

このセクションでは、bSolidのマシンシミュレーション機能について説明します。マシンシミュレーションは、プログラムした加工作業を実際の機械で実行する前にシミュレートする機能です。

## 主な機能

- 実際の機械動作のリアルタイムシミュレーション
- 衝突検出と安全性チェック
- 加工時間の見積もり
- ツールパスの視覚化
- マシン設定の最適化
