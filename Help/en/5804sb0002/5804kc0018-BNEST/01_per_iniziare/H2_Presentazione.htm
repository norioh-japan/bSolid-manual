Information about the environment

The bNest environment is used to create the "Nesting(nesting) Arranging several shapes on a sheet so they are perfectly interlocked." projects.

The term "Nesting" refers to the optimised filling of an area - i.e. a large wooden panel or a large sheet of marble/glass, called a “sheet” - with profiles of various shapes, in order to minimise wastePart left over after a machining operation (residue). In nesting projects this definition also applies to waste. during the machining operation whilst respecting the restraints set (distances between profiles, rotation angles, etc.).

The creation of a nesting project is linked to the following rules:

  •    Creates a machining programType of document (file) containing all the instructions needed for machining the pieces of wood, glass or marble. (bsolid file) containing the shape of the profile to be machined.
  •    Creates the wood type to be associated with the articles(nesting) Profile to be nested. It represents the final piece, with which a machining program and a material are associated. (windows) Object to be catalogued in the archive. and sheets.
  •    Creates the article(nesting) Profile to be nested. It represents the final piece, with which a machining program and a material are associated. (windows) Object to be catalogued in the archive. that the program (with the shape of the profile to be machined) must be associated with. Each article is linked with a type of material(nesting) Type of material to be associated with the articles and sheets of the nesting project. and a specific piece thickness.
  •    Creates a sheet with the dimensions of the panel or sheet to be machined. Each sheet is linked with a type of material and a thickness that must correspond to that of the article to be machined.

The thickness of the piece, and the material it is made of, are the basic characteristics needed for the sheets and articles in order to be inserted in the nesting project. If an article has a piece 30mm thick, but a sheet 30mm thick has not been created in the databaseData bank. Stored data arranged in an organised manner so that they are readily available using a search and management program., an error message appears.

When a project is processed, the result is the creation of at least one bSolid program in which the dimensions of the panel/sheet to be machined are those defined in the sheet used in the project, and the machining operations to be generated are the result of the optimised interlocking of the machining programs inserted in the articles.
