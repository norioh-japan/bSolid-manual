# CAD/CAM

## 概要

このセクションでは、bSolidのCAD/CAM環境について説明します。これはドキュメントの作成専用の環境です。木材の機械加工の命令が所属する画像プロジェクトを扱います。

> **注意**: インターフェースbSolidの各環境がアクティブな機械に接続する部分だけを表示していたとしても、マニュアルのこのセクションの情報は、環境全体の全説明をカバーしています。
