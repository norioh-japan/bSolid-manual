Subspindles

List of the aggregate(tool) Device fitted with tools. spindles.

To view the data list, select the required subspindle.

General data
Absolute ID

ID

Unique number of the subspindle, distinguishing it from all the others.

Name

CnName

Name of the aggregate subspindle.

Relative ID

Rel ID

Unique indicator of the specific subspindle for the aggregate it belongs to.

Parameters
Type of coupling

AttackType

Type of coupling that the tool must have in order to be fitted in the subspindle. Choose the type of connection device in the drop-down menuList of options or instruments which expand vertically or horizontally when you click on the drop-down buttons..

Enabled

Enabled

Enables (box tickedClick on a check box to tick it and thereby enable that function.) or disables the subspindle.

Type

Type

Type of subspindle. Indicates the type of subspindle - i.e. a spindle designed to house boring tools, milling tools, etc. Choose the type from the drop-down menuList of commands (to be selected with the mouse) that develop a sub-menu, i.e. a list of options that allow certain functions/operations to be carried out..

Rotation direction

RotDir

Subspindle rotation direction. Choose one of the following options from the drop-down menu:

Clockwise

Clockwise rotation.

Anti-clockwise

Anti-clockwise rotation.

Automatic

Automatic rotation on the basis of the tool rotation direction.

Reverse the direction for the parent spindle

ParentRevRotDir

Enables (box ticked) or disables the subspindle rotation change.
TickClick on a check box to tick it and thereby enable that function. the box to indicate that, when the subspindle is being used, the aggregate must rotate in the opposite direction to the one specified in the subspindle itself.

Min. rotation speed

MinRotSpeed

Minimum subspindle rotation speed in rpm.

Max. rotation speed

MaxRotSpeed

Maximum subspindle rotation speed in rpm.

Tool classes
Milling

ToolClassesListMill

A ticked box indicates that the subspindle can house the milling cutters.

Cut

ToolClassesListCutting

A ticked box indicates that the subspindle can house the blades.

Boring

ToolClassesListDrill

A ticked box indicates that the subspindle can house the bits.

OffX

Distance in X (ref. A) between the subspindle rotation centre D and the aggregate rotation centre C (figure 1).
Any valuePredefined / pre-set. Standard information useful for the control, if the requested information is not entered. other than zero indicates that the subspindle rotation centre is different from that of the aggregate.

400. Example:

Any value other than zero indicates that the subspindle rotation centre is different from that of the aggregate.

Distance

Distance

Distance between the nose of the subspindle and its rotation centre.

AR

AR

Rotation angle of the subspindle in relation to the aggregate.

On machines with a 3-axis milling unit, the rotation angle must be set by taking into consideration the piece side to be machined. In this case, you must create several identical aggregates with different angles.

Tooling data
Min. tool diameter

MinToolDiam

The minimum diameter that the tool must have, in order to be fitted on the subspindle.

Max. tool diameter

MaxToolDiam

The maximum diameter that the tool must have, in order to be fitted on the subspindle.

Max. tool length

MaxToolLength

The maximum length that the tool must have, in order to be fitted on the subspindle.

Symmetry
Name of the symmetrical spindle

SymmetricSpindleName

Name of the symmetrical subspindle to be associated with the active subspindle.
