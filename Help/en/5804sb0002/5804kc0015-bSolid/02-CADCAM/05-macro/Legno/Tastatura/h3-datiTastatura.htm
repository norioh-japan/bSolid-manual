Main tab

Data

Tracing side

Used to choose the face on which the tracing will be performed.

X position

X coordinate of the starting point of the tracing.

Y position

Y coordinate of the starting point of the tracing.

Z position

Z coordinate of the starting point of the tracing.

Type

Type of thickness tracer to be used for tracing.

Vertical

Fixed thickness tracer of the A type (used for vertical tracing).

Cross

Thickness tracer of the B type, or tracer tool of the C type (both used for vertical and horizontal tracing).

333. Types of thickness tracer
A. Vertical
B. Cross of the type "Fixed"
C. Cross of the type "Radio"

Additional data

Tracer

Used to select the type of thickness tracer.

Fixed

Thickness tracer of the B type.

Radio

Tracer tool of the C type (can be installed in the electrospindle).

Tp number

Used to choose the spindle (or electrospindle) that can take the tracer tool of the C type.

NOTES:

You should check that the selected electrospindle has actually been installed, or that it is compatible with the magazine that houses the tool to be used. If it is not, the optimisation will not be successful.

Safety

Safety distance from the surfaceFigures of a three-dimensional area, that represent a thin open or closed structure. of the piece to be traced during the sections for thickness tracer positioning.

Speed

Tracing speed.

Correction

Used to choose how to correct the tracing position.

Automatic

The correction is automatic.

Save on variable

The correction is saved in the variable defined in the data field below.

Name of the variable

Name of the variable between quotation marks.
