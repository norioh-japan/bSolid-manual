3D scoring

Description of the machining operation

Generates engravings with a tapered tool.

This type of machining operation can be applied to the two-dimensional geometries: heads, single elements, closed and open outlines.

General
Data from tool

EnableToolData

Enables (box tickedClick on a check box to tick it and thereby enable that function.) or disables the visualisation and modification of certain data defined in the properties of the catalogued tool.

Rotation speed

EN_RPM

Tool rotation speed (rpm).

Machining speed

EN_Feed

Advance speed of the tool during the machining operation.

% Lead-in speed

EN_PercFeedIn

Percentage for increasing or reducing the tool advance speed as it enters the piece. This valuePredefined / pre-set. Standard information useful for the control, if the requested information is not entered. affects the speed defined in the Machining speed data field.

For example, if the tool has an advance speed of 3000mm/min and you want to reduce this to 1500mm/min during the entry phase, enter the number 50 in this field (3000 x 50% = 1500).

% Lead-out speed

EN_PercFeedOut

Percentage for increasing or reducing the tool advance speed as it emerges from the piece. This value affects the speed defined in the Machining speed data field.

Z safety position

EN_ZSic

Z position for establishing the distance that the tool must maintain from the piece during unit movement/release operations, in order to avoid any collisions with the table elements or the piece itself.

Z pushing position

EN_ZAcc

Z position for establishing the distance of the tool from the initial machining point. The tool arrives (at maximum speed) at the position established in this data field, then slows down to reach the initial machining point.

Working depth

EN_PAR_DEPTH

Depth of the machining operation.

  •    For surfacesFigures of a three-dimensional area, that represent a thin open or closed structure. created starting from the piece, the start Z position for depth calculation is the active face of the piece.
  •    For surfaces created without the reference piece, the start Z position for depth calculation is the one given by the coordinate system.
Execute pocketing

SV

Enables (box ticked) or disables the pocketing of the selected geometry. In this case, the engraving tool not only engraves the profile but also hollows it out.

Distance between movements

STEPAcronym of "STandard for the Exchange of Product model data". Type of normalised, neutral file used as a graphic data (3D, 2D and wireframe) exchange format between the various CAD and CAM systems.

Centre distance or distance between one movement and the next, in order to hollow out the selected geometry.

Minimum tangency angle

EN_PAR_ANGLE

Angle A (figure 2) for defining whether two consecutive elements are tangent to each other.
If the angle between the two elements is less than this data item, the tool makes the angle without moving away from the piece.

186. Representation of the data

Rotary axis speed

EN_PAR_FEEDROTAX

Tool rotation speed (rpm).

Intermediate Z release points

EN_PAR_ZSVINCV

Intermediate safety position of the tool between one movement and the next.

Minimum section length

EN_PAR_MINLEN

Minimum length of each single step of the trajectory followed by the tool in order to perform the machining operation. Lengths less than this are not machined.

Curve subdivision accuracy

EN_PAR_TOLL

Tolerance that defines the precision of the curvature of the created surfaces.

The value to be entered refers to the chord height necessary to ensure that the arc will be more or less precise. A value less than 0.05 results in more precise arcs; the calculation will be slower; a value greater than 0.05 results in less precise arcs; the calculation will be quicker.

187. Chord height (h)
Machining accuracy

EN_PAR_PREC

Index for defining the precision of the machining operation, in order to calculate the tool trajectories. Values less than 0.05 result in a more precise machining operation.
