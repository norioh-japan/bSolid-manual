Description

Used to enter a brief description.

Primary origin

Used to define the work table origin to be used as the predefined one, if the set origin causes problems during machine load optimisation.

Choose the origin from the drop-down menuList of options or instruments which expand vertically or horizontally when you click on the drop-down buttons..

Shapes

Allows to define the type of elements present in the machine load active. That is, if the machine load relates to the positioning of the sections, the linear elements or the arched elements.

Receives sliding door

Indicates (box tickedClick on a check box to tick it and thereby enable that function.) that the pieces present in the active machined load are specific for the sliding door.

Outward opening

Indicates (box ticked) that the pieces present in the active machined load are specific for the outward opening(doors/windows) Type of outward opening of the sash/door, with rotation around the vertical axis and handle on the right-hand upright of the opening sash/door. door.

Same length

Indicates (box ticked) that the pieces deposited on the work table are of the same length.

Tolerance

Maximum tolerance for establishing the limit beyond which the pieces are no longer considered the same.

Clamp and support parameters

Used to choose the type of automatic locking to be applied to the single machine load.

Supports

Enable the use of the shapeable suction cups. Clamps

Enables the use of the clampsPressure exerted by the presser roller to hold the edgebanding strip in place.. Click on the adjacent button to see the configuration data card.

For the description of the data see bSolid > environmentMachine simulation> Clamp positioning data.

Disabled

Disables automatic positioning.

Use global parameters

Uses the settings defined in the Settings card.

Offset of the pieces for the second machining phase

Distance between the clamp stem and the piece during the locking phase, after activating the specific automatic movement device (for the description of the device, refer to the machine user instructions).

Filter for the elements

Used to insert formulas or conditions for using the machine load.

For example, if the expression LPX>800 is inserted, only pieces longer than 800mm are deposited on the machine.
