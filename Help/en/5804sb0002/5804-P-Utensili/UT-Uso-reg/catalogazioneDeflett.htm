Creating deflectors

1. Click on the button to visualise the dialogue boxNew deflector.
2. Enter the code or name of the chip deflector in the relevant data fieldA defined area containing data or commands (data field or command field)..
3. Fill in the data fieldsA defined area containing data or commands (data field or command field).. (Compilation will vary according to the type of deflector to be registered).

The chip deflector created will be added to the databaseData bank. Stored data arranged in an organised manner so that they are readily available using a search and management program. table.

Creating deflectors via a copy

1. Select the table row with the name of the chip deflector to be copied.
2. Click on the button.
3. Enter the name in the relevant data field of the dialogue box.
4. Click on the button to save the settings and close the dialogue box.
5. Fill in the data fields.

Deleting deflectors

To delete a deflector from the table, select it then click on the button and confirm.

Renaming deflectors

To rename a deflector in the table, select it then click on the button.
Enter the new name in the Name data field, then confirm with the button.

Searching and filtering deflectors

1. Click on the button to visualise the Search for deflector dialogue box.
2. Fill in the relevant field on the basis of the type of search you want to carry out.
3. Click on the button to save the settings, then close the dialogue box and start the search.

Only the filtered chip deflectors will be shown in the table.
