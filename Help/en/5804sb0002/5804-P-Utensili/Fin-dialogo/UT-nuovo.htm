Dialogue box New tool

Tool name

Used to define the name or code of the tool.

Tool class

Used to choose the class that the tool should be associated with. Click(1) Indicates the action of pressing (and immediately releasing) the mouse button on an object (icon, button, etc.) located under the pointer on the screen. (2) (verb) Pressing and immediately releasing the left mouse button to enable the function of a selected command. on the iconGraphic image used to show a data item, button, etc. of the required class.

Milling cutter

Defines all the types of mills to be used in the machining operations that can be programmed as milling operations.

Cutting disc

Defines all the circular-shaped tools (circular blades) to be used in the machining operations that can be programmed as cutting operations.

Boring bit

Defines all the types of boring bits to be used in the machining operations that can be programmed as boring operations.

Tapper

Defines all types of tools"maschio" da utilizzare nelle lavorazioni di tipo TappingThreading of the internal part of the holes made through a special "male" tool designed to prepare the screw seat..

Models available

List of the available models. Choose the required model.

Milling cutter 300

Model to generate the circular blade milling tools used as End milltype milling cutters.

General mill

Model to generate End milltype generic milling tools.

Corner rounding tool

Model to generate the milling tools to round the sharp corners of type End mill.

General blade

Model to generate Bladetype cutting tools .

General bit

Model to generate Normaltype boring tools .

Large boring bit

Model to generate NormalGtype boring tools that allow to perform large boring operations.

Through boring bit

Model to generate Spear-shaped drilltype boring tools that allow to perform through boring operations.

Blank boring bit

Model to generate Normaltype boring tools that allow to perform non-passthrough boring operations.

Countersink boring bit

Model to generate Countersunktype boring tools that allow to perform bores and countersinking at the same time.

Friction tapper

Model for generating End mill type tools to be used for tapping operations.

Comb tapper

Model for generating End mill type tools to be used for tapping operations.
