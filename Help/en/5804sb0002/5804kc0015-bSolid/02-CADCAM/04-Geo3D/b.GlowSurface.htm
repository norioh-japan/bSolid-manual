Selecting the geometry

Geometry

PTG

Allows the user to select the closed coplanar geometries. Geometries inside others (see figure 1) are considered islands, so they are pocketed.

Mode

Delete original geometries

ED

Enables (box tickedClick on a check box to tick it and thereby enable that function.) or disables the deletion of the starting geometries.

If the box is disabled, the surface is created and the origin geometries are maintained.

Accuracy

PR

Index to define the precision of the three-dimensional figureGeometric figure of a three-dimensional space (e.g. the surfaces). created. Values lower than 1,000 make the objectGeneral term used to define any drawing created in the drawing area (e.g. piece, texts, flat figures, etc.). more precise.

Minimum Z

Maximum Z

ZMAX

Z position to establish the surface height. If the "Automatic maximum height" box is ticked, this valuePredefined / pre-set. Standard information useful for the control, if the requested information is not entered. is not taken into consideration.

Automatic maximum height

AUTO

Enables (box ticked) or disables the automatic insertion of the surface height on the basis of the dimensions of the starting geometry. If the box is ticked, the value in the "Maximum Z" data fieldA defined area containing data or commands (data field or command field). is disabled.
