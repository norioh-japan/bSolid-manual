627. Dialogue box Maintenance

Restore factory settings

Performs the reset of the data back to the original conditions established during the installation process.

ATTENTION:

Be warned because reset leads to the loss of all settings and all customized data in the database: pieces, projects, sheets, etc.

Clean up all projects

Allows to perform the cleaning of the projects archive. The options below allow to clean-up the folders, the archive of the sheets, the rests(nesting) A residual piece eliminated during the machining operation, that can be re-used., and the pieces, etc.

Select the desired option, depending on the type of clean-up to be performed. The options are enabled only if the operations are permitted.

Delete parts also

Allows to clean up the archiveArticles.

Delete folders also

Allows to delete the folders created in the archive Articles, only if the previous option has been enabled.

Delete sheets also

Allows to clean up the archiveSheets.

Delete rests also

Allows to clean up the archiveRemainders.

Delete materials also

Allows to clean up the archiveWood types.

The wood types can only be deleted if all the archives have been cleaned up, because these are called-up in the archives of the sheet, pieces and rests.

The deletion of data from the archives Sheets, Remainders and Wood typesis only possible if they have been saved locally; if they have been saved in an external database, connected, the options are disabled.
