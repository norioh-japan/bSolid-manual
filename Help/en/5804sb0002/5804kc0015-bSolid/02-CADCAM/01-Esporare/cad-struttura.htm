Machining operation timeline

Shows the tree list with the nodes of the machining operations.

In this area, machining operations can be ordered and simulated, ISOAcronym of the "International Standardisation Organisation", that determines technological and industrial standards. File format containing the programming language of the numerical control machine. files can be generated, etc.

The machining operation nodes can be moved using the drag and drop(drag and drop) Indicates the action of keeping the left mouse key pressed on a selection (object, icon, node, etc.) in order to drag it or move it. technique.

Each row of the tree list shows the string with the following information:

  •    Name of the piece.
  •    Name of the machining operation.
  •    Name of the tool used.

Example: [piece P1001] - P1006(Milling) - CAT1

This string indicates that the milling operation P1006 was carried out on piece P1001, using the tool CAT1.

The parameters of each single card are grouped according to the section.
To display them, explode the section by clicking on the vertical triangle C positioned before the title (figure 3).
