Description of the machining operation

Horizontal copying

CPH

Movement (expressed in millimetres) made by the scraper aggregate to ensure the lower tool C is in the correct working position (figure 2).

The positioning of the tool is managed in the same way as for the copier.

Copying speed

CPS

Used to define the speed at which the aggregate moves from the copying start point (defined in the "Machining start" data field) to the machining start point.

The machining start point is the point where the aggregate begins to work after completing the copying defined in the "Vertical copying" and "Horizontal copying" data fieldsA defined area containing data or commands (data field or command field)..

Joint movement

OFG

Transfer of the initial machining point (on the same profile trajectory) in relation to the initial programmed geometry point.

A positive value shifts the machining start point after the initial profile point, while a negative value shifts it forward.

Type of lead-in

PEI

Movement performed by the aggregate in order to approach the piece to be machined.

Front

The aggregate descends in front of the edge of the piece to be machined, and approaches the machining start point (ref. A, figure 5).

Side

The aggregate descends alongside the edge of the piece to be machined, and approaches the machining start point (ref. B, figure 5).
