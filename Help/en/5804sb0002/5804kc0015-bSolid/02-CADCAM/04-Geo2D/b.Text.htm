Position parameters

Path

PAT

Used to select the geometry on which to spread the text.

X

X

X co-ordinate of the start point of the linear text.

Y

Y

Y co-ordinate of the start point of the linear text.

Z

Z

Z co-ordinate of the start point of the linear text.

General parameters

Text

TXT

Used to insert different lines of text to appear in the drawing area.

Character

FNT

Type of character. Used to select the type of character required.
You are advised to use only the “True Type” font.

Height

HEI

Text dimensions.

Bold

WEI

Used to apply "bold" style, to emphasise the word. Choose one of the following options:

1: normal text.

2: bold text.

Italic

STY

Used to apply the "italics” style. Choose one of the following options:

1: normal text.

2: text in italics.

Alignment

ALN

Used define the type of text line alignment required, in relation to the start point and the margins of the outline.

  •    The default setting sees linear text flowing from left to right, beginning from the coordinates of its start point and extending towards the right.
  •    The default setting sees clockwise text beginning from the left margin and flowing along the left edge towards the right.
Left

The left or right margin of horizontal text is aligned to the start point or to the left margin of the outline (only for outlines created in a clockwise direction) and flows from left to right. See figure 1.

48. Left-hand alignments
A. Four aligned linear texts, referring to all the direction types: normal, low, high and reverse.
B. Two texts on the path, referring to two directions: above and below.
Centred

Centre-aligned text.
The text is positioned in the centre of the start point or at the centre of the outline. See figure 2.

49. Central alignment
A. Four linear texts, referring to all the direction types.
B. Two texts on the path, referring to two directions: above and below.
Right

The left or right margin of text is aligned to the start point or to the right margin of the outline (only for outlines created in a clockwise direction) and flows from right to left. See figure 3.

50. Right-hand alignments
A. Four aligned linear texts, referring to all the direction types: normal, low, high and reverse.
B. Two texts on the path, referring to two directions: above and below.
Stretched

Extended text, spread across the entire selected path.

Adapted

Returns the text which has been formatted with the "Stretched" option or the "Justified" option, to its normal state, i.e. the original state.

Justified

Aligns the texts uniformly to both margins of the outline.

Advanced parameters

Direction

DIR

Used to select the direction of the row of text in relation to the start point.

0

Straight horizontal text.

1

Vertical text flowing downwards, i.e. below the start-point or on the internal edge of the outline.

2

Horizontal reversed text.

3

Vertical text flowing upwards, i.e. above the start-point or on the external edge of the outline.

Constant spacing

KD

Enables (box tickedClick on a check box to tick it and thereby enable that function.) or disables the automatic insertion of a horizontal distance between the various characters that make up the text.

Spacing

DST

Used to move the characters of the text nearer or further away: a positive valuePredefined / pre-set. Standard information useful for the control, if the requested information is not entered. moves the characters further away, a negative value brings them closer.

Interline

LS

Inserts a vertical space between one line of text and the next.

Accuracy

PRC

Index to define the precision of the characters.

Stretch

STR

Allows to horizontally stretch the characters in the text. The dimension at 100% of the text is equivalent to the value 1,000; values higher than this expand the characters.
