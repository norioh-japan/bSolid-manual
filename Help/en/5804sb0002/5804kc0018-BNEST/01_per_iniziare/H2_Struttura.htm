Using the tables

The tables show the data, arranged into rows and columns. Each cell corresponds to a data fieldA defined area containing data or commands (data field or command field)..

Operations that can be performed in the table:

  •    To select a row, click(1) Indicates the action of pressing (and immediately releasing) the mouse button on an object (icon, button, etc.) located under the pointer on the screen. (2) (verb) Pressing and immediately releasing the left mouse button to enable the function of a selected command. on any part of that row.
  •    To make a multiple selection of all the rows, click on the first row then press the Shift key on the keyboard and click on the last row.
  •    To select scattered rows, click on the first row then press the Ctrl key on the keyboard and click on the other required rows.
  •    To make the columns bigger, position the cursorMobile element that is visible on the screen, and which moves as the mouse is moved. between two titles of the column. When the double arrow appears, press the left mouse key and drag the cursor.
  •    To show or hide the columns, click on the commandTool designed to perform a specific function/operation. For example, the “Print” command allows you to print a document. and activate the required columns.

The filter window

The filter window is used to carry out any type of search in the databaseData bank. Stored data arranged in an organised manner so that they are readily available using a search and management program., showing only the characteristics defined in that filter.
For example, if a filter is set in the graphic/table area of the projects, only the required ones will be listed.

Indicate the name in the appropriate data field.


The white triangle next to the filter field opens other search types: filter based on to the piece thickness and the wood type.
