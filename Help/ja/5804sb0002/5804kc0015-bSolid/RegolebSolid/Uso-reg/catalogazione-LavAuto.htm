自動加工作業用ルールの設定

説明

自動加工作業は、形状から始まる設定された加工作業のセットであり、プログラミングを容易にします。自動加工作業の設定は、考えられる加工作業を自動的に生成するため、システムがCad/ Cam環境で作成された図面にそれを適用できるように、異なるタイプの形状にルールを割当てることを意味します。

新規設定の作成

新規の自動加工作業を作成するには、以下のように進めてください。

1. ボタンをクリック（1）画面上のポインタの下にあるオブジェクト（アイコン、ボタンなど）の上でマウスボタンを押す（そしてすぐに離す）行為を示します。（2）（動詞）選択したコマンドの機能を有効にするため、マウスの左ボタンを押してすぐに離します。します。
2. ボタンをクリックして新規のルールを追加します。新規規則ダイアログボックスが開きます。
3. データフィールドに入力します。

形状の選択

ルールを適用すべき形状のリスト。必要な形状でクリックして、ボタンをクリックする。

加工作業の選択

以前選択した形状と関連づける加工作業のリスト。必要な加工作業でクリックして、ボタンをクリックする。

説明

作成したルールの説明。

4. ボタンをクリックして次のデータページを可視化します。
5. ボタンをクリックして設定を保存し、ダイアログボックスを閉じてください。環境テーブルに、新規のルールが含まれた行が表示されます。
6. "ルールの詳細"セクションおよび機械加工パラメーターエリア形状パラメーターと技術パラメーターを管理するためのカードを備えたデータエリアの一部。のデータフィールドデータまたはコマンドを含む所定のエリア（データフィールドまたはコマンドフィールド）。に入力します。

挿入されたルールは、テーブルで、緑のボタンが"有効"列に表示された場合にのみ検証済みになります。赤のボタンは、ルールが完了していないことを示します（データの欠落）。

自動加工作業の例

注：

木工機械に有効な例

ボーリング生成のために位置に適用されるルール

以下に定義されるルールは、基準コーナー番号1を含むピースの面2上に作成されたすべての位置に適用されます。

362. ピースの面2に描かれた位置
1. ボタンをクリックして新規のルールを追加します。
新規規則ダイアログボックスが開きます。
2. 形状の選択ページのポイントオプションをクリックしてから、ボタンをクリックします。
3. 加工作業の選択ページのボーリング（穴あけ）加工作業をクリックしてから、ボタンをクリックします。
4. 短い説明文を入力してから、ボタンをクリックして確定します。
5. "ルール"セクションで、"Deルール"オプションを選択し、隣接するデータフィールドで以下の構文を入力します。FCRN=”2（1）”

FCRNは"面と角"データフィールドのパラメーターです。2は面番号2を示します。1は基準コーナーを示します。

6. ボーリングデータエリアデータ入力用の特定のエリアを定義する一般的な用語です。では、すべての位置にルールを適用するために、以下のデータを設定します。
• "ツールを使用"データフィールドの "0： 直径"オプション。
• "ツールタイプ"データフィールドのビットのタイプ。
• "直径"データフィールドの直径、およびボア深さ。
ボーリング生成のために円形に適用されるルール

以下のルールは、ピースの面上で作成された半径14mm未満のすべての円形に適用されます。 これらの円形は、直径によって、単一の加工作業の中に一緒にグループ化されます。

363. 例：8個の円形;4個が直径5mmで4個が直径8mm。
1. ボタンをクリックして新規のルールを追加します
新規規則ダイアログボックスが開きます。
2. 形状の選択ページの円オプションをクリックしてから、ボタンをクリックします。
3. 加工作業の選択ページのボーリング（穴あけ）加工作業をクリックしてから、ボタンをクリックします。
4. 短い説明文を入力してから、ボタンをクリックして確定します。
5. "ルールの詳細"セクションの"収集側"データフィールドで、"面"オプションを選択します。
6. "ルールの詳細"セクションの"収集ルール"データフィールドで、パラメーターDを入力します。

このパラメーターは円形の直径を示し、これにより、ボアが直径によってグループ化される必要があることが確認されます。

7. "ルール"セクションで、"Deルール"オプションを選択し、隣接するデータフィールドで以下の構文を入力します。（R<14）

Rは、ルールが適用されるために14mm未満である必要がある 、円形の"半径"データフィールドのパラメーターです。

8. ボーリングデータエリアでは、14mm未満のすべての円形にルールを適用するために、"ツールを使用"データフィールドにアクセスし 、自動オプションを選択します。この場合、ツールは円形の直径に基づいて自動的に選択されます。ボア深さも同様に設定します。

例えば、直径5mmの4個のボアと直径8mmの4個のボアがCad/ Cam環境（図2）に描かれている場合、またこの自動加工作業が適用される場合、2つのボーリング作業が生成されます。1つは直径5mmの4個の円形を一緒にグループ化し、もう1つは直径8mmの4個の円形をグループ化します。

カット生成のために線に適用されるルール

以下のルールは矩形と交差する線に適用されます。

364. 例：矩形と交差する線。
1. ボタンをクリックして新規のルールを追加します。
新規規則ダイアログボックスが開きます。
2. 形状の選択ページのラインオプションをクリックしてから、ボタンをクリックします。
3. 加工作業の選択ページのカット加工作業をクリックしてから、ボタンをクリックします。
4. 短い説明文を入力してから、ボタンをクリックして確定します。
5. "ルール" セクションで"交差エッジルール"オプションを選択します。
6. ボックスを確認します"有効な補正"。
7. カットデータエリアで、ツール（"ツール名"データフィールド）および加工作業深さ（"深さ"データフィールド）を設定します。

この場合、線はカット（図3）と認識されますが、それは"交差エッジルール"オプションが、クローズド形状と交差するすべての線はカットにしなければならないと見なすルールで設定されるためです。

フライス加工作業生成のためにクローズドポリラインに適用されるルール

以下のルールをポリラインに適用し、ピースに変形させます。

365. 例
A. プロファイル形状
B. 内部形状
C. 適用された自動加工作業の結果
1. ボタンをクリックして新規のルールを追加します。
新規規則ダイアログボックスが開きます。
2. 形状の選択ページのポリライン終端部で一緒にリンクされ、単独のオブジェクトとして扱われる1つ以上の連続セグメントで構成される図を作成するためのコマンド。オプションをクリックしてから、ボタンをクリックします。
3. 加工作業の選択ページのフライス加工加工作業クリックしてから、ボタンをクリックします。
4. 短い説明文を入力してから、ボタンをクリックして確定します。
5. "コンテナ形状"ボックスを有効にします。
6. "ルール"セクションで、"フェースルール"オプションを選択します。算術演算子“イコール”（=）を選択し、隣接するデータフィールドに番号0を入力します。

この場合、他の形状を含む（例えばB参照、図4）プロファイルポリライン（A参照、図4）がピースに変形され、プロファイルされます（C参照、図4）。

彫刻作業作成のためにポリラインに適用されるルール

以下のルールは、ピースのゼロ面に描かれた2本のポリラインの彫刻に適用されます。この場合、ピースのゼロ面にポリラインを描き、彫刻という語を両方の形状の"形状名"データフィールドに入力します。この名称は、彫刻を適用するために自動加工作業で使用されるので便利です。

366. 例
A. 刻み目をつける形状
B. 適用された自動加工作業の結果
1. ボタンをクリックして新規のルールを追加します。
新規規則ダイアログボックスが開きます。
2. 形状の選択ページのポリラインオプションをクリックしてから、ボタンをクリックします。
3. 加工作業の選択ページのフライス加工加工作業クリックしてから、ボタンをクリックします。
4. 短い説明文を入力してから、ボタンをクリックして確定します。
5. "ルール"セクションで、"Deルール"オプションを選択し、隣接するデータフィールドで以下の構文を入力します。 NMC=”彫刻”。

この場合、"形状名"詳細なカードの データフィールド内に彫刻という語を有するすべての形状にルールが適用されます。

6. フライス加工データエリアで、彫刻を作成するためのフライス（ツールデータフィールド）および加工作業深さ（深さデータフィールド）を設定します。
