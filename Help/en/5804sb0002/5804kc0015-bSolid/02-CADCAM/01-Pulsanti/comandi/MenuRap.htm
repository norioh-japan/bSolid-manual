Quick menu commands

The quick menu in the drawing area

To view the quick menuList of options that appears on the screen when you press the right-hand mouse key on an area of the application software. The menu commands vary, depending on the context displayed or selected. in the drawing areaArea dedicated to the creation and modification of the geometries/machining operations., select any objectGeneral term used to define any drawing created in the drawing area (e.g. piece, texts, flat figures, etc.). and click(1) Indicates the action of pressing (and immediately releasing) the mouse button on an object (icon, button, etc.) located under the pointer on the screen. (2) (verb) Pressing and immediately releasing the left mouse button to enable the function of a selected command. with the right-hand mouse buttonCommand field that is responsive to mouse clicks and that enables operations to be activated..

Boxes and options of the quick menu

View

A box containing the list of the most common options for managing the viewsEnables an object to be displayed from any point of space..
For the description of the commands and procedures, refer to: "View management tools" and "Managing the views".

Creation

A box containing the list of the most common options for creating an object:

Snaps

Shows the list for activating the SnapConnects the point of the new outline to the elements of the existing objects, such as the end of a line, the centre of a circle, the mid-point of a side, etc. tools (see "Snap tools").

Element type

Shows the list for choosing the next geometric elementAn abstract element associated with specific properties: the point, line, plane and space are all geometric elements. (only for polylinesSet of consecutive straight lines, connected to each other at the ends.).

Close polyline

Closes the polylineSet of consecutive straight lines, connected to each other at the ends..

Close FreeForm

Closes the FreeForms.

Ok

Terminates the creation of the object.

Undo

Annuls the creation of the object.

Modify

Box containing a list of the most common options for modifying (copying, pasting, deleting etc.) the selected objects.
For the description of the commands, refer to: "Standard bar ".

Transform

A box containing the list of the most common options for modifying the selected objects.
The list varies according to the type of object selected.
For the description of the commands, refer to: "CAD options bar"; "Drawing bar".

Piece

A box containing the list of the most common options for creating a piece:

Face selection

Activates the cross cursorType of cross-shaped cursor. for selecting the objects.

Piece from finished geometry

Used to generate the shape of the rough piece, starting from a selected flat figureTwo-dimensional geometric figure (e.g. polygon, circle, etc.) belonging to a single plane..

For the description of the commands, see "Piece ".

Pieces from finished geometries

Opens the dialogue box for generating the piece shapes, starting from several flat figures selected.

For the description of the commands, see "Piece ".

Machining operations

A box containing the list of the machining operations to be applied to the selected object.
For the description of the commands, refer to: "Machining operations bar".

The quick menu on a tree list node

To view the quick menuList of commands (to be selected with the mouse) that develop a sub-menu, i.e. a list of options that allow certain functions/operations to be carried out. in the data areaGeneral term to define a specific area for data entry., select a single nodeFor the definition, refer to Tree list. of the tree listStructure containing a number of items, known as nodes, which in turn have branches and secondary branches like a tree. The node at the highest level is called the root. All the others derive from one single node and, in turn, can have other nodes. and click with the right-hand mouse button.

Quick menu options

Show all

Enables the graphic visualisation of all the objects in the tree list; all the nodesFor the definition, refer to Tree list. have the bulb lit up .

Hide all

Disables the graphic visualisation of all the objects in the tree list; all the nodes have the bulb switched off .

Show selected objects

Enables the graphic visualisation of only those objects selected from the tree list; the selected nodes have the bulb lit up .

Hide selected objects

Disables the graphic visualisation of only those objects selected from the tree list; the selected nodes have the bulb switched off .

Enable all

Selects all the nodes in the tree list; the relative objects of the drawing area change colour to show they have been selected.

Disable all

Deactivates the selection of previously selected nodes.

Expand all

Automatically opens all the nodes of the tree list, showing the single internal elements.

Compress all

Automatically closes all the nodes, hiding the elements inside them.

Cut

For the description, refer to: "General tools".

For the description, refer to: "General tools".

Paste

For the description, refer to: "General tools".

Delete

For the description, refer to: "General tools".

Modify

Opens the parameters areaPart of the data area with the cards for managing the geometric and technological parameters. to modify the data associated with the selected node.

Rename

Renames the selected node.

Remove from the unit

Separates the machining operations or geometries that have been grouped together.

Set executability

For the description of the commandTool designed to perform a specific function/operation. For example, the “Print” command allows you to print a document., refer to the "Execute machining" or "Execute geometry" data fieldA defined area containing data or commands (data field or command field)..
