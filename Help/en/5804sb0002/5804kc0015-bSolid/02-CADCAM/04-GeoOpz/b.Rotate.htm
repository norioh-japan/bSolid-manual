Rotate

Description

Rotates the selected objectGeneral term used to define any drawing created in the drawing area (e.g. piece, texts, flat figures, etc.)..

This commandTool designed to perform a specific function/operation. For example, the “Print” command allows you to print a document. can be used to generate a rotated copy of the original.

Associated procedure
1. Select the geometry to be rotated.

2. Click(1) Indicates the action of pressing (and immediately releasing) the mouse button on an object (icon, button, etc.) located under the pointer on the screen. (2) (verb) Pressing and immediately releasing the left mouse button to enable the function of a selected command. on the buttonCommand field that is responsive to mouse clicks and that enables operations to be activated. of the CAD options barBar listing the optional tools useful for drawing (e.g. the commands to rotate or resize the drawing, etc.).

To enable the copy, click on the Enable copy box of the data area.

3. Click on a point in the area to establish the centre of rotation.

4. Drag the cursorMobile element that is visible on the screen, and which moves as the mouse is moved. to create the rotated copy. The rotated object will be shown in red.

5. Click to lock the object rotation in the required point. The rotated object will turn black.

List of data fields

Enable copy

EC

Enables or disables the copy of the object to be rotated.

Rotation axis

AXS

Used to select the axis around which the selected object will be rotated. To choose the axis, click on one of the following options:

X axis

Rotates the object around the X-axis.

Y axis

Rotates the object around the Y-axis.

Z axis

Rotates the object around the Z-axis.

X

X1

X co-ordinate of the rotation point.

Y

Y1

Y co-ordinate of the rotation point.

Z

Z1

Z co-ordinate of the rotation point.

Rotation angle

ANG

Rotation of the object, starting from the rotation point. Enter the required angle.
