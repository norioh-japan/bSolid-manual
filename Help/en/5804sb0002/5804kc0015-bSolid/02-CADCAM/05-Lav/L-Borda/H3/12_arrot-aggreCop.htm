Rounding with aggregate copier AGGRE122

Description of the machining operation

Used to round off the corners of the edgebanded piece, using the aggregate(tool) Device fitted with tools. with copier (AGGRE122 model).

For the description of the aggregate, refer to the relative aggregate user instructions.
