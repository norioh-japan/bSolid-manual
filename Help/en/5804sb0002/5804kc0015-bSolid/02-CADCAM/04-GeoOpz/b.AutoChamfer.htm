Corner type

CHT

Used to chamfer single intersecting elements or several open intersecting polylinesSet of consecutive straight lines, connected to each other at the ends..

92. Example
A. Single starting elements selected
B. Longest paths; two intersecting polylines are generated.
C. Nearest points; two separate polylines are generated.
Longest paths

At each intersection point, the system searches for the longest path and applies the chamfers in the non-tangent points between one element and the next (ref. B, figure 2).

Nearest points

At each intersection point, the system searches for the nearest element and applies the chamfers in the non-tangent points between the two elements (ref. C, figure 2).

No optimisation

Apply the chamfers in the non-tangent points, without optimising the path.
