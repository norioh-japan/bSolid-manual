Morphology

Description of the tab

Shows the data fieldsA defined area containing data or commands (data field or command field). for putting together the tool: the bodies, shape and size.

The complete list of all the data fields of the card Morphologyis given below; their visualisation in the interfaceVisible part of a program which - with the aid of icons, fields, buttons etc. - allows access to the various system functions. varies according to the type of tool selected or created.

Type of connection device

ConeType

List of the existing types of connection device (only for tools to be installed in the electrospindles). From the list, choose the type to be associated with the tool. Once you have made your choice, both drawings (3DThree-dimensional geometric figures (constructed on three dimensions). and positioned) are updated.

386. Coupling
A. ISO30 coupling
B. HSK coupling
Body type

KERNEL_TypeSelector

Allows to choose the type of geometry of the tool body.

  •    Spherical

Spherical shape (e.g. finishing tools). Defines all types of mills or grinding wheels with a semi-sphere bit.

  •    Conical

Tapered shape (e.g. tools for boring or engraving). Defines all types of mills or grinding wheels with an overturned cone or hourglass shape.

  •    Cylinder

Cylindrical shape (e.g. a tool with a flat bit). Defines all types of cylinder-shaped mills or grinding wheels with a flat bit.

  •    From the geometry

Geometric shape imported from a drawing that shows its right part in relation to the tool axis (e.g. when the tool has a blade). Permitted file formats: DXFAcronym of "Drawing Interchange Format". Type of two-dimensional Vector graphics file used as a graphic data exchange format between the various CAD systems., ICPType of file native to the ICAM application software., BSOLID.

  •    LFC Shape

LFC (Line Fillet ChamferChamfering produces a corner. The chamfer can be linear or rounded.). Geometric form with a part of the profile chamfered or connected (e.g. chamfering tool).

  •    Helix

Option active only for the thread tools of class Tapper"Male" tool used to thread, i.e. to generate the female threads inside the holes for the introduction of screws..

  •    Comb

Option active only for the thread tools of class Tapper.
