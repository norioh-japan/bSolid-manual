Direction tools

Showing or hiding - in the drawing area - the arrows that indicate the two directions (tool advance and outline creation).

Sets the tool movement direction, from right to left, on all selected open outlines.
In this case, the start point of all selected outlines is inverted when the arrow is pointing to the left.

Sets the tool movement direction, from left to right, on all selected open outlines.
In this case, the start point of all selected outlines is inverted when the arrow is pointing to the right.

Sets the tool movement in a clockwise direction on all selected closed outlines.
In this case, the start point of all selected outlines is inverted when the arrow is pointing in a clockwise direction.

Sets the tool movement in an anti-clockwise direction on all selected closed outlines.
In this case, the start point of all selected outlines is inverted when the arrow is pointing in an anti-clockwise direction.

Invert the tool advance direction. The selected outlines for which the arrow was pointing right will now have it pointing left, and vice versa.
