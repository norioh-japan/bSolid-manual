# 寸法と注釈

**ファイル名**: `02-05-03_dimension.md`  
**カテゴリ**: 編集ツール - 寸法と注釈

## 概要

寸法と注釈機能は、設計図面に必要不可欠な寸法線、測定値、説明テキストを追加するためのツール群です。正確な製造指示、設計意図の伝達、図面の完成度向上に重要な役割を果たします。

## 測定ツール

### 長さの測定 ![長さ測定アイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Utility_Measure_Lenght.png)

線分、円弧、ポリラインなどの要素の長さを測定します。

**操作手順**:
1. 測定バーの ![長さ測定アイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Utility_Measure_Lenght.png) ボタンをクリック
2. 測定したい要素を選択
3. 結果がデータエリアに表示される

**測定可能な要素**:
- 直線（ライン）
- 円弧（アーク）
- ポリライン
- スプライン曲線
- 円周（全周）

### 角度の測定 ![角度測定アイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Utility_Measure_Angle.png)

2つの線分間の角度を測定します。

**操作手順**:
1. 測定バーの ![角度測定アイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Utility_Measure_Angle.png) ボタンをクリック
2. 第1の線分を選択
3. 第2の線分を選択
4. 角度値がデータエリアに表示される

**角度表示単位**:
- 度（°）：0～360度
- ラジアン：0～2π
- グラジアン：0～400グラジアン

### 2点間距離の測定 ![2点距離測定アイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Utility_Measure_Distance2Points.png)

任意の2点間の直線距離を測定します。

**操作手順**:
1. 測定バーの ![2点距離測定アイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Utility_Measure_Distance2Points.png) ボタンをクリック
2. 第1の点をクリック
3. 第2の点をクリック
4. 距離値がデータエリアに表示される

**測定情報**:
- **直線距離**: 2点間の最短距離
- **X方向距離**: X軸方向の距離成分
- **Y方向距離**: Y軸方向の距離成分
- **Z方向距離**: Z軸方向の距離成分（3D）

### 要素間距離の測定 ![要素間距離測定アイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Utility_Measure_Distance2Entities.png)

2つの要素間の最短距離を測定します。

**操作手順**:
1. 測定バーの ![要素間距離測定アイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Utility_Measure_Distance2Entities.png) ボタンをクリック
2. 第1の要素を選択
3. 第2の要素を選択
4. 最短距離が表示される

**距離タイプ**:
- **平行要素**: 平行距離
- **同心円**: 半径差
- **点と線**: 垂直距離
- **線と円**: 中心からの距離

### 面積の測定 ![面積測定アイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Utility_Measure_Area.png)

閉じた図形の面積を測定します。

**操作手順**:
1. 測定バーの ![面積測定アイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Utility_Measure_Area.png) ボタンをクリック
2. 閉じた図形を選択
3. 面積値がデータエリアに表示される

**測定可能な図形**:
- 円・楕円
- 矩形・多角形
- 閉じたポリライン
- 複合図形
- 3D面

**表示項目**:
- **面積**: 図形の面積
- **周囲長**: 図形の周囲の長さ
- **重心**: 図形の重心座標

## 寸法記入ツール

### 水平寸法線 ![水平寸法アイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Utility_Dimension_Horizontal.png)

水平方向の寸法線を作成します。

**操作手順**:
1. 寸法バーの ![水平寸法アイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Utility_Dimension_Horizontal.png) ボタンをクリック
2. 第1の基準点を選択
3. 第2の基準点を選択
4. 寸法線の位置を指定

**パラメーター**:
- **距離**: 寸法線と基準線の距離
- **値**: 表示する寸法値
- **精度**: 小数点以下の桁数
- **単位**: mm、inch等の単位表示

### 垂直寸法線 ![垂直寸法アイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Utility_Dimension_Vertical.png)

垂直方向の寸法線を作成します。

**操作手順**:
1. 寸法バーの ![垂直寸法アイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Utility_Dimension_Vertical.png) ボタンをクリック
2. 第1の基準点を選択
3. 第2の基準点を選択
4. 寸法線の位置を指定

### 傾斜寸法線 ![傾斜寸法アイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Utility_Dimension_Aligned.png)

任意の角度の寸法線を作成します。

**操作手順**:
1. 寸法バーの ![傾斜寸法アイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Utility_Dimension_Aligned.png) ボタンをクリック
2. 第1の基準点を選択
3. 第2の基準点を選択
4. 寸法線の位置を指定

**特徴**:
- 基準点間の実際の距離を表示
- 寸法線は基準線に平行
- 任意の角度に対応

### 角度寸法線 ![角度寸法アイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Utility_Dimension_Angular.png)

角度の寸法線を作成します。

**操作手順**:
1. 寸法バーの ![角度寸法アイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Utility_Dimension_Angular.png) ボタンをクリック
2. 第1の線分を選択
3. 第2の線分を選択
4. 角度弧の位置を指定

**角度表示オプション**:
- **内角**: 2線分間の内角
- **外角**: 2線分間の外角
- **補角**: 180°からの差
- **余角**: 90°からの差

### 半径寸法線 ![半径寸法アイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Utility_Dimension_Radial.png)

円・円弧の半径寸法線を作成します。

**操作手順**:
1. 寸法バーの ![半径寸法アイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Utility_Dimension_Radial.png) ボタンをクリック
2. 円または円弧を選択
3. 寸法線の位置を指定

**表示オプション**:
- **半径**: R + 数値
- **直径**: φ + 数値
- **中心マーク**: 中心点の表示

### 寸法指示線 ![指示線アイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Utility_Dimension_Indicator.png)

説明テキストへの指示線を作成します。

**操作手順**:
1. 寸法バーの ![指示線アイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Utility_Dimension_Indicator.png) ボタンをクリック
2. 指示したい箇所をクリック
3. 指示線の方向を指定
4. テキスト位置を指定

**指示線タイプ**:
- **直線**: 単純な直線指示線
- **折れ線**: 複数セグメントの指示線
- **曲線**: 滑らかな曲線指示線
- **矢印**: 矢印付き指示線

## テキスト注釈ツール

### 直線テキスト ![直線テキストアイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Draw_Text_Linear.png)

水平または任意角度のテキストを作成します。

**操作手順**:
1. テキストバーの ![直線テキストアイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Draw_Text_Linear.png) ボタンをクリック
2. テキストの開始点をクリック
3. データエリアでテキスト内容と書式を設定

**基本パラメーター**:
- **テキスト内容**: 表示する文字列
- **フォント**: フォントの種類
- **サイズ**: 文字の高さ（mm単位）
- **角度**: テキストの回転角度

**配置オプション**:
- **左揃え** ![左揃えアイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Draw_Text_Just_Left.png): テキストを左に配置
- **中央揃え** ![中央揃えアイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Draw_Text_Just_Center.png): テキストを中央に配置
- **右揃え** ![右揃えアイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Draw_Text_Just_Right.png): テキストを右に配置

### パス上テキスト ![パス上テキストアイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Draw_Text_OnPath.png)

曲線に沿ったテキストを作成します。

**操作手順**:
1. テキストバーの ![パス上テキストアイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Draw_Text_OnPath.png) ボタンをクリック
2. 基準となる曲線を選択
3. テキストの開始位置を指定
4. データエリアでテキスト内容を設定

**パス要素**:
- **直線**: 線分に沿ったテキスト
- **円弧**: 円弧に沿ったテキスト
- **ポリライン**: 複合曲線に沿ったテキスト
- **スプライン**: 滑らかな曲線に沿ったテキスト

**配置制御**:
- **開始位置**: パス上のテキスト開始点
- **間隔**: 文字間の距離
- **上下反転**: テキストの表裏
- **パス追従**: 曲線の湾曲に追従

## 寸法スタイルの設定

### 寸法線の書式

**寸法線の外観**:
- **線種**: 実線、破線、一点鎖線等
- **線幅**: 寸法線の太さ
- **色**: 寸法線の色
- **延長線**: 寸法線の延長長さ

**矢印の設定**:
- **矢印タイプ**: 黒塗り、白抜き、ドット等
- **矢印サイズ**: 矢印の大きさ
- **矢印角度**: 矢印の開き角

### 寸法値の書式

**数値表示**:
- **精度**: 小数点以下の桁数
- **単位**: mm、inch、m等
- **単位位置**: 前置、後置、なし
- **丸め**: 指定値での丸め処理

**テキスト書式**:
- **フォント**: 数値のフォント
- **サイズ**: 数値の文字サイズ
- **色**: 数値の色
- **背景**: 数値背景の設定

### 位置と配置

**寸法線位置**:
- **自動配置**: 最適位置への自動配置
- **手動配置**: ユーザー指定位置
- **間隔制御**: 複数寸法線の間隔

**テキスト位置**:
- **線上**: 寸法線上に配置
- **線外**: 寸法線外に配置
- **中央**: 寸法線中央に配置
- **任意**: 任意位置に配置

## 品質管理とチェック

### 検証ツール

#### 接触のチェック ![接触チェックアイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Utility_Measure_CheckTangency.png)

2つの曲線の接触状態を確認します。

**検証項目**:
- **接触点**: 接触している点の座標
- **接触角度**: 接触点での角度
- **継続性**: 滑らかな接続の確認

#### 連続性のチェック ![連続性チェックアイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Utility_Measure_CheckContinuity.png)

曲線の連続性を確認します。

**検証内容**:
- **G0連続性**: 位置の連続性
- **G1連続性**: 接線の連続性
- **G2連続性**: 曲率の連続性

#### 小要素のチェック ![小要素チェックアイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Utility_Measure_SmallEntities.png)

製造上問題となる小さな要素を検出します。

**検出対象**:
- **短線分**: 指定長さ以下の線分
- **小円弧**: 指定半径以下の円弧
- **狭角**: 指定角度以下の角度

### Zレベルチェック ![Zチェックアイコン](../../../Help/ja/5804sb0002/FIGURE/15-icone/b15b0001/Utility_Measure_VerifyZ.png)

3D要素のZ座標を確認します。

**確認項目**:
- **Z値の一貫性**: 同一平面上の要素確認
- **高さの妥当性**: 設計仕様との整合性
- **層間の関係**: 複数層間の位置関係

## 作業効率向上のテクニック

### 一括操作

**寸法線の一括作成**:
1. 複数の基準点を連続選択
2. 自動的に連続寸法線を作成
3. 統一された書式で表示

**スタイルの一括適用**:
1. 寸法スタイルを定義
2. 複数の寸法線を選択
3. スタイルを一括適用

### テンプレートの活用

**寸法スタイルテンプレート**:
- 会社標準スタイルの保存
- プロジェクト固有設定の管理
- 図面タイプ別設定の使い分け

**自動配置ルール**:
- 寸法線の自動配置設定
- 間隔の自動調整
- 重複回避の自動処理

## エラー対処法

### よくある問題と解決策

**問題**: 寸法値が正しく表示されない
**解決策**:
- 単位設定を確認
- 精度設定を確認
- 基準点の選択を見直し

**問題**: 寸法線が重なってしまう
**解決策**:
- 自動配置設定を有効化
- 手動で位置を調整
- 寸法線間隔を調整

**問題**: テキストが読みにくい
**解決策**:
- フォントサイズを拡大
- 色設定を確認
- 背景設定を調整
