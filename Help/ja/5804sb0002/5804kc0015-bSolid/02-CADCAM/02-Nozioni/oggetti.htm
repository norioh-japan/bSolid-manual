オブジェクトの形およびサイズを修正する

1. ダブルクリックをして修正するオブジェクトを選択すると、オブジェクトが赤に変わる。
2. 十字のカーソル画面上に表示される可動エレメントであり、マウスの移動に合わせて移動します。 を修正する図またはアウトラインの上に当て、 マウスを動かす。値事前定義/プリセット。要求された情報が入力されていない場合、制御に便利な標準的な情報。を入力するには、データエリアの各形状のデータフィールドデータまたはコマンドを含む所定のエリア（データフィールドまたはコマンドフィールド）。を使用します。

オブジェクトを扱う手順のいくつかを以下で説明する。

鏡面反射コピーを作成してオブジェクトを複製する
1. 重複させる形状を選択します。

2. CADオプションバーのボタンをクリックします。
3. データエリアの"コピーは有効"ボックスを選択します。
4. 第1基準点をクリックし、カーソルをドラッグしてコピー用の中心軸を作成する。コピーは赤色で表示されます。

5. 第2基準点をクリックし、中心軸を確立します。コピーは黒色になります。

オブジェクトを回転する
1. 回転させる形状を選択します。

2. CADオプションバーのボタンをクリックします。

コピーを有効にするには、データエリアのコピーは有効ボックスをクリックします。

3. エリア内の点をクリックし、回転中心を確立します。

4. カーソルをドラッグして回転させたコピーを作成します。回転オブジェクトは赤色で表示されます。

5. 必要な点でクリックし、オブジェクトの回転を止めます。回転オブジェクトは黒色になります。

オブジェクトを縮小拡大する
1. スケーリングする形状を選択します。

2. CADオプションバーのボタンをクリックします。

コピーを有効にするには、データエリアのコピーは有効ボックスをクリックします。

3. エリア内の点をクリックし、スケールの中心を確立します。

4. カーソルをドラッグしてスケーリングしたコピーを作成します。スケーリングしたオブジェクトは赤色で表示されます。

5. 必要な点でクリックし、オブジェクトのサイズ変更を止めます。オブジェクトは黒色になります。

直線の面取りまたはコネクタを挿入する
1. 必要な形状を選択します。

2. ボタンをクリックして線形面取りを挿入するか、ボタンをクリックして曲線面取りを挿入します（ CADオプションバー）。
コピーを有効にするには、データエリアの該当ボックスをクリックします。
3. オブジェクトの第1側面をクリックします。該当するデータフィールドで、線形面取りまたは円弧の寸法を設定します。

4. オブジェクトの第2側面をクリックすると、オブジェクトは黒色になります。

オブジェクトを選択する

描画エリア形状/加工作業の作成と変更専用のエリア。のすべてのオブジェクトを選択するには、データエリアの ボタンをクリックします。

描画エリアのオブジェクトを選択すると、関連するノード定義については、ツリーリストを参照します。がデータエリアにハイライトされます。

十字のポインターで複数のオブジェクトを選択する
1. ボタンをクリックする（十字のカーソル）。
2. マウスの左キーを押します。
3. 画像の上にカーソルを当ててドラッグし、必要なオブジェクトの周りに選択エリアを作成します。
十字のポインターで1回につき1つのオブジェクトを選択する
1. ボタンをクリックする（十字のカーソル）。
2. アクティブ化したい様々なオブジェクトをクリックします。
ピースまたは表面に変換したい形状を選択する

様々な方法で、変換したい形状を選択することができます。

  •    描画エリアで形状を選択し、必要な描画ツールを選択し、コマンドエリアの記号をクリックします。
  •    ピースコマンドまたは面ツールを選択し、コマンドエリアのカウンターツールを使用して描画エリアの形状を選択します。
  •    ピースコマンドまたは面ツールを選択し、ドロップダウンメニュードロップダウンボタンをクリックすると、垂直または水平方向に広がるオプションまたは手段のリスト。を使用して形状を選択します。

オブジェクトの選択解除

選択したすべてのオブジェクトを選択解除するには、データエリアのボタンをクリックします。

十字のポインターでオブジェクトの選択解除をする
1. ボタンをクリックする（十字のカーソル）。
2. 選択解除をするオブジェクトをクリックします。

オブジェクトを削除する

描画エリアからオブジェクトを削除するには、ツリーリストノードと呼ばれる複数の項目を含む構造。木のように分岐（枝）や二次分岐があります。最も高レベルにあるノードはルートと呼ばれます。そのたすべては単独のノードから派生し、他のノードを持つことができます。または描画エリアからオブジェクトを選択し、以下のいずれかの方法を使います：

  •    キーボードのCanc キーを押すと、オブジェクトが削除されます。
  •    ボタンをクリックして、オブジェクトを削除します。

最後の操作を取り消すことができ、うっかり削除したオブジェクトを復元することができます。そのためには、ボタンをクリックします。

オブジェクトを切り取ってコピーする

描画エリアからオブジェクトを切り取ってコピーする手順：

1. ツリーリストまたは描画エリアからオブジェクトを選択し、以下のいずれかの方法を使います：
• ボタンをクリックして、オブジェクトを切り取る。あるいは、キーボードのCtrl + Xキーを押します。
• ボタンをクリックして、オブジェクトをコピーする。あるいは、キーボードのCtrl + Cキーを押します。
2. 切り取ったまたはコピーされたオブジェクトを貼り付ける（オブジェクトを貼り付ける参照）。

最後の操作を取り消すことができ、切り取ったオブジェクトを復元することができます。そのためには、ボタンをクリックします。

オブジェクトを貼り付ける

描画エリアにあるオブジェクトを貼り付けるには、ボタンをクリックします。あるいは、キーボードのCtrl + Ｖキーを押します。

この場合、貼り付けるオブジェクトは、赤色で表示されます。十字のカーソルが表示され、位置を選択することができます。

最後の操作を取り消すことができ、切り取ったオブジェクトを復元することができます。そのためには、ボタンをクリックします。

オブジェクトを表示および非表示にする

描画エリアで作成したオブジェクトを表示あるいは非表示にする手順：

  •    右のツリーリストにアクセスして、電球の記号をクリックして、あるレベルに属する特定のオブジェクトのみを非表示または表示します。

黄色の電球 は、ノードがアクティブで、そのノードが含むオブジェクトは描画エリアで見ることができることを示します。

灰色の電球 は、ノードがアクティブでなく、そのノードが含むオブジェクトは描画エリアで見ることができないことを示します。

  •    あるレベルのすべてのオブジェクトを表示させるまたは非表示にするかは、ステータスバーの点灯している電球上に現れます。

電球が点灯していたら 、それは、レベルがアクティブである（オブジェクトを見ることができる）ことを示します。

電球が消灯していたら 、それは、レベルがアクティブでない（オブジェクトが非表示になっている）ことを示します。

オブジェクトの色を修正する

ある特定のレベルに属するプロファイルの線の色は、レベルそのものの中で決められる。特定のオブジェクトの線を修正するには、非標準色を採用し、ツールを使用すること。

ワイヤーフレーム（ドア/窓）壁または脇柱に取り付け、窓を吊り下げる窓構造物。 表示およびフォトリアリスティック表示は、カラーパレットで選択した色を使用して修正することができます。

以下のように進めます。

1. ボタンをクリックする（十字のカーソル）。
2. 色付けしたいオブジェクトのみを選択します。
3. ステータスバーのボタンをクリックします。
4. カラーパレットから色を選択し、確定します。

カラーパレットから選択した色はレベル全体に適用されるのではなく、選択したオブジェクトのみに適用されます。

画像の色

描画したオブジェクト（図形、面など）の色は、レベル内またはカラーパレット内で定義される（オブジェクトの色を修正する 参照）。

オブジェクトに適用される加工作業の色はレベル内で修正することはできず、ツールそのものの設定によって決まります。
しかし、赤と緑の2つの標準色があり、ツールの進入パス（緑）および最高速度のツールのパス（赤）を示します。
