Snap tools

“MagnetConnects the point of the new outline to the elements of the existing objects, such as the end of a line, the centre of a circle, the mid-point of a side, etc.” commands for connecting the various geometric elements: search for mid-points, outline extremities, etc.

The Snap tools can be activated and deactivated using the relevant buttonsCommand field that is responsive to mouse clicks and that enables operations to be activated..

When the buttons are highlighted, the relevant snap tools are active.

13. Snap


Icon buttons and text buttons

Snap end point

Link the cursorMobile element that is visible on the screen, and which moves as the mouse is moved. to the ends of the objectGeneral term used to define any drawing created in the drawing area (e.g. piece, texts, flat figures, etc.). segments (created as lines, arcs, polylinesSet of consecutive straight lines, connected to each other at the ends. or FreeForms).

14.  

If the object is a three-dimensional figureGeometric figure of a three-dimensional space (e.g. the surfaces)., the magnet is activated on the corners.

Snap middle point

Link the cursor to the mid-points of the object segments (created as lines, arcs or polylines). This commandTool designed to perform a specific function/operation. For example, the “Print” command allows you to print a document. cannot function on objects created as FreeForms.

15.  

If the object is a three-dimensional figure, the magnet is activated on the mid-points of the corners and edges.

Snap intersection

Link the cursor to the intersection points of the segments of an object (created as lines, arcs or polylines). This command cannot function on objects created as FreeForms.

16.  

Snap perpendicular

Force the cursor onto the point on an object that forms the perpendicular with the geometric elementAn abstract element associated with specific properties: the point, line, plane and space are all geometric elements. being created. In this way, you can generate objects that are perpendicular to others.

It is possible to make two lines perpendicular (or a line and an arc, or a line and a circle).

17.  

Snap near point

Link the cursor to any point of an object.

18.  

Snap parallel

Makes the cursor remain parallel to the reference object.

19.  

Snap tangent

Link the cursor to the tangency point between two consecutive segments.

20.  

Snap centre of gravity

Link the cursor to the centre of a closed geometric figureFlat figure in which the first and last extremities coincide (they meet)..

21.  

Snap centre

Links the cursor to the centre of a circle or arc.

22.  

Snap quadrant

Link the cursor to the end points of a circular figure (arc, circle or ellipse) on which the quadrant axes pass. The position of these points is determined by the construction planePlane on which a flat geometry is constructed. The default plane is the XY plane. set; considering the xy plane, the points are at 0°, 90°, 180° and 270°.

23.  

Snap intersection with active drawing plane

Link the cursor to the active construction plane.

Layer tools

Commands for managing the levels.

24. Layer

Set colour from layer

Reset the colours defined in the level, applying them to all the objects (even those with personalised colours).

Select colour

Used to personalise the colour of the selected objects compared with the standard colour (i.e. the one applied to all the objects of the level).

The colour selected from the colour table will be associated with the selected object only (all the other objects in the level will maintain the colour defined by the level itself).

Layer

Show the list of levels created.

Show

Indicates that the level is active (i.e. the objects it contains are visible).

Click on this iconGraphic image used to show a data item, button, etc. to switch off the light-bulb and hide the level.

Hide

Indicates that the level is hidden (i.e. the objects it contains are not visible).

Click on this icon to switch on the light-bulb and show the level.

Block

Indicates that the level is locked. In this case, the objects belonging to the active level cannot be modified.

By clicking on this icon, the lock opens and the level is unlocked.

Release

Indicates that the level is released. In this case, the objects belonging to the active level can be modified.

By clicking on this icon, the lock closes and the level is locked.

Modify layer

Show the dialogue box for managing the level (setting the lines, colours and materials of the three-dimensional objects, etc.).

The colours of the objects are defined in each single level.

Tools for machining operations

Display rapids

Enables or disables the visualisation of the paths travelled by the tool at maximum speed (quick tool movements).

Display the tool in the machining operations

Enables or disables the visualisation of the tool.

Draw each machining operation using a different colour

Apply (button enabled) a different colour to each single machining operation, to distinguish it from the others.
