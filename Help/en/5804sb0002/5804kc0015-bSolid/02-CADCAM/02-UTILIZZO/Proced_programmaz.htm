Programming phases

Below, you will find the procedure to follow in order to create a machining programType of document (file) containing all the instructions needed for machining the pieces of wood, glass or marble..

1. Create a new documentElectronic worksheet on which the geometric and technological parameters are inserted. (Creating or opening a document).
2. Create the piece (Create a standard piece; Creating non-standard faces).
3. Project the geometry with the aid of the tools needed to draw flat figures and surfacesFigures of a three-dimensional area, that represent a thin open or closed structure.. The projection can be made in the following ways:
• By importingInsert, in an open document (file), information created with another program and in a different format from the one currently being used. files created with other application softwareA synonym of "program" - software with the specific task of managing data, graphics, etc. packages.
• By using the drawing tools .

For the projection, you can also use the sequences (Applying the sequences).

4. Apply the machining operations to the surfaces or geometries generated, using the specific tools of the machining operations barBar listing the tools for applying the machining operations. - (Machining operations bar) commandTool designed to perform a specific function/operation. For example, the “Print” command allows you to print a document..
5. Simulate the machining operation in order to check it is correct.
6. Tool the work table.
7. Save the document.
8. Process the machining program to generate the file with ISOAcronym of the "International Standardisation Organisation", that determines technological and industrial standards. File format containing the programming language of the numerical control machine. extensionIndicates the type of file (i.e. the type of application) that the file is associated with. Consists of 3 or 4 characters preceded by a dot. For example, the extension "txt" indicates a text-only file that can be read with any text Editor., so that it can be executed on the machine.

Create a standard piece

The piece can be created using one of the following methods:

Create by entering dimensional parameters

Create a standard piece by entering the size data in the relative area of the parameters "Piece ".

1. Click(1) Indicates the action of pressing (and immediately releasing) the mouse button on an object (icon, button, etc.) located under the pointer on the screen. (2) (verb) Pressing and immediately releasing the left mouse button to enable the function of a selected command. on the buttonCommand field that is responsive to mouse clicks and that enables operations to be activated. of the commands area.
2. Enter the size data in the relative area of the parameters Piece:
• Piece length
• Piece width
• Piece thickness

At this point, you can either enter the machining operations or create profiles to which you will then attribute machining operations.

Creating by selecting a flat figure
1. Make the drawing.
2. Click on the button of the commands area.
3. Use the "Finished piece geometry" and "Blank piece geometry" data fieldsA defined area containing data or commands (data field or command field). to select the type of shape you want to generate.

By drawing two flat figures, you can create two pieces: one represents the shape of the finished piece (ref. A) and the other represents the shape of the rough piece (ref. B).

4. Enter the piece thickness in the "Piece thickness" data fieldA defined area containing data or commands (data field or command field)..

If the geometry representing the finished piece (ref. C) is not rectangular, you can generate the shape of the rough piece automatically (ref. D) by tickingClick on a check box to tick it and thereby enable that function. the "Rectangular rough" box.

Creating non-standard faces

To generate non-standard (or customised) faces, proceed as follows:

1. Select the face of the piece and make the drawing.
2. Select the drawing.
3. Click on the button of the commands area, then fill in the data fields in the relative parameters areaPart of the data area with the cards for managing the geometric and technological parameters. and confirm.

Applying the sequences

The sequences are applied in the Cad/Cam environment.

To show (in the data areaGeneral term to define a specific area for data entry.) the card with the list of sequences to be applied, proceed as follows:

1. Click on the and buttons of the machining operations bar.
2. Choose the machining operation sequence to be applied to the geometry, then confirm.
3. Insert the geometry in the relevant sequence field.

4. Modify the type of correction and the lead-in point if necessary.
5. Click on the Calculate command, then confirm.

Parameterising the program

Program parameterisation involves entering predefined variables (e.g. LPX; LPY, etc.), created variables, and system parameters in any field of the geometry or machining operation data area. This produces a parameterised program.

Each program can be fully or partially parameterised, according to the result you wish to obtain. If the variables are only entered in certain data fields, you will obtain a partially parameterised program.

When you use variables to make all the geometries parametric, you cannot make any modifications using the tools of the CAD options barBar listing the optional tools useful for drawing (e.g. the commands to rotate or resize the drawing, etc.) without losing the parametric values.

Applying the automatic machining operations

The automatic machining operations are applied in the Cad/Cam environment.

To automatically assign machining operations to the geometries created, click on the and buttons of the machining operations bar.

On the basis of the rules applied, the system will process the geometries and create the possible machining operations in the tree listStructure containing a number of items, known as nodes, which in turn have branches and secondary branches like a tree. The node at the highest level is called the root. All the others derive from one single node and, in turn, can have other nodes..

Applying the macros

The macros are applied in the Cad/Cam environment.

Proceed as follows:

1. Create a new document or open an existing program.
2. Visualise the macros bar by clicking on the button.
3. Click on the required macro.
4. Fill in the data fields, then save.

Simulating the machining operation

To perform simulations in the Cad/Cam environment, open the simulator by clicking on the button and then proceed as follows:

1. Select the machining operation from the tree list "Machining operation timeline".
2. Launch the simulation by clicking on the button.
During the simulation, you can move on to the next stepAcronym of "STandard for the Exchange of Product model data". Type of normalised, neutral file used as a graphic data (3D, 2D and wireframe) exchange format between the various CAD and CAM systems. or back to the previous one by using the buttons.
3. At the end of the machining operation, click on the button.

To slow down or speed up the simulation, use the plus or minus buttons of the "Override" field. Use the relevant commands to show or hide the graphics of the tool, machining operations, etc. (see Settings).
