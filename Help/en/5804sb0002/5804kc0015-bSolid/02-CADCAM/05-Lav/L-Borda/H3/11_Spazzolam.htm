Horizontal copying

CPH

Forward movement (ref. B, figure 3), above the edge to be cleaned, carried out by the aggregate to correctly position the tool in such a way that the brushes perfectly clean the edge.

The reference point on the tool, from which to start, is defined in the Control points data field.

Vertical copying

CPV

Movement in z (ref. B, figure 4) of the aggregate to raise or lower the tool so that the brushes perfectly clean the edge.

The reference point on the tool, from which to start, is defined in the Control points data field.
