General parameters

X dimension of piece

Used to define the default value of the x dimension of the piece.

This value is automatically measured if it was not specified in the imported DXF layer.

Y dimension of piece

Used to define the default value of the y dimension of the piece.

This value is automatically measured if it was not specified in the imported DXF layer.

Z dimension of piece

Used to define the default value of the piece thickness.

This value is automatically measured if it was not specified in the imported DXF layer.

DXF tolerance

Used to define the accuracy of the decimal figures for converting the DXF files.

Discretise ellipses

Enables (box tickedClick on a check box to tick it and thereby enable that function.) or disables the decomposition of the elliptical geometries found in the DXF files, breaking them up into numerous curved segments.

This function may be handy when the ellipse arc is part of a profile.
If the box is disabled, the ellipse can be imported as a single part.

Import layer in alphabetic order

Enables (box ticked) or disables the order for importing the layers of the DXF files.

  â¢    If the box is enabled, this indicates that the DXF layers are imported in alphabetic order.
  â¢    If the box is disabled, this indicates that the imported layers are not subjected to any rule; they maintain the order in which they were created.
Symbols recognise upper case letters

Enables (box ticked) or disables the "case sensitive" function, i.e. the ability to distinguish lower case letters from upper case ones.

For example, in order to be recognised by the system, the keys in the table must correspond to the original ones defined in the layers. The upper or lower case letters must be written in the correct way: the word "panel" cannot be recognised as "Panel" or "PANEL", but only as "panel".

If the box is disabled, this indicates that the key can be written in either upper or lower case.

String separator

Default character for separating the alphanumeric values (to be used for the layer syntax).
