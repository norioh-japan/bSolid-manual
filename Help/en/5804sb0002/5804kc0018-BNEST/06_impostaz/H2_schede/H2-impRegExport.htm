Extension

ExtensionIndicates the type of file (i.e. the type of application) that the file is associated with. Consists of 3 or 4 characters preceded by a dot. For example, the extension "txt" indicates a text-only file that can be read with any text Editor. of the file to save without the dot (e.g. CSV).

Destination folder

Used to define the folderContainer for storing files or documents. where the files exported during the transmission phase will be saved.

Images folder

Used to define the folder where the files of the images exported during the transmission phase will be deposited.

The images saved have the extension .PNG and are not overwritten; the system always creates new ones with different data.

Formatting options
Enables formatting of the date and time

Enables (box ticked) or disables the possibility to define a date and time format for the information defined in the file saved.

This setting is very useful in order that the date and time appear in the name of the exported file each time it is saved; in this way you will have many copies with different dates and the file will never be overwritten.

Date

Choose the type of date from the drop-down menuList of options or instruments which expand vertically or horizontally when you click on the drop-down buttons.: M (month); D (day); Y (year).

NOTES:

By default, the date is separated by a slash (/); entering the date in the data field "File name", the characters not permitted will be replaced automatically, because for the operating system the name of the files cannot contain the following characters: \ / : * ? " < > |.

Time

Choose the type of time in the drop-down menuList of commands (to be selected with the mouse) that develop a sub-menu, i.e. a list of options that allow certain functions/operations to be carried out.: H (hours); m (minutes); s (seconds).

NOTES:

By default, the time is separated by a colon; entering the time in the data field "File name", the characters not permitted will be replaced automatically, because for the operating system the name of the files cannot contain the following characters: \ / : * ? " < > |.

Format

Displays the final format. This field allows to enter a custom date and time format.

Example:

Entering MM-yyy_HH-mm-ss and setting the data "Date and time" in the field"File name", the name of the saved file will have the prefix ART_ followed by the month, year and time.

Enables formatting of numbers
