Processes the projects:

In the project table

Launches the calculation of the nesting projects selected in the table.

In the card Articles of the project

Launches the calculation of the active nesting project.

This calculation indicates all the files needed: machining programs, work lists, etc.

If in section " Remainder calculation" boxes Enable generation of rests(nesting) A residual piece eliminated during the machining operation, that can be re-used. and Modify rest layoutDiagram showing the arrangement of the document to be printed. have been enabled, the appropriate window bNest opens to manage the remainders produced during processing.

623. Dialogue box bNest

Example of remainders produced:

Used to modify the remainders produced, and to save them in the Remainders archive.

  •    The left-hand area shows the results of the processing, with the patterns produced and the remainders highlighted.
  •    The right-hand area shows the selected pattern, so it can be modified if necessary.

Sends the data:

In the project table

Sends all the files (obtained from the calculation of the nesting projects selected in the table) and sends them to the machine.

In the card Articles of the project

Sends all the files obtained from the calculation of the active nesting project to the machine.
