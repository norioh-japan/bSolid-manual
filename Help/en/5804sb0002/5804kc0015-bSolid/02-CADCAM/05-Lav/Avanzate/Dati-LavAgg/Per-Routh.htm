Additional machining data

Optimisable

OPT

Optimisation of the machining operation. TickClick on a check box to tick it and thereby enable that function. the box to enable optimisation of the machining operation.

  •    If the box is disabled, the machining operation is not optimised and is performed respecting the position indicated in the programming rows. In this case, the corresponding programming row maintains its execution position.

If this box is disabled, an orange circle will appear in the machining operation iconGraphic image used to show a data item, button, etc. in the Machining operation timeline tabLabel which, if selected with the cursor, displays a list of data..

  •    If the box is enabled, the machining operation is optimised (processed by the CAMAcronym of "Computer Aided Manufacturing". Type of software allowing the use of a CAD to control and manage the functioning of the operating machines. A CAD/CAM system enables pieces to be designed, drawn and machined.) and is performed without respecting the sequence established in the programming rows. In this case, the corresponding programming row does not maintain its execution position.
Safe descent on channel

TOS

Enables or disables the calculation of the tool safety position starting from the surfaceFigures of a three-dimensional area, that represent a thin open or closed structure. obtained from a channel, but only if a valuePredefined / pre-set. Standard information useful for the control, if the requested information is not entered. has been specified in the "Z channel" data fieldA defined area containing data or commands (data field or command field)..

  •    Field enabled: the data item defined in the "Z channel" field is not considered, and the calculation for the tool safety position starts from the piece surface.
  •    Field disabled: the data item defined in the "Z channel" field is taken into consideration, and the calculation for the tool safety position starts from the surface obtained from the channel.

Example in figure 1

Let's suppose that a value of 10mm is set in the fieldsA defined area containing data or commands (data field or command field). relating to tool safety, and a value of 5mm in the "Z channel" field. If the "Safe descent on channel" field is enabled, the tool will position itself 10mm from the piece surface. If the "Safe descent on channel" field is left disabled, the tool will position itself 10mm from the Z position (i.e. 5mm from the piece surface).

Z

Spindle

SPI

Used to indicate the electrospindle to be used for the machining operation.

This field may be useful when you need to carry out a machining operation with an aggregate(tool) Device fitted with tools./tool that is only supported by a specific spindle.

NOTES:

You should check that the selected electrospindle has actually been installed, or that it is compatible with the magazine that houses the tool to be used. If it is not, the optimisation will not be successful. The empty data field authorises the CAM to choose the electrospindle.

Type of positioning

SHT

Used to choose how to position the suction hood during the machining operation.

  •    By position
  •    By distance from piece
Hood position

SHP

Hood position during the machining operation.

This data item is only valid on machines fitted with the special device for correcting the hood position.

Enter a value from 1 to 6:

  •    A value of 1 corresponds to the hood up.
  •    A value of 0 activates the automatic positioning of the hoods (managed on the basis of the set machining data).

If the value entered is not compatible with the settings defined in the machining operation, the hoods will in any case be positioned at a limit height automatically worked out on the basis of the defined data.

Distance from piece

SHD

Distance from the piece surface for positioning the suction hood.

Floating

SWI

Enables (box tickedClick on a check box to tick it and thereby enable that function.) or disables the use of the copier or other similar devices (e.g. the aggregates with the copier).

Blower

BLW

Enables (box ticked) or disables the use of the blower to clean the piece whenever the copier or presser is used.

Presser

PRS

Enables (box ticked) or disables the use of the presser.

Chip cleaning blow

BFC

Enables (box ticked) or disables the use of the blower on the piece, to remove chips left behind during machining operations.

AirTool

ETB

Enables (box ticked) or disables the cooling of the inner part of the tool during the machining operation, thanks to a jet of air.

AirJet

AJT

Enables (box ticked) or disables the cooling of the tool bit during the machining operation, thanks to a jet of air.

Ioniser

ION

Enables (box ticked) or disables the cleaning of the tool bit during the machining operation, thanks to a jet of ionised air.

Lubricator

LUBMNZ

Enables (box ticked) or disables the lubrication of the tool during the machining operation, thanks to the emission of a liquid.

Connect machining operation

EML

Enables (box ticked) or disables the possibility to link one machining operation with another that has the same properties. If the box is ticked, the tool will not rise between one machining operation and the next.

Explode into tangent sections

ETG

Enables (box ticked) or disables the generation of multiple milling operations on the basis of the non-tangent points found on the profile being machined. If the box is ticked, the machining path is divided into several milling operations; every time non-tangent points are found on the profile, the tool rises and begins machining the second tangent section again.

Tangency tolerance

ETGT

Distance useful for the system for establishing that two consecutive geometric elements should be considered tangent.

Enabling the cutter management

ETAG

Automatically enables (box ticked) the following data fields:

  •    TCP tracking(see "Milling ")
  •    Explode into tangent sections
NOTES:

Only machines for processing plastic materials.

Enable aggregate from below

A21ON

Enables (box ticked) or disables the use of the aggregate for machining operations on the lower face of the piece.
