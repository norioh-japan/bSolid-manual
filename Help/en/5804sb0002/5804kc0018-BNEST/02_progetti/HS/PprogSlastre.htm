NOTES:

By enabling the boxUse cutting optimiser, this section is disabled.

Multiple corner mode

Enables (box tickedClick on a check box to tick it and thereby enable that function.) or disables the use of several corners when positioning the profiles.

If this box is ticked, the positioning begins from the corners towards the inside of the sheet.

Optimise direction on sheets

Enables (box ticked) the next data fieldA defined area containing data or commands (data field or command field)..

If this box is not ticked, the project is processed in such a way as to align the profiles on the sheet in the best possible way.

Pieces insertion direction

Used to choose the direction for aligning the profiles positioned on the sheet.

Along X

The positioning is made by aligning the profiles in x.

Along Y

The positioning is made by aligning the profiles in y.

Piece insertion direction on last sheet

Used to choose the direction for aligning the profiles positioned on the last sheet.

For a description of the options, see the previous data field.
