Carriage transfer

Description

In the tree listStructure containing a number of items, known as nodes, which in turn have branches and secondary branches like a tree. The node at the highest level is called the root. All the others derive from one single node and, in turn, can have other nodes. "Machining operation timeline", insert the instruction for moving the carriages.

To enter the instruction
1. Click(1) Indicates the action of pressing (and immediately releasing) the mouse button on an object (icon, button, etc.) located under the pointer on the screen. (2) (verb) Pressing and immediately releasing the left mouse button to enable the function of a selected command. on the card"Machining operation timeline" .
2. Position the cursorMobile element that is visible on the screen, and which moves as the mouse is moved. between two machining operations.
3. Open the functions bar.

4. Click on the buttonCommand field that is responsive to mouse clicks and that enables operations to be activated..

Main data

Name

NAME

Used to indicate the number of the carriage to be moved, with the relative number of the mobile support (to be entered between quotation marks).

Practical example:

To move carriage number 2 of mobile support number 3, the syntax to be used is: "3,2" where 3 is the support and 2 is the carriage.

Increase position

INCPOS

Used to determine if the data item set in the "Position" fieldA defined area containing data or commands (data field or command field). indicates the positioning co-ordinate or an incremental positioning valuePredefined / pre-set. Standard information useful for the control, if the requested information is not entered..

True

Incremental value.

False

Positioning co-ordinate.

Position

POS

Carriage positioning.

  •    If "True" is displayed in the "Increase position" field:

The carriage moves towards corners 2 and 3 of the piece, by the amount of the value indicated in this field.

e.g. figure 1: if a value of 100 is entered in this field, and the "True"option is selected, the carriage will be moved 100 millimetres from its current position.

  •    If "False" is displayed in the "Increase position" field:

The carriage moves, starting from corner 1 of the piece, by the amount of the value indicated in this field.

e.g. , figure 11: if a value of 250 is entered in this field, and the "False"option is selected, the carriage is positioned 250mm from corner of the piece.

133. Example
A. Value 100mm.
B. Co-ordinate at 250mm, referring to corner 1 of the piece.

 

Carriage reference

REF

Used to select the reference for carriage positioning.

134. Example
A. Origin 1
B. Upper side
C. Central side
D. Lower side
Upper

Carriage side facing towards corners 1 and 4 of the piece (ref.B, figure 2).

Central

Centre of the carriage (ref.C, figure 2).

Lower

Carriage side facing towards corners 2 and 3 of the piece (ref.D, figure 2).

Tab Advanced

Name

NMC

Function name.

The text entered in this field will rename the function text string (the one near the iconGraphic image used to show a data item, button, etc. of the chosen tool) to distinguish it from the others in the "Geometries and machining operations" tree list. If this field is left empty, the system will automatically enter the default text.

Execute machining

EXE

Allows the user to enter a condition for a function to be performed or not.

Insert a formula or choose one of the two options:

  •    False: do not carry out.
  •    True: carry out.
Save Default

Used to save the data set as default - always shown when using the specific function.
