Description

The Working environment definition dialogue box is used for all the operations needed for defining the characteristics of the working area of the Cad/Cam environment.

3. Working environment definition

For the description of the data fields:

Point of origin

Used to define the Cartesian co-ordinates (x1 and y1) of the limit point in the lower left part of the drawing areaArea dedicated to the creation and modification of the geometries/machining operations..

Extension point

Used to define the Cartesian co-ordinates (x2 and y2) of the limit point in the upper right part of the drawing area.

Extension in Z

Used to define the z co-ordinates of the upper (z+) and lower (z-) limit points of the drawing area, with regards the depth.

Adapt the dimensions to the project

Used to adapt the dimensions of the drawing area to the objectsGeneral term used to define any drawing created in the drawing area (e.g. piece, texts, flat figures, etc.). drawn.

Plane height

Used to define the height of the drawing planes (xy, yz and zx). Enter the valuePredefined / pre-set. Standard information useful for the control, if the requested information is not entered. in the text boxes below.

Grid

Enables the data fieldA defined area containing data or commands (data field or command field). "Grid step".

Grid step

Used to define the distance between the points of the gridA network of lines. For a detailed description, see the paragraph dealing with the grid. (i.e. between the vertices of the little rectangles that form the grid itself).

Snap

Enables the data field "Snap step".

Snap step

Used to define the snap gaps (i.e. the gap between one point and the next) for attracting the cursorMobile element that is visible on the screen, and which moves as the mouse is moved. to the grid.

Absolute co-ordinates

Used to define the co-ordinate system (x, y, z) whose reference is the origin of the axes.

Relative co-ordinates

Used to define the co-ordinate system (x, y, z) whose reference is the last drawn point.

Polar coordinates

Used to define the co-ordinate system (l, a, z) in space, whose reference is the last drawn point.

• l: distance from the last drawn point.
• a: angle formed with the x-axis.
• z: Cartesian co-ordinate.
Orthogonal mode activated

Enable the orthogonal mode. In this case, the points inserted in the drawing area take the forced direction parallel to the three axes x, y and z, and angled at 45°.

Orthogonal mode deactivated

Disable the orthogonal mode.

Configuring the drawing area

Dimensions of the drawing area

To define the dimensions of the drawing area, proceed as follows:

1. Click on the button.
2. Click on the button.
3. In the "Point of origin" section, define the Cartesian co-ordinates x1 and y1.
4. In the "Extension point" section, define the Cartesian co-ordinates x2 and y2.
5. In the "Extension in Z" section, define the Cartesian co-ordinates z+ and z-.

To adapt the dimensions of the drawing area to the objects, press the "Adapt the dimensions to the project" button.

Positioning the drawing planes

To define the position of the drawing planes xy, yz and zx, proceed as follows:

1. Click on the button.
2. Click on the button.
3. In the "Plane height" section, define the height of the various planes.
• XY data field: define the z position.
• YZ data field: define the x position.
• ZX data field: define the y position.
