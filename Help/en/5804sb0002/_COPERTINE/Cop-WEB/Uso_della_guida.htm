Using the search field

To search for texts within the guide, proceed as follows:

  •    Enter the text in the search fieldA defined area containing data or commands (data field or command field).. To search for a specific phrase, use the quotation marks (" ").
  •    Click on the search command to see a list of the topics containing the required text.
  •    Bring the cursorMobile element that is visible on the screen, and which moves as the mouse is moved. onto the link.
