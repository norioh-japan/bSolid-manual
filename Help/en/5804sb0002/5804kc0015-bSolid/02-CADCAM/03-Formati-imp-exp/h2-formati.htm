File formats supported by bSolid

Associated procedures
To export the data from the environment Cad/Camand save them in a different format :
1. Click(1) Indicates the action of pressing (and immediately releasing) the mouse button on an object (icon, button, etc.) located under the pointer on the screen. (2) (verb) Pressing and immediately releasing the left mouse button to enable the function of a selected command. on the buttonCommand field that is responsive to mouse clicks and that enables operations to be activated..
2. Use fieldA defined area containing data or commands (data field or command field). A to search for the folderContainer for storing files or documents. where the documentElectronic worksheet on which the geometric and technological parameters are inserted. is to be saved.
3. Specify the file name in field B.
4. Save by clicking on button C.
To import graphic and/or technological files into the environmentCad/Cam:
1. Click on the button of the standard bar. The system windowA window for navigating within the Windows operating system. will open.
2. Click on button A to search for the folder containing the files to be imported.
3. Select the required file. To visualise only files of a certain type, use drop-down menuList of options or instruments which expand vertically or horizontally when you click on the drop-down buttons. B and choose the required file format.
4. Confirm the importInsert, in an open document (file), information created with another program and in a different format from the one currently being used. by clicking on button C.
