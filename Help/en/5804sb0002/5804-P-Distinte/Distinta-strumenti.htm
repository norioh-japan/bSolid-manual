Populates the work list with the programs indicated in the bar code scanned with the reader, and executes them.

Used to manage the work list by reading the bar codes.

NOTES:

This type of mode allows you to manage the insertion of programs and the execution of the list even without using the scanner.

This command is only visible if bar code management has been activated.

How to activate bar code management:

1. Access the General settings environment.
2. Click on the BarCode card.
3. Click on the check box Bar code reception activated.
4. Enter the data needed to configure the reader.

Clicking on the button displays the following pull-down menu:

Program added

Using the bar code reader:

Add the program to the list via the reading of the bar code.

Without using the bar code reader:

Enables the command for adding the program to the list when programs are already being executed.

In order to add the program to the list, the file must physically reside in the program folderContainer for storing files or documents. defined in the General settings environment.

Select and run the program in the work list

Search the list for the program read in the bar code, and perform it.

In this case, the codes to be read must only show the name of the label Label (which must be unique), as the search starts from the program label.

The search for the label name is made amongst the list rows. Once it has been found, the program in the relative row is launched.

Program added and run

Using the bar code reader:

Add the program to the list via the reading of the bar code, and perform it.

Without using the bar code reader:

Enables the command for adding the program to the list and automatically executing it.

In order to add the program to the list, the file must physically reside in the program folder defined in the General settings environment.
