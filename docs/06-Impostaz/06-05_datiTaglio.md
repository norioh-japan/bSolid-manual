# 6.5 切断データ設定（Dati Taglio）

**カテゴリ**: システム設定 > 切断パラメータ  
**技術レベル**: 中級〜上級  
**最終更新**: 2025-01-30

---

## 📋 **概要**

切断データ設定は、bSolidのCAM機能における切断作業の精度、効率、安全性を制御する重要なパラメータ群です。木材、金属、複合材料など様々な材料に対応した高精度切断を実現するための包括的な設定システムを提供します。

### **主要機能**
- **高精度切断制御**: μm単位での精密切断パラメータ管理
- **材料別最適化**: 材料特性に応じた自動パラメータ調整
- **AI支援最適化**: 機械学習による切断条件の自動最適化
- **リアルタイム監視**: 切断プロセスの動的調整機能

## 🔧 **基本切断パラメータ**

### **ブレード許容差（Tolleranza Lama）**

```yaml
設定範囲: 0.01mm - 2.00mm
推奨値:
  - 精密切断: 0.05mm
  - 一般切断: 0.10mm
  - 粗切断: 0.20mm
```

#### **技術仕様**
- **許容差範囲**: ±0.01mm 〜 ±2.00mm
- **測定精度**: 0.001mm分解能
- **自動補正**: レーザー測定による動的調整
- **材料別設定**: 木材、金属、プラスチック対応

#### **設定詳細**

| 材料タイプ | 推奨許容差 | 備考 |
|------------|------------|------|
| **広葉樹** | 0.05-0.08mm | 高精度加工対応 |
| **針葉樹** | 0.08-0.12mm | 繊維方向考慮 |
| **MDF/パーティクルボード** | 0.10-0.15mm | 密度変動対応 |
| **プラスチック** | 0.03-0.06mm | 熱変形考慮 |
| **アルミニウム** | 0.02-0.04mm | 金属専用設定 |

### **複数ラン設定（Multi-Run）**

高効率生産を実現する多段階切断制御システムです。

#### **安全度モード（Safety Mode）**
```
動作シーケンス:
1. 初期位置 → 切断開始点
2. 第1回切断実行
3. 安全位置へ退避（Z軸+50mm）
4. 次の切断位置へ移動
5. 降下 → 第2回切断実行
```

**技術仕様**:
- **安全高度**: 25-100mm（可変設定）
- **移動速度**: 5000-15000mm/min
- **位置精度**: ±0.02mm
- **衝突検知**: 3Dセンサー統合

#### **インパクトモード（Impact Mode）**
```
動作シーケンス:
1. 第1回切断完了
2. 同一Z高度で次位置へ移動
3. 材料表面接触を維持
4. 連続切断実行
```

**技術仕様**:
- **接触圧力**: 0.1-2.0N（可変）
- **移動速度**: 8000-20000mm/min
- **表面追従**: ±0.1mm精度
- **振動制御**: アクティブダンピング
