FreeForm from points

Description

Creates a FreeForm by joining the selected points.

96. Example

A.  Start point or polyline to be joined
B.  FreeForm type - Interpolating
C.  FreeForm type - Approximating
D.  Point outside - see Distance

Approximating

See the description of the "Approximating"(FreeForm) data field.

FreeForm closure

SE

Used to select the type of outline to be created (open or closed).

Open

Generates an open outlineFlat figure in which the first and last extremities do not coincide (they do not meet)..

Closed

Generates a closed outlineFlat figure in which the first and last extremities coincide (they meet)..

Distance

D

Maximum distance between two points used to construct the FreeForm.

Start point or polyline to be joined

GEO1

Used to select the initial point of the FreeForm or, in the case of a closed FreeForm, the polylineSet of consecutive straight lines, connected to each other at the ends. to be joined with the FreeForm.
