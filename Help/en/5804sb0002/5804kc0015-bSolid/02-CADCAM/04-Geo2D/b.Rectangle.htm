Rectangle

Description

Creates a parallelogram: square, rectangular or rhomboidal.

Steps for drawing a parallelogram
1. Click(1) Indicates the action of pressing (and immediately releasing) the mouse button on an object (icon, button, etc.) located under the pointer on the screen. (2) (verb) Pressing and immediately releasing the left mouse button to enable the function of a selected command. on the buttonCommand field that is responsive to mouse clicks and that enables operations to be activated. of the drawing barBar listing the drawing tools..
2. Insert the start point by clicking in the drawing areaArea dedicated to the creation and modification of the geometries/machining operations..
To specify the point or precise dimensions (e.g. to generate a square), use the relevant fields of the data area.

3. Click on the point where you want to end the drawing.
4. Click on the Undo button to close the Rectangle card.

Advanced parameters

Rotation

R

Rotation of the rectangle on the xy construction planePlane on which a flat geometry is constructed. The default plane is the XY plane. starting from the start point. Enter the required angle.

Chamfer type

CT

Used to chamferChamfering produces a corner. The chamfer can be linear or rounded. to sharp corners of the selected geometry, inserting arcs or straight lines.

No chamfer

Sharp corners will not be chamfered.

Circular chamfer

Inserts curved chamfers.

Linear chamfer

Inserts straight chamfers.

Diversified chamfers

CS

Enables (box tickedClick on a check box to tick it and thereby enable that function.) or disables the choice of the type of chamfer for each single corner of the rectangle.

Chamfer type 1

CT1

Used to choose the type of chamfer to be applied to the first corner of the rectangle.

This data item is only visible if the "Diversified chamfers" box has been ticked.

For the description of the options, see "Chamfer type".

Chamfer type 2

CT2

Used to choose the type of chamfer to be applied to the second corner of the rectangle.

This data item is only visible if the "Diversified chamfers" box has been ticked.

For the description of the options, see "Chamfer type".

Chamfer type 3

CT3

Used to choose the type of chamfer to be applied to the third corner of the rectangle.

This data item is only visible if the "Diversified chamfers" box has been ticked.

For the description of the options, see "Chamfer type".

Chamfer type 4

CT4

Used to choose the type of chamfer to be applied to the fourth corner of the rectangle.

This data item is only visible if the "Diversified chamfers" box has been ticked.

For the description of the options, see "Chamfer type".

Chamfer value

CD

For arc chamfers:

Radius of the arc used to round off the two consecutive, non-tangent elements.

For linear chamfers:

Distance D from the corner of the two consecutive, non-tangent elements.
