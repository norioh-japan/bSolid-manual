Tab Configurations

Shows the data fieldsA defined area containing data or commands (data field or command field). for setting the configurations to be applied to the window project:

Frame configurations

Used to choose the documentElectronic worksheet on which the geometric and technological parameters are inserted. for the frame(doors/windows) Window structure to be fixed to the wall or jamb, on which the window is hung. configuration created in the relative environment (see Archive Frame configuration).

Sash configurations

Used to choose the document for the sash configuration created in the relative environment (see Archive Sash configuration).

Hardware configuration

Used to choose the configuration document for the hardware created in the relative environment (see Archive Hardware configuration).

Machine load configuration

Used to choose the configuration for the machine load(doors/windows) Positioning of the window pieces on the work table. created in the relative environment (see Archive Machine load configurations).
