Grouping objects and machining operations

To create foldersContainer for storing files or documents. for grouping together objectsGeneral term used to define any drawing created in the drawing area (e.g. piece, texts, flat figures, etc.). and/or machining operations, proceed as follows:

1. Create an object and rename it.
2. Create a second object with the same name as the first one. In this case, the system creates a folder and puts both objects inside it.

The coloured pencil iconGraphic image used to show a data item, button, etc. is used to activate the folder containing all the objects and/or machining operations generated from the moment of its activation.

Modifying the parameters of the geometries/machining operations

To modify a created object or a machining operation by intervening directly on the parameters, proceed as follows:

1. Check the closure of the parameters areaPart of the data area with the cards for managing the geometric and technological parameters. (see Parameters area) linked to the previously selected commandTool designed to perform a specific function/operation. For example, the “Print” command allows you to print a document..
2. Click twice on the nodeFor the definition, refer to Tree list. of the card "Geometries and machining operations" to modify the required geometry or machining operation.

In the lower part of the data areaGeneral term to define a specific area for data entry., you will see the two cards of the parameters area.


3. Modify the parameters, then save.

Showing or hiding the machining operations

To show or hide the machining operations, access the tree listStructure containing a number of items, known as nodes, which in turn have branches and secondary branches like a tree. The node at the highest level is called the root. All the others derive from one single node and, in turn, can have other nodes. "Geometries and machining operations" and click on the light-bulb symbol.

  •    The yellow bulb indicates that the node is switched on; the tool paths are shown in the drawing areaArea dedicated to the creation and modification of the geometries/machining operations..
  •    The grey bulb indicates that the node is switched off; the tool paths are hidden in the drawing area.

Activating or deactivating the machining operations

To activate or deactivate the machining operations, click twice on the machining operation of the tree list "Geometries and machining operations", then proceed as follows:

  •    To enable the machining operation, select the True option in the Execute machining data fieldA defined area containing data or commands (data field or command field).. In this case, the machining operation will be performed.
  •    Select the False option in the Execute machining data field to disable the machining operation.
In this case, the text string of the command icon shown in the tree list will turn light grey, and the machining operation will not be performed.
