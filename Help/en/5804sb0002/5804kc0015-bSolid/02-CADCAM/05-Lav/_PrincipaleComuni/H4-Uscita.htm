Lead-out

In this section, you can define how the tool should emerge from the machined piece.

The fieldsA defined area containing data or commands (data field or command field). are set automatically (with predefined values) after you have selected a tool. These fields may vary according to the type of tool chosen and the type of machining operation.

Type

Movement performed by the tool in order to exit the machined piece.

Choose the desired output type from the drop-down menuList of options or instruments which expand vertically or horizontally when you click on the drop-down buttons..

Same as the lead-in

Use the same option set in the "Type" data fieldA defined area containing data or commands (data field or command field)..

None

The tool emerges from the piece directly at the machining end point, rising perpendicularly to the face.

752. None



Line

The tool emerges from the piece by following a short linear path (along which it corrects its position), then ascends perpendicularly to the face.

753. Line



Arc

The tool emerges from the piece by following a short arc path (along which it corrects its position), then ascends perpendicularly to the face.

754. Arc



Line-arc tangent

The tool emerges from the piece by forming a tangent line and arc, then ascends perpendicularly to the face. The correction is made along the line and arc.

755. Line-arc tangent



3D line-arc

The tool emerges from the piece by following a short arc path (along which it corrects its position), then ascends by following a tilted line.

756. 3D line-arc



Helix

The tool emerges from the piece by following a helicoidal path, then traces a line in the air, parallel to the face, in order to correct its position.

757. Helix



3D line

The tool emerges from the piece with a tilted ascent, then traces a line in the air, parallel to the face, in order to correct its position.

758. 3D line

3D arc

The tool emerges from the piece with an arc ascent, then traces a line in the air, parallel to the face, in order to correct its position.

759. 3D arc



Corrected line

The tool emerges from the piece with an ascent perpendicular to the face, then traces a line in the air, parallel to the face, in order to correct its position.

760. Corrected line



Follow 3D profile

The tool emerges from the piece directly at the machining end point, then follows a tilted line of ascent. The use of this type of exit is only allowed with closed profiles.

761. Follow 3D profile



Element extension

Used to delay the exit of the tool from the piece. The tool emerges from the piece by following a short linear path (along which it corrects its position), then ascends perpendicularly to the face. Indicate the length of the linear path in the "Section length" data field.

762. Element extension



3D element extension

Used to delay the emergence of the disc-shaped tool from the piece.
The tool emerges from the piece by following a short linear path (along which it corrects its position), then ascends perpendicularly to the face. Indicate the length of the linear path in the "Section length" data field.

Correction in air

Enables (box tickedClick on a check box to tick it and thereby enable that function.) or disables the correction of the tool position at the end of its re-ascent.

Angle

Tool lead-out angle from the piece.

The type of angle to be entered varies according to the type of lead-out previously selected.

Section length

Linear length of the segment for delaying the exit of the tool from the piece in order to correct its position.

Percentage radius

Percentage for modifying the radius of the arc created by the tool when it moves away from the piece.

Insert tabbing

Enables (box ticked) or disables the linear transfer of the tool within the piece exit trajectory, in order to create a small stepAcronym of "STandard for the Exchange of Product model data". Type of normalised, neutral file used as a graphic data (3D, 2D and wireframe) exchange format between the various CAD and CAM systems.. This transfer is calculated on the programmed machining depth.

If the box is ticked, the following data fields are shown: "Tabbing length"; "Tabbing position"; "Variance".

Tabbing length

Linear length of the segment for transferring the tool as it emerges.

• See ref. C, figure 12.
Tabbing position

Position, in mm, for determining the transfer end point, calculated starting from the Z co-ordinate of the milling end point.

• See ref. B and P, figure 12.
Variance

Distance in millimetres from the lead-out point of the machining operation for transferring the tool so that it ascends with a slight tilt.
