Explode geometry

Dismantles a complex objectGeneral term used to define any drawing created in the drawing area (e.g. piece, texts, flat figures, etc.). such as a polylineSet of consecutive straight lines, connected to each other at the ends., showing only the individual geometric elements.

Invert

Inverts the start and end points of the selected geometric elements.

In this case, the two points of a single element (such as a line) - or of all the elements belonging to an outline - are inverted: the start point becomes the end point.

Render tangents

Creates geometric continuity between consecutive, non-tangent curved segments. In this case, the two segments become tangent.

Divide at intersections

Breaks up the incident geometric elements. In this case, the incidence (or intersection) point becomes the point in which the elements are broken up.

Eliminate overlaps

Deletes all geometric elements overlapping other ones.

Cut element

Eliminates the selected geometric elements.

Element extension

Lengthens the selected geometric elementAn abstract element associated with specific properties: the point, line, plane and space are all geometric elements., bringing it closer to the specified one.

(2) (verb) Pressing and immediately releasing the left mouse button to enable the function of a selected command. on this buttonCommand field that is responsive to mouse clicks and that enables operations to be activated. and select the element towards which you want to extend it.

Corner extension

Joins several geometric elements to create one single outline (polylineSet of consecutive straight lines, connected to each other at the ends.).

Median line

Creates a median line between two selected lines.

Wireframe Creator

Generates the skeleton of the selected three-dimensional object, creating single geometric elements and polylines.
