Automatic edgebanding

Description of the machining operation

Used to manage the piece edgebandingSet of operations for covering the edge of the piece (gluing, cleaning, rounding off, etc.). operations: gluing(1) Set of operations for attaching the edgebanding strip to the piece., edge trimming(1) Piece edge trimming: operation for making cuts on the side faces (heads) in order to square the piece. (2) Border edge trimming: cuts at the ends of the edgebanded side., blowing(1) Cleaning of the piece to be machined, or the surface to be edgebanded., etc.

After selecting a side or a profile in the drawing areaArea dedicated to the creation and modification of the geometries/machining operations., the applicationA synonym of "program" - software with the specific task of managing data, graphics, etc. of the machining operations that appear in the adjacent data areaGeneral term to define a specific area for data entry. depends on the settings defined in the configuration used - i.e. only the activated machining operations are inserted (iconGraphic image used to show a data item, button, etc. ) (see Edgebanding operation configuration data).

Configuration selection

Allows to enter in the program the configuration defined in the archive "Edgebanding rules card " of the environment bSolid rules.

The buttonCommand field that is responsive to mouse clicks and that enables operations to be activated. shows the New configuration from process window, for saving the machining operations.

In this case, a new configuration documentElectronic worksheet on which the geometric and technological parameters are inserted. for the edgebanding operations is created in the archive of the bSolid rules environment (see "Edgebanding rules card ").

Usage procedure:
1. Indicate the name of the new configuration in the relevant data fieldA defined area containing data or commands (data field or command field)..
2. Choose the edgebanding strip.
3. Select the machining operations in the left-hand area.
4. Confirm the operation carried out.

Section selection

Area dedicated to the selection of geometries and the visualisation of the edgebanding strips applied.

Edgebanding strip

Used to choose the edgebanding strip from the archive.

In the list shown, the active edgebanding strip is highlighted by a green LED at the top right.

Model

Used to choose a model.

Machining operations

Shows a list of the machining operations applied to the geometry selected in the drawing area.

The list varies according to the type of configuration used and the type of geometry selected: closed or open geometry.

The button shows the relevant dialogue box for saving the data modified in the machining operation configuration document created in the bSolid rules environment (see "Edgebanding rules card ").

Usage procedure:
1. Choose the data fieldsA defined area containing data or commands (data field or command field). to be saved.
2. Choose the machining operation configuration documents where you want to save the data.
3. Confirm the operation carried out.
