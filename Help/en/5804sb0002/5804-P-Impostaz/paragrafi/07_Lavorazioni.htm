Mode

Used to define how the optimiser should process the program, evaluating all the possible tests, in order to generate detailed diagnosticsIndications for resolving any errors or malfunctioning..

Choose one of the following options:

Automatic

The optimiser chooses the method on the basis of the number of machining operations programmed.

Perform all tests

The optimiser makes all the possible checks on the compatibility between the programmed tools and the tool-holders/spindles on the machine.

This function is only enabled if the number of machining operations does not exceed the value defined in the " Number of machining operations " field.

Stop at the first error

The optimiser makes the appropriate checks on the compatibility between the programmed tools and the tool-holders/spindles on the machine, stopping at the first error found.

Reduced test and stop at the first error
