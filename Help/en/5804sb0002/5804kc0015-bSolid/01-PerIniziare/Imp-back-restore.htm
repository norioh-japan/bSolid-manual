Reset - machine data storage

Import the machine data

1. Disable the check boxEnable machine data synchronisation in the environment General settings.
2. Access the Machine data environment.
3. Click(1) Indicates the action of pressing (and immediately releasing) the mouse button on an object (icon, button, etc.) located under the pointer on the screen. (2) (verb) Pressing and immediately releasing the left mouse button to enable the function of a selected command. on the buttonCommand field that is responsive to mouse clicks and that enables operations to be activated. and enter the password.
4. Click on the button to launch the data importInsert, in an open document (file), information created with another program and in a different format from the one currently being used. window.
5. In the Select the d_xnc folderContainer for storing files or documents. to be imported data fieldA defined area containing data or commands (data field or command field)., enter the name of the folder containing the data to be imported. To search for the folder, use the search button alongside.
6. In the Select the folder containing the machine templates data field, enter the name of the folder containing the machine graphics to be imported, in order to generate 3-D images of the machine and its components (e.g. the folder of the DVD where the data will be taken from: E:\MachineTemplates_v1.1.0.). To search for the folder, use the search button alongside.
7. Click on the Next button.
8. In the Family drop-down menuList of options or instruments which expand vertically or horizontally when you click on the drop-down buttons., choose the type of family that the machine belongs to.
9. In the Model drop-down menuList of commands (to be selected with the mouse) that develop a sub-menu, i.e. a list of options that allow certain functions/operations to be carried out., choose the machine model.
10. Click on the Next button.
11. Wait for the processing phase, and choose the required options of the following data fieldsA defined area containing data or commands (data field or command field).:
5-axis head

Drop-down menu with a list of the types of flange.

Calibration jig

Drop-down menu with a list of the shapeable suction cups. Choose the one used during the machine testing phase.

Type of coupling

Drop-down menu with a list of the types of tool coupling.

Machine with EC safety system

Used to import the machine with the EC guards and safety devices.

The Data imported table contains a list of all the data saved during the import phase, along with any errors.

12. Click on the Import button to launch the import operation.
13. Wait for the end of the import process and the appearance of the following page:

To check the imported data and any errors, click on the button in the red circle and scroll though the Data imported table.

14. Click on the Next button.
15. Click on the Perform the tool import/exportOperation involving the generation of a file (document) that will be read by other software programs. radio button to import or export the list of tools.
  •    The green button next to each tool indicates its presence in the databaseData bank. Stored data arranged in an organised manner so that they are readily available using a search and management program. of the bSolid application softwareA synonym of "program" - software with the specific task of managing data, graphics, etc.. In this case, you can overwrite the tools by means of the green arrows.
• the right-hand arrow overwrites the tool of the bSolid database, copying it from the NC;
• the left-hand arrow overwrites the tool of the NC, copying it from the bSolid database. If the arrows are deactivated, the tools are not copied.
  •    The red button next to each tool indicates the absence of the tool in the bSolid database or the NC.
16. Click on the green arrows corresponding to the tools you want to import or export, and click on the Import button.
17. Click on the Next button.
18. Click on the Perform the import/export of aggregates and deflectors option button to import or export the list of aggregates and/or deflectors. The import operations are similar to those described for the tools - see point 15.
19. Click on the Next button.
20. Click on the Perform the tooling import option button to import the tooling configuration for magazines and working unitsMachine parts that house tools needed for the machining operation. For the description of the individual units, refer to the machine user instructions..
21. Click on the Next button.
22. Enter the name of the machine in the Name data field.
  •    To set the imported machine as the standard machine, tickClick on a check box to tick it and thereby enable that function. the Set as default machine box.
  •    To define the data for the simulator, tick the Set simulation data box.

Create data backup copies

1. Access the General settings environment.
2. Click on the General settings card.
3. Click on the Perform backup button. The system windowA window for navigating within the Windows operating system. will open.
4. Use field A to search for the folder where the backup files will be deposited.
5. Specify the file name in field B.
6. Save by clicking on button C.
7. Wait for the end of the data copying procedure, then confirm.

Reset the stored data

NOTES:

The basic data reset overwrites everything and adds tools and data that are not present (see also point 5).

1. Search for the folder where the backup files were deposited.
2. Click twice on the required .bBack file. The data reset window opens.
3. Tick the boxes corresponding to the data you want to reset:
Tools

Save the list of tools in the tool database. In this case, tools with the same name will be overwritten.

Aggregates

Save the list of aggregates in the tool database. In this case, aggregates with the same name will be overwritten.

Deflectors

Save the list of deflectors in the tool database. In this case, deflectors with the same name will be overwritten.

Sequences

Save the list of sequences in the tool database.

Automatic machining operations

Save the list of automatic machining operations in the tool database.

Default values

Overwrite the data of the machining operations and geometries saved as default in the Advanced tabLabel which, if selected with the cursor, displays a list of data. of the parameters areaPart of the data area with the cards for managing the geometric and technological parameters..

Edgebanding devices

Save the list of edgebandingSet of operations for covering the edge of the piece (gluing, cleaning, rounding off, etc.). aggregates in the tool database.

Aggregates with the same name will be overwritten.

Edgebanding strips

Save the list of edgebanding strips in the tool database.

Edgebanding strips with the same name will be overwritten.

Labels

Save the list of labels in the tool database.

Labels with the same name will be overwritten.

Macro

Save the list of macros in the tool database.

Macros with the same name will be overwritten.

Models

Save the list of models (the ones used to create a new documentElectronic worksheet on which the geometric and technological parameters are inserted. - machining programType of document (file) containing all the instructions needed for machining the pieces of wood, glass or marble.).

CAM files

Update the contents of the "...\Biesse\bSolid\BswFiles" folder.

4. In the Select the machines to be restored data field, click on the required machines to overwrite their data; your selection will be highlighted with a blue frame(doors/windows) Window structure to be fixed to the wall or jamb, on which the window is hung..
5. Only tick the Delete all the data before restoring the backup box if, during the reset phase, you want to delete all the data and replace them with those taken from the backup fileCompressed file (usually a zip format file) containing a copy of the required data, that can be restored when necessary..
Otherwise (to overwrite the existing data and add new data), leave the box unticked.
6. Click on the Restore button.
7. Close the dialogue box and launch the bSolid applicationA synonym of "program" - software with the specific task of managing data, graphics, etc. software.
