Serial port settings

Port name

Name of the serial port that the reader is connected to. The list shows only the ports available in the PC.

Parity

Used to define whether or not to consider the “parity bit” used to detect transmission errors. Choose the check from the drop-down menuList of options or instruments which expand vertically or horizontally when you click on the drop-down buttons..

Data bits

Number of bits for transmitting a character between the reader and the PC.

Baud rate

Speed of data transmission from the serial port.

Barcode settings

Bar code reception activated

Enables (box tickedClick on a check box to tick it and thereby enable that function.) or disables the reading of the bar codes for compiling and performing the work list.

In this case, the list is populated using the specific reader.

Enable multiple reading for a single row

Enables (box ticked) or disables the modification of the following data fieldsA defined area containing data or commands (data field or command field)..

Connection character for multiple reading

Separating character for reading several bar codes.

Standby time between one multiple reading and the next (milliseconds)

Time after which the arrival of new characters is considered invalid.

Barcode end sequence (ASCII codes separated by ,)

ASCII codes determining the end of the bar code reading. To indicate several codes, use commas.

Field separator character

Enables (box ticked) or disables the insertion of the separator character between the fields of the string, and disables the fields of the table shown below.

Inserting a separator character, it is possible to read a string consisting of fields separated by a character (a semi-colon, by default) without having to respect the number of characters defined in the table below.

Cad delimiter character variables

Allows to define the character used to enclose and then to interpret the custom variables and the parameters LPX, LPY and LPZ of the environmentCad/Cam.

For example, by entering an asterisk, if the code contains the string LPZ=100 delimited by two asterisks *LPZ=100*, the reader interprets that LPZ is a parameter of the environment Cad/Cam.

For the description of the variables, see "Basic concepts on the variables".

NC delimiter character variables

Allows to define the character used to enclose and then to interpret the specific variables of the ISOAcronym of the "International Standardisation Organisation", that determines technological and industrial standards. File format containing the programming language of the numerical control machine. file.

For example, by entering a hashtag, if the code contains the string G68=100 delimited by two hashtags #G68=100#, the reader interprets that GS8 is an ISO variable.
