Using the list via the reading of the bar codes

NOTES:

The List environment opens, always showing the last card used. In this case, you can create a new list, open one already in the databaseData bank. Stored data arranged in an organised manner so that they are readily available using a search and management program., or fill in the one currently open.

Activating the bar code reader

To enable the user of the bar code reader, proceed as follows:

1. Access the General settings environment.
2. Click(1) Indicates the action of pressing (and immediately releasing) the mouse button on an object (icon, button, etc.) located under the pointer on the screen. (2) (verb) Pressing and immediately releasing the left mouse button to enable the function of a selected command. on the BarCode card.
3. Click on the check box Bar code reception activated.
4. Enter the data needed to configure the reader.

Creating a new work list

1. Access the List environment.
2. Click on the buttonCommand field that is responsive to mouse clicks and that enables operations to be activated..
3. Enable the use of the bar code reading by clicking on the iconGraphic image used to show a data item, button, etc..

The blue colour of the icon indicates that barcode mode is active.

4. Use the options of the drop-down menuList of options or instruments which expand vertically or horizontally when you click on the drop-down buttons. to populate the work list.

Compiling and performing the work list

Using the options of the drop-down menuList of commands (to be selected with the mouse) that develop a sub-menu, i.e. a list of options that allow certain functions/operations to be carried out. next to the icon, you can perform the following operations:

Add programs
1. Creating a new work list.
2. Select the Program added option.
3. Use the specific reader and scan the bar codes to be inserted in the list.

The reading of each single bar code inserts a new row in the list.

4. Click on the button to perform the compiled list.
Select and perform the programs
1. Open the list containing the programs to be performed.
2. Enable the use of the bar code reading by clicking on the icon.
3. Select the Select and run the program in the work list option.
4. Use the specific reader and scan the bar codes to be selected from the current list.
The bar codes to be read must only indicate the name of those labels indicated in the Label data fieldA defined area containing data or commands (data field or command field). of the rows in the list.

This procedure allows you to automatically launch the programs read by the reader; each subsequent reading adds the program to the end of the list.

Add and perform the programs
1. Creating a new work list.
2. Select the Program added option.
3. Use the specific reader and scan the bar codes to be inserted in the list.

The reading of each single bar code inserts a new row and automatically launches the selected program.

Import a list
1. Creating a new work list.
2. Select the Work list importInsert, in an open document (file), information created with another program and in a different format from the one currently being used. option.
3. Use the reader to scan the bar code.

The list will be populated by the same number of rows as the number of programs indicated in the list read in the bar code.

4. Click on the button to perform the compiled list.
