Execute geometry

EXE

Used to enter a condition for performing or not performing the commandTool designed to perform a specific function/operation. For example, the “Print” command allows you to print a document..

Insert a formula or choose one of the two options:

  •    False: do not carry out.
  •    True: carry out.

If the command execution is deactivated, the text string of the command icon in the geometries and machining operations tree list will turn light grey.

For example, if you enter the formula X1>200 in the rectangle command, the rectangle is only visualised and executed if X1 (Start X) is greater than 200.

Layer

LAY

Used to select the level to be assigned to the geometry created.

Construction inversion

INVD

Used to invert the geometry construction direction.

Direction of normal construction

Brings the geometry construction direction back to its initial status.

Direction of inverted construction

Inverts the geometry construction direction. In this case, the initial creation point becomes the end point.

Save Default

Used to save the data set as default - always shown when using the specific drawing command.
