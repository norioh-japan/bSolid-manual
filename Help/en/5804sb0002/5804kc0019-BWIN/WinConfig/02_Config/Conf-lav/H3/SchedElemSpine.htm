502. Description of the tab
A. Range area
B. Dowel management area

Add range

Creates a new rangeValues included between a maximum and a minimum. in the "Range for sections" box.

Remove range

Eliminates the range selected in the "Range for sections" box.

Set the range as default for preview

Sets the range selected in the "Range for sections" box as the standard valuePredefined / pre-set. Standard information useful for the control, if the requested information is not entered..

Preview recalculation

Updates the drawing in the preview area.

Range for sections

Box where the ranges can be entered: minimum and maximum width of the pieces. The machining operation is only applied if the width A of the pieces is within the values indicated in each single range.

Tilt

Enables (tickedClick on a check box to tick it and thereby enable that function. box) or disables the creation of tilted bores.

Dowels length

Length of the dowels.

Add

Adds a new bore.

Remove

Deletes the bore from the list.

X position

POSX

X co-ordinate of the bore, starting from the reference corner (at the top left).

Y position

POSY

Y co-ordinate of the bore, starting from the reference corner (at the top left).

Z position

POSZ

Z co-ordinate of the bore, starting from the reference corner (at the top left).

Radius

RADIUS

Radius of the bore.

Screw

SCREW

Enables (box ticked) or disables the use of screws instead of dowels. In this case, the borings are applied on both the inner and outer sides in order to generate a through bore for inserting the screw.

Perforation

MARK2

Enables (box ticked) or disables blind bores for inserting small screws.

The following operations can be performed in the preview area:

  •    show or hide the piece;
  •    make the piece transparent;
  •    modify the piece dimensions (thickness and width).
