Use of the environment Tooling

Using the middle key of the mouse

The central key of the mouse can be used for the following operations:

  •    Zoom the display of the objects in or out.

Procedure:

1. Press the central mouse key (scroll wheel).
2. Rotate the scroll wheel to zoomProcess of reducing or enlarging the graphic display. in or out on the objectsGeneral term used to define any drawing created in the drawing area (e.g. piece, texts, flat figures, etc.)..
  •    Move the display of the objects.

Procedure:

1. Press the central mouse key (scroll wheel) to display the hand cursorMobile element that is visible on the screen, and which moves as the mouse is moved..
2. Keep the central key (scroll wheel) pressed, then move the mouse to shift the display and drag the object in the required direction.
  •    Rotate the view associated with the Ctrl key of the keyboard.

Procedure:

1. Press the Ctrl key of the keyboard and the centre key of the mouse (scroll wheel).
2. Keeping both pressed, move the cursor.

Tooling the operating section

To tool the operating sectionSet of parts: working units and devices., access the tooling environment by clicking on the buttonCommand field that is responsive to mouse clicks and that enables operations to be activated. of the environments barButtons that allow you to access the various environments of the application software..

Inserting the tools in the spindles

The installed tools can be easily identified in the relevant tool area by the green circle around the iconsGraphic image used to show a data item, button, etc..

When tools are inserted in the spindles, the tool name appears in the right-hand tree listStructure containing a number of items, known as nodes, which in turn have branches and secondary branches like a tree. The node at the highest level is called the root. All the others derive from one single node and, in turn, can have other nodes., near the spindle name.

To deselect the active spindles, use the Esc key on the keyboard.

Tooling a single spindle

Observe one of the following methods:

  •    Select a tool in the relevant area and drag it onto the graphics of the spindle to be tooled (Drag and drop(drag and drop) Indicates the action of keeping the left mouse key pressed on a selection (object, icon, node, etc.) in order to drag it or move it.).
  •    Use the cross cursorType of cross-shaped cursor. to select (in the graphics area) the spindle to be tooled.

Select a tool in the relevant area, and insert it with a double click(1) Indicates the action of pressing (and immediately releasing) the mouse button on an object (icon, button, etc.) located under the pointer on the screen. (2) (verb) Pressing and immediately releasing the left mouse button to enable the function of a selected command..

  •    In the right-hand tree list, select the icon of the required unit or electrospindle in order to open the list of spindles, then click on the spindle to be tooled; the selection and the spindle will both turn blue.

Click twice on the icon of the tool in the relevant area.

Multiple spindle tooling

Observe one of the following methods:

  •    Using the cross cursor , press the Ctrl key on the keyboard and click on the graphics of the spindles to be tooled; the selections will turn blue.

Move the cursor to the relevant area and click twice on the icon of the tool to be inserted in the active spindles.

  •    In the right-hand tree list, select the icon of the required unit or electrospindle in order to open the list of spindles, then click on each single spindle, keeping the Ctrl key of the keyboard pressed; the selections and spindles will turn blue.

Move the cursor to the relevant area and click twice on the icon of the tool to be inserted in the active spindles.

Removing tools from the spindles

Observe one of the following methods:

  •    Use the cross cursor to select (in the graphics area) the spindle to be untooled. To select several spindles, use the Ctrl key of the keyboard as well.

Untool by clicking on the button; alternatively, press the Canc key on the keyboard.

  •    To untool the entire operating section, click on the button.
NOTES:

This commandTool designed to perform a specific function/operation. For example, the “Print” command allows you to print a document. untools the whole machine: tool-holders and spindles.

  •    In the right-hand tree list, select the icon of the required unit or electrospindle; the list of spindles will appear, highlighting the unit in the graphic area.

On the tree list, select all the spindles you want to untool, then click on the button.

Tooling the tool magazine

To tool the tool magazine, access the tooling environment by clicking on the button of the environments bar.

Inserting the tools in the tool-holders

The installed tools can be easily identified in the relevant tool area by the green circle around the icons.

When tools are inserted in the tool-holders, the tool name appears in the right-hand tree list, near the tool-holder name.

To deselect the active tool-holders, use the Esc key on the keyboard.

Tooling a single tool-holder

Observe one of the following methods:

  •    Select a tool in the relevant area and drag it onto the graphics of the tool-holder to be tooled (Drag and drop).
  •    Use the cross cursor to select (in the graphics area) the tool-holder to be tooled.

Select a tool in the relevant area, and insert it with a double click.

  •    In the right-hand tree list, select the icon of the required magazine in order to open the list of tool-holders, then click on the one you want to tool.

The selection and tool-holder will both turn blue.

Click twice on the icon of the tool in the relevant area.

Multiple tool-holder tooling

Observe one of the following methods:

  •    Using the cross cursor , press the Ctrl key on the keyboard and click on the graphics of the tool-holders to be tooled; the selections will turn blue.

Move the cursor to the relevant area and click twice on the icon of the tool to be inserted in the active tool-holders.

  •    In the right-hand tree list, select the icon of the required magazine in order to open the list of tool-holders, then click on each single tool-holder, keeping the Ctrl key of the keyboard pressed; the selections and tool-holders will turn blue.

Move the cursor to the relevant area and click twice on the icon of the tool to be inserted in the active tool-holders.

Removing tools from the tool-holders

Observe one of the following methods:

  •    Use the cross cursor to select (in the graphics area) the tool-holder to be untooled. To select several tool-holders, use the Ctrl key of the keyboard as well.

Untool by clicking on the button; alternatively, press the Canc key on the keyboard.

  •    To untool the whole magazine, click on the button.
NOTES:

This command untools the whole machine: tool-holders and spindles.

  •    In the right-hand tree list, select the icon of the required magazine; the list of tool-holders will appear, highlighting the magazine in the graphic area.

On the tree list, select all the tool-holders you want to untool, then click on the button.

Saving the machine tooling

After tooling the operating section and tool magazine, click on the button to store the configuration of the new tooling.
