Structure of the DXF files to be imported

NOTES:

bSolid is only able to interpret 2DTwo-dimensional geometric figures (constructed on two dimensions). DXF files, commonly known as DXF-2D.

To ensure that bSolid can recognise the profiles in the DXF files as technological elements (and therefore importInsert, in an open document (file), information created with another program and in a different format from the one currently being used. them correctly), it is necessary to follow specific design rules.

The design of DXF files generated with general CADAcronym of "Computer Aided Design". Type of editor for three-dimensional (3D) and/or two-dimensional (2D) graphic design. programs is linked to the creation of levels (layers) with specific syntaxes relating to technological information about the profiles in the level itself.

The layers are the “levels” present in the CAD which allow one drawing to be separated from another, and each layerLevels of a document created to carry out a drawing. can be named by inserting numbers and keys. The operation to be carried out thus relates to the procedure used to name the layer in such a way that the drawing associated with it will be interpreted correctly.

Precautions when creating the DXF layers

  •    Each level must show certain information about the profile it contains and about the type of machining operation. Thus, a circle can only be interpreted as a bore if the string that identifies the layer associated with it contains the boring data.
  •    The syntax of the layers used in the DXF files to be imported must respect the keys defined in the relevant dialogue box DXF definition. The syntax of the levels, and the rules to be respected, will vary according to the type of DXF file to be imported:
• Biesse Works DXF
• CNI DXF
  •    Geometric elements with no layer are interpreted as simple geometries.

For the procedures relating to the construction of the DXF files compatible with bSolid, see:

• Designing the layer profiles and syntax (Biesse Works DXF)
• Designing the layer profiles and syntax (CNI DXF)

DXF files (the creation of the keys and parameters) is carried out in the General settings environment (see Dialogue boxDXF definition ).

4. Click on the Type active drop-down menuList of options or instruments which expand vertically or horizontally when you click on the drop-down buttons., and choose one of the two options:
• Biesse Works DXF

Make sure the keys in the relevant tables correspond to those used in the DXF files to be imported.

Importing DXF files

In order to use the geometries found in a DXF file, perform the import (see Importing files in the document) then proceed as follows:

1. Select all the geometries.
2. Click on the button and set the position at 0.
3. Click on the button to make sure all the geometries have been correctly moved.
4. Select all the geometries, then click on the button.
5. Select all the geometries, then click on the button and set the position at 0.
6. Select all the geometries, then click on the button.
7. Select all the geometries, then click on the button.
8. Select all the geometries and create the piece (see Create a standard piece).
